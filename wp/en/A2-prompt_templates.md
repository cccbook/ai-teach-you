# A2. Complete Collection of Common Prompt Templates

## Part 1: Website Building

### Website Overall Planning

```
【Website Type Judgment Prompt】
Role: You are a website strategy consultant
Task: Based on the following description, help me choose the most suitable website type and platform

Background:
- Main purpose: [Showcase/Sell/Share/Brand]
- Target audience: [Description]
- Budget: [Budget range]
- Technical ability: [Complete beginner/Some experience/Experienced]

Needs:
1. Recommend most suitable website type
2. Recommend 2-3 suitable platforms
3. Explain pros and cons of each platform
4. Estimate required time and cost
```

```
【Website Architecture Planning Prompt】
Role: You are a UX architect
Task: Help me plan the website's page structure and navigation

Background:
- Website type: [Type]
- Main functions: [Function list]
- Target audience: [Description]

Needs:
1. List all necessary pages
2. Design navigation structure
3. Explain main purpose of each page
4. Mark linking relationships between pages
```

### Website Copy

```
【Landing Page Hero Copy Prompt】
Role: You are a high-conversion landing page copywriter
Task: Help me write copy for the Hero section

Background:
- Product/Service: [Name]
- Main selling point: [Core value]
- Target audience: [Description]
- Action goal: [CTA]

Needs:
1. Main headline (8-12 words, pain point or benefit)
2. Subheadline (within 20 words, detailed explanation)
3. CTA button text (5-8 words)
Format: Present in Markdown table
```

```
【Brand Story Prompt】
Role: You are a brand strategy expert and copywriter
Task: Write a realistic and moving brand story

Background:
- Brand name: [Name]
- Founder: [Name]
- Origin: [Why started]
- Philosophy: [Core values]

Needs:
1. Structure: Origin → Challenge → Turning point → Now → Vision
2. Length: 600-800 words
3. Tone: Sincere, not exaggerated
4. Add 1-2 small stories
```

```

```

## Usage Tips

### How to Use These Templates

1. **Copy framework**: Copy Prompt template to document
2. **Fill in background**: Fill in content in `[...]` based on your actual situation
3. **Adjust requirements**: Add or remove items based on specific needs
4. **Test and iterate**: Adjust Prompt based on AI responses

### Ways to Improve Effectiveness

- Provide more specific details
- Saying what you don't like is also important
- Requesting multiple versions allows comparison
- Let AI explain thinking process
- Deepen gradually based on responses

# 10. Designing E-commerce Landing Pages

## What Is a Landing Page

A landing page is a "single page specifically designed for a goal," unlike general websites with multiple page navigation, a landing page has only one goal:

> **Persuade visitors to complete a specific action**

### Landing Page vs General Website

| Item | Landing Page | General Website |
|------|--------------|-----------------|
| Goal | Single conversion goal | Multiple goals |
| Links | Few (keep focused) | Many (for browsing) |
| Navigation | Minimal or none | Complete |
| Suitable for | Promotion, campaigns, product launch | Brand image, long-term operation |

### Common Landing Page Types

1. **Product launch**: Introducing new products
2. **Limited-time promotion**: Holiday, anniversary campaigns
3. **Free trial**: Samples, trial offers
4. **Course registration**: Online courses, workshops
5. **Event registration**: Seminars, events

## AI Copywriting Generation Prompts

### Hero Section Copy

```
Role: You are a conversion rate optimization expert

Task: Help me write copy for landing page Hero section

Product: [product name]
Product type: [product type]
Main benefit: [what problem does the product solve]
Target audience: [who will buy this product]

Needs:
1. Main headline (8 words or less, powerful, pain point or benefit)
2. Subheadline (one sentence explaining biggest selling point)
3. CTA button text (10 words or less, action-oriented)

Example style:
Headline: "Weight loss is no longer a painful synonym"
Subheadline: "Revolutionary method for lazy people to easily lose weight"
CTA: "Try free for 14 days"
```

### Problem Resonance Copy

```
Role: You are a copywriter who understands target customer psychology

Task: Help me write copy for problem resonance section

Product: [product name]
Target audience's pain points: [problems customers usually face]

Needs:
1. List 3-4 pain points
2. Each pain point described in one sentence with visual imagery
3. Tone should be "I understand you" not "You're wrong"
4. Format: emoji + short phrase + pain description

Format example:
😫 Failed diet countless times
   Every time I set my mind to it, I give up halfway...

🍕 Can't resist delicious food
   Diet plan always starts from tomorrow...
```

### Product Feature Copy

```
Role: You are a professional product copywriter

Task: Help me write copy for product feature section

Product: [product name]
Product features: [3-4 main features]

Needs:
1. Each feature presented as title + description
2. Titles should be specific, strong action words
3. Descriptions should address pain points, explain what changes it brings
4. Avoid empty adjectives like "high quality" "the best"

Format example:
🚀 30 seconds to get started
   No manual needed, open box and use

💰 Save 50% of time
   Work that took 2 hours, now done in 1 hour
```

### Client Testimonial Copy

```
Role: You are a copywriter skilled at writing customer voices

Task: Help me write 3 client testimonials

Product: [product name]
Target audience: [typical buyers]
Common usage scenarios: [how customers usually use this product]

Needs:
1. 3 different background testimonials
2. Each testimonial includes:
   - Problem before use
   - Change after use (data is better if possible)
   - One sentence summary
3. Testimonials should sound like real user feedback
4. Can add profession, age and other background to make testimonials more vivid

Format example:
"I'm [background], before [problem],
after using [product], [specific change],
[summary]. — [name], [profession]"
```

### FAQ Copy

```
Role: You are a customer service expert

Task: Help me write common FAQs

Product: [product name]
Target audience: [questions buyers might have]

Needs:
1. List 4-6 most common questions
2. Questions should be specific, not "Is this product good?"
3. Answers should be detailed, eliminate doubts
4. Can sneak in product promotion at the end of answers

Common FAQ directions:
- Is this suitable for me?
- How much? Any discounts?
- Shipping and delivery?
- Return/exchange policy?
- How to use?
- How is it different from other products?
```

## Landing Page Visual Design Tips

### Hero Visual Design Principles

1. **Product must be clear**: Should immediately see what the product is
2. **Contextual images add points**: Product in life usage context
3. **Before/After comparison**: Before/After visual impact is strong

### CTA Button Design

CTA (Call to Action) is the most important element of a landing page:

**Colors should stand out**:
- Contrast with page's main color
- Red, orange, green are common CTA colors
- Avoid blending with background

**Copy should be specific**:
- ❌ "Submit" "Send" "Click here"
- ✅ "Try Free" "Get Now" "Sign Up Now"

**Multiple locations**:
- Hero section
- After feature introduction
- After testimonial section
- After FAQ section
- Footer

### Elements That Eliminate Doubts

Make customers buy with confidence:

1. **Security payment badges**: Credit card company logos, SSL badges
2. **Return/exchange guarantee**: 30-day return, money-back guarantee
3. **Free shipping display**: Free shipping across site, free shipping over X amount
4. **Limited/limited-time**: Low stock, offer countdown
5. **User data**: Already X thousand people purchased

## Landing Page Checklist

### Conversion Optimization Check

- [ ] Page load time < 3 seconds
- [ ] Call to action button clearly visible
- [ ] Mobile CTA button large enough (at least 44px)
- [ ] Value proposition clearly conveyed in 5 seconds
- [ ] At least 2-3 testimonials
- [ ] FAQ answers common doubts
- [ ] Urgency elements at the end (countdown, stock)

### Technical Check

- [ ] All links work
- [ ] Form can submit properly
- [ ] GA/tracking code properly installed
- [ ] Mobile version displays properly
- [ ] Images have Alt text
- [ ] Has SSL certificate

### Content Check

- [ ] Headline is eye-catching
- [ ] Product features clearly explained
- [ ] Price is transparent
- [ ] Contact information easy to find
- [ ] No typos or grammar errors

## A/B Testing Concepts

When not sure which version is better, you can do A/B testing:

### Suggested Test Items

| Test Item | Version A | Version B |
|-----------|-----------|----------|
| CTA button color | Red | Green |
| Main headline | Feature-oriented | Benefit-oriented |
| Image | Product photo | Contextual photo |
| Price display | Original price struck through | Directly show discounted price |

### Test Duration

Each version should run at least 2-4 weeks, and each version should have at least 1000 visits.

# 9. Building a Brand Image Website

## Core Purpose of Brand Image Website

A brand image website is different from other types of websites. Its primary task is:

> **Make visitors believe in 5 seconds that you are trustworthy**

### Essential Elements for Brand Websites

| Element | Purpose |
|---------|---------|
| Brand story | Build emotional connection |
| Professional capability showcase | Prove your ability |
| Client testimonials/cases | Increase credibility |
| Clear contact information | Lower the barrier to action |

### About Page Design Points

The "About" page is key to building trust:

1. **Opening**: Who you are, what you do
2. **Story**: Why you do this
3. **Experience**: Professional background, certifications, experience
4. **Philosophy**: Your values and work style
5. **Photos**: Your photos, work scenes

### Service Page Design Points

Service page should clearly explain:

1. **Service name**
2. **Service description** (2-3 sentences)
3. **Service process** (how to start)
4. **Price range** (optional)
5. **CTA**: Book/Contact

## AI Helps You Write Brand Content

### Brand Story Generation Prompt

```
Role: You are a brand strategy expert and copywriter

Task: Help me write a real, warm brand story

Background:
- Brand name: [name]
- Founder: [name]
- Industry: [industry]
- Main service: [service content]
- Why started: [why you started this business]
- Core philosophy: [your values]

Needs:
1. Structure: Origin → Challenge → Turning point → Now → Future vision
2. Tone: Sincere, not exaggerated, make people want to keep reading
3. Length: 600-800 words
4. Add 1-2 small stories or details
5. Should ultimately echo the brand's core values
```

### Service Description Generation Prompt

```
Role: You are a professional marketing copywriter

Task: Help me write descriptions for service items

Service info:
- Service name: [name]
- Target customers: [who needs this service]
- Service content: [detailed description]
- Expected effect: [what customers can get]

Needs:
1. Each service description 100-150 words
2. Format: "Service name" + 3 description sentences + Call to action
3. Tone professional but not stiff
4. Emphasize "benefits" for customers, not just "features"
5. Start with verbs, increase sense of action
```

### Client Testimonial Generation Prompt

```
Role: You are a copywriter who understands customer psychology

Task: Help me write a few client testimonial examples

Background:
- Industry: [your industry]
- Typical customers: [who are typical clients]
- Common pain points: [what problems clients usually face]
- Service effect: [how your service solves problems]

Needs:
1. Write 3-5 testimonials
2. Each testimonial includes: one sentence summary + detailed feedback
3. Format: "Quote" — Name, Company, Position
4. Testimonials should be specific, data is better
5. Tone natural, not like advertising
```

## Visual Elements for Brand Websites

### Color Selection

Brand colors should consider:

1. **Industry characteristics**: Tech uses blue, food uses warm colors, nature uses green
2. **Target audience**: Young people use bright colors, business uses dark colors
3. **Emotional connection**: What feeling do you want to convey?

**Common brand color schemes**:

| Style | Primary Color | Accent Color |
|-------|---------------|--------------|
| Professional Business | Deep Blue #1a365d | Gold #d69e2e |
| Creative Tech | Purple #6b46c1 | Blue #3182ce |
| Natural Health | Green #38a169 | Beige #faf5eb |
| Fashion Boutique | Black #1a1a1a | Gold #ffd700 |
| Warm Service | Orange #dd6b20 | White #ffffff |

### Image Selection Principles

Images on brand websites should note:

1. **Consistency**: All images should have consistent tone and style
2. **Authenticity**: Best to have real work photos
3. **Professionalism**: Avoid low-resolution or amateur-looking photos
4. **Emotional connection**: Images should resonate with target customers

## Making Brand Websites More Persuasive

### Elements That Increase Trust

1. **Professional certifications/licenses**: Lawyer license, professional certifications
2. **Media interviews**: News, interviews you've been in
3. **Award experience**: Industry awards, competition results
4. **Partner logos**: Well-known clients, vendor logos
5. **Let data speak**: Number of people served, success rate

### Common Mistakes

1. **Trying to include everything**: Put all services on, blurring the focus
2. **Too self-centered**: Keep saying how great you are, not what you can do for clients
3. **Too official**: Tone like government documents, not approachable
4. **Casual images**: Use some unprofessional stock photos
5. **Lacking action**: After reading, don't know what to do next

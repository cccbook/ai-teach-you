# 第一章：經典物理學的危機——從以太到量子

## 1.1 歷史背景：物理學的黃金時代

十九世紀末的物理學界可以說是處於一個前所未有的黃金時代。在那個年代，牛頓力學、電磁學、熱力學與統計力學這四大支柱共同構成了一座宏偉而完整的物理學大廈大多數物理學家認為，這座大廈已經趨於完工，剩下的工作僅僅是一些細節上的補充與測量工作而已。這種樂觀的情緒在當時的物理學界普遍存在，彷彿自然界的所有奧秘都已經被解開，只剩下一些邊邊角角的問題等待解決。

牛頓力學以其驚人的精確性統治著宏觀世界。從行星的軌道運動到機械裝置的運作，從橋樑的結構設計到炮彈的飛行軌跡，牛頓第二定律 $F = ma$ 彷彿能夠描述一切力學現象。拉格朗日和哈密頓進一步發展的分析力學更是將牛頓力學推向了一個優雅而抽象的高度，使得力學系統的研究變得更加系統化和美麗。這種方法論的發展不僅簡化了複雜問題的求解，更重要的是揭示了力學系統內在的對稱性與守恆定律之間的深刻聯繫。

馬克士威方程組則統一了電學、磁學與光學這三個原本看似獨立的領域。馬克士威在1865年發表的著名論文中，展示了電場與磁場可以統一為单一的电磁场，並預言了电磁波的存在。根據馬克士威方程組，电磁波在真空中的傳播速度約為每秒三億米，這與當時已經精確測得的光速（約 $c = 2.99792458 \times 10^8$ 米/秒）驚人地一致！這一事實使馬克士威斷言：光正是一種 electromagnetic wave。

1887年，德國物理學家海因里希·赫茲（Heinrich Hertz）在實驗室中首次成功地產生了和檢測到了 electromagnetic wave，證實了馬克士威的理論預言。赫茲的實驗不僅是經典物理學的巔峰之作，更被當時的許多物理學家視為物理學大廈最終完工的標誌。誰也沒有想到，正是在這個看似完美的理論體系中，卻隱藏著足以颠覆整個物理學的深刻危機。

熱力學的發展同樣令人振奮。熱力學第一定律（能量守恆）與第二定律（熵增原理）為人類理解熱現象提供了堅實的理論基礎。熵的概念的引入，不僅解決了熱機效率的根本限制問題，更深刻地揭示了時間的单向性——為什麼過去和未來是不同的。與熱力學並行發展的是統計力學，這門由詹姆斯·克拉克·麥�克斯韋、路德維希·波茲曼和約西亞·威拉德·吉布斯等人發展起來的學科，將微觀粒子行為與宏觀熱力學性質緊密聯繫起來。統計力學的核心思想是：宏觀系統的性質可以從大量微觀粒子的統計平均中得到，而不需要追蹤每一個粒子的具體運動。

然而，統計力學的發展也帶來了一些深刻的哲學問題。波茲曼的熵公式 $S = k \ln W$ 將熵與系統的微觀狀態數聯繫起來，但這引發了一個根本性的問題：如果微觀運動是確定性的（由牛頓力學決定），那麼宏觀的「不可逆性」是如何從微觀的可逆性中產生的？這個被稱為「洛施密特悖論」和「吉布斯悖論」的問題，至今仍是統計力學基礎研究中的熱門話題。儘管大多數物理學家接受了統計力學的實用價值，但當時對於原子和分子是否真實存在，物理學家們仍然存在激烈的爭論。化學家們早已使用原子論來解釋化學反應，但一些頂尖的物理學家，如恩斯特·馬赫，卻堅決反對原子假說，認為它只是一個有用的虛構概念。

## 1.2 邁克爾孫-莫雷實驗：光速的困惑

在這個看似完美的物理學體系中，第一個重大的裂縫出現在光學領域。十九世紀的物理學家普遍相信，存在一種稱為「以太」（ether）的無所不在的介質，它充滿整個宇宙空間，电磁波正是通過這種介質傳播的。以太的概念可以追溯到古希臘時期，但在十九世紀，它被賦予了新的科學內涵。為了解釋光在真空中的傳播，物理學家假設整個空間充滿了一種看不見、摸不著的介質，這種介質必須滿足以下條件：它是完全透明的，不影響光的傳播；它無處不在，充滿整個宇宙；它非常稀薄，以至於不會對行星運動產生可察覺的阻力；它是絕對靜止的，可以作為描述所有運動的絕對參考系。

根據這個以太假設，當地球在以太中運動時，沿不同方向傳播的光應該會有不同的相對速度。具體來說，如果地球相對於以太的速度為 $v$，那麼沿地球運動方向的光速應為 $c - v$（類似於在河流中順流而下），而垂直於運動方向的光速應為 $\sqrt{c^2 - v^2}$（這是速度向量合成的結果）。兩者之間的差異雖然很小，但原則上應該可以被精密的干涉儀檢測出來。

阿爾伯特·邁克爾孫（Albert A. Michelson）是一位極具才華的實驗物理學家，他設計了一台極其精密的干涉儀來測量光速的這種差異。邁克爾孫干涉儀的基本原理是：將一束光分成兩束相互垂直的光線，經過等距離傳播後再合併。由於兩束光的光程相同，任何微小的速度差異都會導致光程差，從而產生干涉條紋的移動。

1887年，邁克爾孫與愛德華·莫雷（Edward W. Morley）合作，在美國克利夫蘭進行了這個著名的實驗。他們預期會觀察到約 $10^{-2}$ 個條紋的移動，這在當時的技術條件下是完全可以測量的。然而，實驗結果令所有人震驚：無論地球處於軌道上的哪個位置，無論實驗在一年中的什麼時候進行，干涉條紋都沒有任何可測量的移動！

這個「負結果」對經典物理學產生了深遠的影響。它意味著：要么不存在所謂的以太，要么光速在所有慣性參考系中都是相同的。後來，愛因斯坦在1905年提出的狹義相對論中採納了第二種解釋，徹底顛覆了牛頓力學中的絕對時空觀。愛因斯坦認識到，不僅不存在絕對靜止的以太，更重要的是，光速 $c$ 是一個普適常數，在所有慣性參考系中都具有相同的數值。這一認識導致了時間和空間概念的根本變革——時間不再是絕對的、普適的，而是與空間一樣，取決於觀察者的運動狀態。

邁克爾孫-莫雷實驗在物理學史上具有里程碑意義。它不僅結束了以太假說的命運，更重要的是，它開啟了相對論的大門。愛因斯坦曾經說過，如果沒有邁克爾孫的實驗，相對論的發現可能會推遲很多年。這個實驗也成為了科學方法論的典範——它展示了如何通過精確的實驗來檢驗理論預言，即使這些預言看起來是多麼地不可思議。

## 1.3 黑體輻射災難：紫外方向的困惑

與以太問題並行發展的是另一個同樣深刻的危機，這就是著名的「黑體輻射」問題。這個問題不僅最終催生了量子論，更深刻地揭示了微觀世界運作方式的根本轉變。

所謂黑體，是指一種能夠完全吸收所有入射輻射的理想物體。現實中並不存在絕對的黑體，但一個帶有小孔的空腔可以被視為極佳的黑體近似：進入小孔的輻射在空腔內壁來回反射，最終會被完全吸收，幾乎沒有機會逃逸出來。當黑體被加熱時，它會發射出輻射，這就是所謂的「黑體輻射」。

實驗物理學家發現，黑體輻射的光譜只與溫度有關，與黑體的具體化學成分無關。這是一個非常神奇的規律：無論是用鐵、碳還是陶瓷製成的黑體，只要溫度相同，它們發出的輻射光譜是完全一樣的。這意味著存在某種普適的輻射規律，它只取決於系統的溫度，而與具體物質的詳細結構無關。

威廉·維恩（Wilhelm Wien）在1896年從經驗上發現了一個描述輻射光譜的公式，這就是後來被稱為「維恩近似」或「維恩分布」的表達式：

$$u(\nu, T) = \frac{8\pi h\nu^3}{c^3} e^{-h\nu/kT}$$

在這個公式中，$u(\nu, T)$ 表示頻率為 $\nu$ 的輻射在溫度 $T$ 下的能量密度，$h$ 是後來被命名為普朗克常數的常數，$c$ 是光速，$k$ 是波茲曼常數。維恩公式在高频（短波長）區域與實驗數據符合得很好，但在低頻（長波長）區域則明顯偏離實驗結果。

與此同時，Lord瑞利（John William Strutt, 3rd Baron Rayleigh）和詹姆斯·金斯（James Jeans）則從經典統計力學出發，試圖推導黑體輻射公式。根據經典物理學的能量均分定理，每一個电磁波模式都應該有平均能量 $kT$——這是因為在熱平衡時，每個自由度的平均能量都應該是 $\frac{1}{2}kT$，而电磁場有兩個自由度（電場和磁場），所以總平均能量是 $kT$。

讓我們詳細推導瑞利-金斯公式。在頻率 $\nu$ 到 $\nu + d\nu$ 之間，空間中允许的电磁波模式數為：

$$dN = \frac{8\pi V}{c^3} \nu^2 d\nu$$

這個結果來自於對邊界條件的考慮：如果我們假設一個邊長為 $L$ 的立方形空腔，那麼波動方程的解必須滿足邊界條件，這導致波數分量必須是 $\frac{2\pi n}{L}$ 的整數倍，其中 $n$ 是整數。經過計算，我們可以得到上述的模式密度公式。

因此，輻射能量密度為：

$$u(\nu, T) d\nu = \frac{8\pi kT}{c^3} \nu^2 d\nu$$

這就是著名的「瑞利-金斯定律」（Rayleigh-Jeans Law）。

瑞利-金斯公式在低頻（長波長）區域與實驗符合，但在高頻（短波長）區域則完全失效。根據這個公式，當頻率趨向無窮大時，輻射能量密度也會趨向無窮大：

$$u(\nu, T) \propto \nu^2 \quad \text{當} \quad \nu \to \infty$$

這顯然是荒謬的！實驗上從未觀察到這種「紫外災難」（ultraviolet catastrophe）。事實上，在非常高的頻率下，輻射能量趨向於零，而不是無窮大。維恩公式在高频區域有效，瑞利-金斯公式在低頻區域有效，但它們都無法同時描述整個光譜。

## 1.4 瑞利-金斯公式的詳細推導

讓我們更詳細地探討瑞利-金斯公式的推導過程，這對於理解後續的普朗克革命至關重要。

考慮一個邊長為 $L$ 的立方形空腔，內部充滿了處於熱平衡的电磁輻射。电磁場滿足波動方程：

$$\nabla^2 \mathbf{E} = \frac{1}{c^2} \frac{\partial^2 \mathbf{E}}{\partial t^2}$$

對於沿 $x$ 方向傳播的平面波，電場可以寫成：

$$E_x = E_0 \sin(k_x x) \sin(\omega t)$$

邊界條件要求電場在導體壁處為零，這導致 $k_x = \frac{n_x \pi}{L}$，其中 $n_x$ 是正整數。同樣地，$k_y = \frac{n_y \pi}{L}$，$k_z = \frac{n_z \pi}{L}$。

每組正整數 $(n_x, n_y, n_z)$ 對應於一個可能的波動模式。注意，對於每一組 $(n_x, n_y, n_z)$，實際上有兩個獨立的極化方向。因此，在波數空間中，半徑為 $k$ 到 $k+dk$ 的球殼內的模式數為：

$$dN = 2 \times \frac{1}{8} \times 4\pi k^2 dk \times \left(\frac{L}{\pi}\right)^3 = \frac{4\pi V}{(2\pi)^3} k^2 dk$$

其中 $V = L^3$ 是空腔體積，因子 $1/8$ 是因為我們只考慮 $n_x, n_y, n_z > 0$ 的區域。

利用關係 $k = \frac{2\pi \nu}{c}$，我們可以得到頻率間隔 $d\nu$ 內的模式數：

$$dN = \frac{8\pi V}{c^3} \nu^2 d\nu$$

這就是我們前面提到的結果。

每個模式的平均能量為 $kT$（根據經典能量均分定理），所以能量密度為：

$$u(\nu, T) = \frac{8\pi kT}{c^3} \nu^2$$

這就是瑞利-金斯公式。

## 1.5 經典物理學的根本困境

邁克爾孫-莫雷實驗與黑體輻射問題只是十九世紀末經典物理學面臨的眾多困難中的兩個。隨著物理學家對原子、分子與輻射現象的深入研究，越來越多的實驗現象無法用經典理論解釋：

第一個難題是光電效應。1887年，赫茲在發現电磁波的同時也觀察到，當照射某些金屬表面時會產生電火花。進一步的研究發現，只有當光的頻率超過某一臨界值時，才能觀測到電子逸出；而且逸出電子的最大動能與入射光的頻率成正比，而與光強無關。1905年，愛因斯坦提出了光子的概念，成功地解釋了這一現象。他假設光是由離散的「光子」組成的，每個光子的能量為 $E = h\nu$，這與經典的光波理論完全矛盾。

第二個難題是原子光譜。1885年，瑞士中學教師約翰·巴爾末（Johann Balmer）發現氫原子光譜可以用一個簡單的公式描述，這暗示了原子內部結構的量子化。巴爾末公式為：

$$\lambda = b \frac{n^2}{n^2 - 4}$$

其中 $b = 3645.6$ 埃，$n = 3, 4, 5, \ldots$。這個經驗公式的背後隱藏著什麼樣的物理原理？這個問題困擾了物理學家們很多年。

第三個難題是原子穩定性。根據經典電動力學，電子繞原子核運動時會不斷輻射电磁波，損失能量，最終會螺旋墜入原子核。但現實中的原子卻是穩定的，電子不會墜入原子核。這是經典物理學無法解釋的。

第四個難題是比熱問題。根據能量均分定理，雙原子分子的熱容量應該是 $\frac{7}{2}k$（平移動能 $\frac{3}{2}k$ + 轉動動能 $k$ + 振動動能 $\frac{3}{2}k$），但實驗測得的熱容量在低溫下遠小於這個值。這表明在低溫下，振動自由度被「冻结」了，但經典統計力學無法解釋這種冻结。

最後一個難題是 X 射線的產生。1895年，威廉·倫琴（Wilhelm Röntgen）發現了 X 射線，但這種射線的性質長期無法用經典理論解釋。

## 1.6 普朗克的突破：能量子假說

面對這些困境，馬克斯·普朗克（Max Karl Ernst Ludwig Planck，1858-1947）開始了對黑體輻射問題的系統研究。普朗克是一位德國理論物理學家，出生於基爾，他的科學生涯開始於熱力學研究。1879年，他提交了關於熱力學第二定律的博士論文，此後一直在熱力學與物理化學領域進行研究。在提出能量子假說之前，普朗克已經是熱輻射領域的權威專家。

1896年，普朗克開始系統地研究黑體輻射問題，並試圖從熱力學與电磁理論推導出維恩定律。經過數年的努力，他發現只有引入一個革命性的假設才能解釋實驗數據。1900年10月19日，普朗克在德國物理學會會議上提出了一個能夠完美擬合實驗數據的經驗公式。兩個月後，即1900年12月14日，他在一篇論文中給出了這個公式的理論推導，這一天被認為是量子理論的誕生日。

普朗克的推導過程如下：首先，他假設黑體腔壁由大量的「振子」組成，每個振子具有一定的頻率 $\nu$，可以與輻射場交換能量。根據經典統計力學，在熱平衡時振子的平均能量應該由麥克斯韋-玻爾茲曼分佈決定。但普朗克發現，如果繼續使用經典統計力學，無法得到與實驗一致的結果。

於是，他提出了一個革命性的假設：**振子只能以離散的「能量子」的形式吸收或發射能量**。假設每個頻率為 $\nu$ 的振子，其能量只能是基本能量單元 $h\nu$ 的整數倍：

$$E_n = n h\nu, \quad n = 0, 1, 2, 3, \ldots$$

其中 $h$ 是一個新的自然常數，後來被稱為「普朗克常數」。其數值為：

$$h = 6.62607015 \times 10^{-34} \text{ J·s}$$

這個常數的發現標誌著物理學的一個新時代的開始。

## 1.7 經典物理學危機的深遠意義

經典物理學的這些危機預示著一場物理學革命的來臨。1900年12月14日，普朗克在德國物理學會發表演講，提出了能量子的概念，這一天被認為是量子理論的誕生日。

普朗克的突破不僅解決了黑體輻射的問題，更重要的是，它開創了一種全新的物理學思維方式。能量子的概念表明，在某些情況下，物理量可能是離散的，而不是連續的。這種「量子化」的思想後來被推廣到物理學的各個領域，從原子結構到固體物理，從統計力學到量子場論。

同時，邁克爾孫-莫雷實驗的「負結果」也催生了愛因斯坦的相對論。1905年，愛因斯坦發表了著名的狹義相對論，提出了光速不變原理和相對性原理，完全顛覆了牛頓的絕對時空觀。

這兩大革命——相對論和量子論——共同構成了現代物理學的兩大支柱。相對論主要處理高速運動和強引力場的問題，而量子論則處理微觀世界的問題。兩者的結合產生了量子場論，這是當代理論物理學的最高成就。

## 1.8 本章小結

本章我們回顧了十九世紀末經典物理學面臨的危機。邁克爾孫-莫雷實驗否定了以太的存在，揭示了光速的絕對性；黑體輻射的「紫外災難」暴露了經典統計力學的根本缺陷；光電效應、原子光譜、原子穩定性等問題也無法用經典理論解釋。這些實驗與理論的困境促使物理學家重新思考光與物質的本質，最終催生了量子理論。在下一章中，我們將詳細探討普朗克的能量子假說及其歷史意義。

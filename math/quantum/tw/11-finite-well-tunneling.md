# 第十一章：有限勢阱與隧穿效應——量子世界的神奇現象

## 11.1 從無限到有限：更真實的模型

在前一章中，我們討論了無限勢阱——粒子被限制在勢能無窮大的盒壁內。在現實世界中，沒有真正的無窮大勢能。有限勢阱是一個更真實的模型，它讓我們看到量子力學一些更加引人入勝的現象。

有限勢阱與無限勢阱的本質區別在於：粒子的波函數可以穿透阱壁進入經典物理學中「禁止」進入的區域。這種被稱為「隧穿」的特殊量子效應，在宏觀世界是不可想像的，但在微觀世界中卻是普遍存在且至關重要的。

## 11.2 有限勢阱的數學表述

**有限勢阱**（Finite Potential Well）的勢能函數定義為：

$$V(x) = \begin{cases} 0 & |x| < a \\ V_0 & |x| \geq a \end{cases}$$

這是一個對稱的有限勢阱，深度為 $V_0$，寬度為 $2a$。

這個模型的特點是：
1. 在 $|x| < a$ 區域（阱內），勢能為零
2. 在 $|x| \geq a$ 區域（阱外），勢能為常數 $V_0$
3. 阱壁是有限高度的，而不是無窮高的

我們將考慮兩種情況：
- **束縛態**：$0 < E < V_0$，粒子被約束在阱內
- **游離態**：$E > V_0$，粒子可以離開勢阱

本章將重點討論束縛態，因為它是有限勢阱最有趣的量子效應的來源。

## 11.3 分區求解薛丁格方程式

讓我們分區求解束縛態的薛丁格方程式。

### 11.3.1 區域劃分

將空間劃分為三個區域：
- 區域 I：$x < -a$
- 區域 II：$-a < x < a$（阱內）
- 區域 III：$x > a$

### 11.3.2 區域 II（阱內）

在阱內，$V = 0$，薛丁格方程式為：

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} = E\psi$$

整理得：

$$\frac{d^2\psi}{dx^2} + k^2\psi = 0$$

其中：

$$k = \sqrt{\frac{2mE}{\hbar^2}}$$

這與無限勢阱的阱內方程相同。通解為：

$$\psi_{II}(x) = A\sin(kx) + B\cos(kx)$$

### 11.3.3 區域 I 和 III（阱外）

在阱外，$V = V_0$，薛丁格方程式為：

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} + V_0\psi = E\psi$$

整理得：

$$\frac{d^2\psi}{dx^2} - \kappa^2\psi = 0$$

其中：

$$\kappa = \sqrt{\frac{2m(V_0 - E)}{\hbar^2}}$$

這是一個指數衰減方程。通解為：

$$\psi_I(x) = Ce^{\kappa x} + De^{-\kappa x}$$
$$\psi_{III}(x) = Fe^{\kappa x} + Ge^{-\kappa x}$$

## 11.4 邊界條件的應用

讓我們應用邊界條件來確定波函數的形式。

### 11.4.1 束縛態的邊界條件

對於束縛態，要求波函數在無窮遠處為零：

$$\lim_{x \to -\infty} \psi_I(x) = 0, \quad \lim_{x \to +\infty} \psi_{III}(x) = 0$$

這要求：
- 在區域 I：$D = 0$（因為 $e^{-\kappa x}$ 在 $x \to -\infty$ 時發散）
- 在區域 III：$F = 0$（因為 $e^{\kappa x}$ 在 $x \to +\infty$ 時發散）

因此：

$$\psi_I(x) = Ce^{\kappa x} \quad (x < -a)$$
$$\psi_{III}(x) = Ge^{-\kappa x} \quad (x > a)$$

### 11.4.2 連續性邊界條件

在勢能跳躍的界面上（$x = \pm a$），波函數及其一階導數必須連續：

$$\psi_I(-a) = \psi_{II}(-a)$$
$$\psi_I'(-a) = \psi_{II}'(-a)$$
$$\psi_{II}(a) = \psi_{III}(a)$$
$$\psi_{II}'(a) = \psi_{III}'(a)$$

## 11.5 對稱與反對稱解

由於勢阱是對稱的，我們可以利用宇稱來簡化問題。

### 11.5.1 偶宇稱解

偶宇稱（偶函數）波函數滿足 $\psi(-x) = \psi(x)$。

在阱內：
$$\psi_{II}^{(e)}(x) = A\cos(kx)$$

在阱外：
$$\psi_I^{(e)}(x) = Ce^{\kappa x}$$
$$\psi_{III}^{(e)}(x) = Ce^{-\kappa x}$$

在 $x = \pm a$ 處應用邊界條件，得到：

$$A\cos(ka) = Ce^{-\kappa a}$$
$$-Ak\sin(ka) = C\kappa e^{-\kappa a}$$

由第二式除以第一式：

$$-k\tan(ka) = \kappa$$

或者等價地：

$$\tan(ka) = -\frac{k}{\kappa}$$

### 11.5.2 奇宇稱解

奇宇稱（奇函數）波函數滿足 $\psi(-x) = -\psi(x)$。

在阱內：
$$\psi_{II}^{(o)}(x) = B\sin(kx)$$

在阱外：
$$\psi_I^{(o)}(x) = Ce^{\kappa x}$$
$$\psi_{III}^{(o)}(x) = -Ce^{-\kappa x}$$

應用邊界條件得到：

$$\frac{k}{\kappa}\tan(ka) = -1$$

或者等價地：

$$\cot(ka) = \frac{k}{\kappa}$$

## 11.6 能量量子化條件

讓我們整理能量量子化條件。

定義無量綱參數：

$$u = ka = a\sqrt{\frac{2mE}{\hbar^2}}$$

$$v = \kappa a = a\sqrt{\frac{2m(V_0 - E)}{\hbar^2}}$$

由能量關係：
$$u^2 + v^2 = \frac{2mV_0 a^2}{\hbar^2} \equiv \alpha^2$$

**偶宇稱方程**：
$$\tan u = -\frac{u}{v}$$

**奇宇稱方程**：
$$\cot u = \frac{u}{v}$$

這兩個方程的解決定了系統的能級。

## 11.7 能量本徵值的圖形求解

能量本徵值方程通常需要數值求解。我們可以使用圖形法來理解解的結構。

### 11.7.1 圖形法

考慮函數圖像：
- $y = \tan u$（偶宇稱）
- $y = \cot u$（奇宇稱）
- $y = -u/v = -\sqrt{\frac{E}{V_0 - E}}$（與能量有關）

兩個圖像的交點給出能量本徵值。

### 11.7.2 解的個數

對於給定的勢阱深度 $V_0$ 和寬度 $a$，存在有限的束縛態數目。這個數目由參數 $\alpha = a\sqrt{2mV_0/\hbar^2}$ 決定。

讓我們分析可能的解：
- 當 $\alpha < \pi/2$ 時，最多只有一個束縛態（總是偶宇稱）
- 當 $\pi/2 < \alpha < \pi$ 時，最多有兩個束縛態
- 當 $n\pi < \alpha < (n+1)\pi/2$ 時，最多有 $2n+1$ 個束縛態

### 11.7.3 與無限勢阱的比較

在無限勢阱中，總是有無限多個束縛態。在有限勢阱中，束縛態的數目是有限的。當 $V_0 \to \infty$ 時，有限勢阱趨於無限勢阱，束縛態數目趨於無窮多。

## 11.8 隧穿效應

**隧穿效應**（Tunneling Effect）是量子力學最神奇、最重要的現象之一。

### 11.8.1 什麼是隧穿

考慮一個有限厚度的勢壘：

$$V(x) = \begin{cases} 0 & x < 0 \\ V_0 & 0 < x < a \\ 0 & x > a \end{cases}$$

設入射粒子能量 $E < V_0$。

在經典物理學中，如果粒子的能量低於勢壘高度，它就無法越過勢壘。但！在量子力學中，粒子有一定的概率「穿過」勢壘。

### 11.8.2 隧穿的物理解釋

讓我們分析波函數在勢壘區域的行為。

在勢壘區域（$0 < x < a$），波函數的形式為：

$$\psi(x) = Ce^{\kappa x} + De^{-\kappa x}$$

其中 $\kappa = \sqrt{2m(V_0 - E)}/\hbar$。

波函數在勢壘內是指數衰減的，而不是完全為零。這意味著在勢壘內部，粒子有一定的出現概率！

當波函數到達勢壘的另一側時，它仍然是有限值，因此有一定的概率透射過去。

### 11.8.3 隧穿的數學描述

讓我們計算隧穿概率。

考慮一個粒子從左側入射。我們有：
- 入射波：$e^{ikx}$
- 反射波：$Re^{-ikx}$
- 透射波：$Te^{ikx}$

其中 $k = \sqrt{2mE}/\hbar$。

利用邊界條件，可以計算反射係數 $R$ 和透射係數 $T$。

## 11.9 穿透係數的計算

**穿透係數**（Transmission Coefficient）或**隧穿機率**定義為透射粒子流與入射粒子流之比：

$$T = \frac{|T|^2}{|A_{\text{in}}|^2}$$

### 11.9.1 方形势壘的結果

對於方形勢壘，可以計算得到：

$$T = \frac{1}{1 + \frac{V_0^2 \sinh^2(\kappa a)}{4E(V_0 - E)}}$$

### 11.9.2 厚勢壘近似

當 $k_2 a \gg 1$（厚勢壘或高勢壘）時，$\sinh(\kappa a) \approx \frac{1}{2}e^{\kappa a}$，因此：

$$T \approx \frac{16E(V_0 - E)}{V_0^2} e^{-2\kappa a}$$

或者：

$$T \approx \frac{16E(V_0 - E)}{V_0^2} \exp\left(-\frac{2a}{\hbar}\sqrt{2m(V_0 - E)}\right)$$

這個結果非常重要：穿透係數隨勢壘寬度 $a$ 指數衰減！

### 11.9.3 隧穿的指數依賴

隧穿概率對勢壘參數非常敏感：
- 勢壘越寬，隧穿概率越小
- 勢壘越高，隧穿概率越小
- 粒子質量越大，隧穿概率越小

## 11.10 隧穿效應的實際應用

隧穿效應在物理學和技術中有許多重要應用。

### 11.10.1 α衰變

放射性α粒子的衰變可以用隧穿效應解釋。α粒子在原子核內受到很強的吸引力（核力），但必須隧穿過一個由庫侖排斥力形成的勢壘才能逃逸出去。

這個理論由喬治·伽莫夫（George Gamow）在1928年提出，成功解釋了α衰變的指數規律。

### 11.10.2 掃描隧道顯微鏡（STM）

**掃描隧道顯微鏡**（Scanning Tunneling Microscope，STM）是利用隧穿效應的最著名應用之一。

STM的工作原理：
- 一個非常尖銳的金屬探針接近樣品表面
- 當距離足夠近時，電子可以隧穿過真空勢壘
- 隧穿電流與探針-樣品距離高度敏感
- 通過測量電流，可以獲得原子級分辨率的表面形貌

STM的分辨率可達原子尺度，這是經典顯微鏡無法比擬的。

### 11.10.3 半導體異質結

在半導體物理中，電子可以隧穿過pn結中的勢壘。這是以下技術的基礎：
- **隧道二極體**：利用隧穿效應的特殊半導體器件
- **量子阱雷射**：電子隧穿進入量子阱區域
- **共振隧穿二極體**：利用共振隧穿效應

### 11.10.4 超導 Josephson 結

超導電子對可以隧穿過絕緣層，形成 Josephson 效應。這是以下技術的基礎：
- **超導量子干涉儀（SQUID）**：極其靈敏的磁場探測器
- **超導量子位元**：量子計算機的核心元件

## 11.11 有限與無限勢阱的比較

讓我們總結有限勢阱與無限勢阱的主要區別。

| 特性 | 無限勢阱 | 有限勢阱 |
|------|----------|----------|
| 束縛態數量 | 無限多 | 有限多，取決於 $V_0$ 與 $a$ |
| 基態能量 | $E_1 = \frac{\pi^2\hbar^2}{2ma^2}$ | 略高於無限阱 |
| 波函數 | 邊界處嚴格為零 | 邊界外指數衰減 |
| 零點能 | 非零 | 非零（更大） |
| 宇稱 | $\psi_n$ 有確定的宇稱 | 有偶宇稱與奇宇稱解 |
| 隧穿 | 無（波函數在阱外為零） | 有（波函數可滲透到阱外） |

## 11.12 有限勢阱模型的推廣

有限勢阱模型可以推廣到更複雜的情況。

### 11.12.1 非對稱勢阱

實際的勢阱往往不是對稱的。這時問題更加複雜，但基本方法是相同的。

### 11.12.2 多勢阱系統

當多個勢阱靠近時，相鄰阱中的波函數會發生重疊，導致能級分裂。這是分子軌道理論和固體能帶理論的基礎。

### 11.12.3 有限深量子阱

在半導體物理中，電子被限制在由不同半導體材料形成的異質結構中，形成有限深的量子阱。這些結構在光電子器件中有廣泛應用。

## 11.13 本章小結

本章我們討論了有限勢阱與隧穿效應。有限勢阱中能量量子化，束縛態數量有限，波函數在阱外指數衰減。隧穿效應是量子力學特有的現象：粒子可以穿過經典力學中不可逾越的勢壘。穿透係數隨勢壘寬度指數衰減。隧穿效應在α衰變、掃描隧道顯微鏡、半導體器件、超導Josephson結等領域有重要應用。

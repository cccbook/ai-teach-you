# 第三十八章：變分法

## 38.1 變分原理的物理基礎

變分原理是物理學中最優美、最普適的理論框架之一。從費馬原理到最小作用量原理，從古典力學到量子場論，變分原理無處不在。在量子力學中，變分原理提供了一種強大的近似方法，用於估計系統的能量和波函數。

變分原理的基本思想是：**對於一個保守系統，真實的運動軌跡使得作用量泛函取得極值**。這個原理的量子力學版本是：對於一個量子系統，其基態波函數使得能量期望值取得全域最小值。

變分法的核心是瑞利-里茲變分原理（Rayleigh-Ritz Variational Principle）。對於一個給定的哈密頓量$H$，對於任意歸一化的試探波函數$|\psi\rangle$，能量的期望值為：

$$E[\psi] = \frac{\langle\psi|H|\psi\rangle}{\langle\psi|\psi\rangle} \geq E_0$$

其中$E_0$是系統的基態能量。等號成立的條件是當且僅當$|\psi\rangle$是基態（或與基態簡並的態）。

這個原理的證明非常簡單：將任意試探波函數在$H$的本徵態基底下展開。如果基態能量為$E_0$，其他本徵值為$E_1, E_2, \ldots$，則：

$$\langle\psi|H|\psi\rangle = \sum_n E_n |c_n|^2 \geq E_0 \sum_n |c_n|^2 = E_0 \langle\psi|\psi\rangle$$

因此，$E[\psi] \geq E_0$。

## 38.2 變分法的基本步驟

變分法的實施包含以下步驟：

1. **選擇試探波函數**：根據物理直覺選擇一組帶有變分參數的試探波函數$\psi(\mathbf{r}; \alpha_1, \alpha_2, \ldots)$。試探波函數應滿足問題的對稱性和邊界條件。

2. **計算能量期望值**：計算能量的期望值$E(\alpha_1, \alpha_2, \ldots) = \langle\psi|H|\psi\rangle / \langle\psi|\psi\rangle$。

3. **極小化能量**：對變分參數求偏導並令其為零：
$$\frac{\partial E}{\partial \alpha_i} = 0, \quad i = 1, 2, \ldots$$

4. **求解參數**：解這些方程得到最優參數值$\alpha_i^*$。

5. **估計結果**：將最優參數代回能量表達式，得到基態能量的上界$E_{var} = E(\alpha_1^*, \alpha_2^*, \ldots)$。

變分法的精度取決於試探波函數的選擇。一個好的試探波函數應該：
- 具有正確的對稱性
- 滿足邊界條件
- 包含足夠的靈活性（可變參數）
- 在物理直覺上合理

## 38.3 氫原子基態的變分計算

讓我們用氫原子說明變分法的應用。氫原子的哈密頓量為：

$$H = -\frac{\hbar^2}{2m}\nabla^2 - \frac{e^2}{4\pi\epsilon_0 r}$$

我們知道氫原子基態的精確波函數為$\psi_0(r) = \frac{1}{\sqrt{\pi a_0^3}} e^{-r/a_0}$，其中$a_0$是玻爾半徑。現在用變分法來估計基態能量。

選擇試探波函數：
$$\psi(r; \alpha) = \frac{1}{\sqrt{\pi \alpha^3}} e^{-r/\alpha}$$

這個波函數的形式與真實基態相同，但允許長度尺度$\alpha$變化。當$\alpha = a_0$時，我們得到精確基態。

計算動能期望值：
$$T = -\frac{\hbar^2}{2m} \int \psi^* \nabla^2 \psi d^3r = \frac{\hbar^2}{2m\alpha^2}$$

計算勢能期望值：
$$V = -\frac{e^2}{4\pi\epsilon_0} \int \frac{|\psi|^2}{r} d^3r = -\frac{e^2}{4\pi\epsilon_0 \alpha}$$

總能量：
$$E(\alpha) = T + V = \frac{\hbar^2}{2m\alpha^2} - \frac{e^2}{4\pi\epsilon_0 \alpha}$$

對$\alpha$求導並令其為零：
$$\frac{dE}{d\alpha} = -\frac{\hbar^2}{m\alpha^3} + \frac{e^2}{4\pi\epsilon_0 \alpha^2} = 0$$

解得$\alpha = \frac{4\pi\epsilon_0 \hbar^2}{me^2} = a_0$，代回得到：
$$E = -\frac{me^4}{2(4\pi\epsilon_0)^2\hbar^2} = -13.6 \text{ eV}$$

這正是氫原子基態的精確能量！變分法在這個情況下給出了精確結果，因為我們選擇了包含真實基態的試探波函數族。

## 38.4 氦原子基態的變分計算

氦原子是多電子原子中最簡單的系統，包含兩個電子和一個核。儘管看似簡單，氦原子是個三體問題，沒有精確解析解。變分法是處理氦原子問題的主要工具之一。

氦原子的哈密頓量為（原子單位）：
$$H = -\frac{1}{2}\nabla_1^2 - \frac{1}{2}\nabla_2^2 - \frac{Z}{r_1} - \frac{Z}{r_2} + \frac{1}{r_{12}}$$

其中$Z=2$是原子序數，$r_1$和$r_2$是電子到核的距離，$r_{12}$是兩個電子之間的距離。

作為第一近似，忽略電子間相互作用$(1/r_{12})$，每個電子的波函數如同氫類離子$He^+$中的電子：
$$\psi_0(\mathbf{r}) = \left(\frac{Z^3}{\pi a_0^3}\right)^{1/2} e^{-Zr/a_0}$$

總能量為兩個電子能量之和：$2 \times (-2 \times 13.6 \text{ eV}) = -108.8$ eV。但實驗值約為$-79.0$ eV。差異約為30 eV，正好來自電子間排斥。

考慮電子排斥的一個簡單方法是引入有效核電荷$Z_{eff}$。設試探波函數：
$$\psi(\mathbf{r}_1, \mathbf{r}_2) = \frac{Z_{eff}^3}{\pi a_0^3} e^{-Z_{eff}(r_1+r_2)/a_0}$$

計算能量並對$Z_{eff}$優化，得到$Z_{eff} = Z - 5/16$。對於氦，$Z_{eff} = 2 - 5/16 = 27/16$。

能量期望值：
$$E(Z_{eff}) = -(Z_{eff}^2 - Z_{eff}/2) \times 4 \times 13.6 \text{ eV}$$

代入$Z_{eff} = 27/16$，得到$E = -77.5$ eV，與實驗值$-79.0$ eV相當接近。

## 38.5 線性變分法

線性變分法是將試探波函數寫成一組基函數的線性組合：

$$|\psi\rangle = \sum_{i=1}^N c_i |\phi_i\rangle$$

變分原理給出廣義本徵值方程式：

$$\mathbf{H}\mathbf{c} = E\mathbf{S}\mathbf{c}$$

其中$H_{ij} = \langle\phi_i|H|\phi_j\rangle$是哈密頓矩陣元，$S_{ij} = \langle\phi_i|\phi_j\rangle$是重疊矩陣。

這個方程式可以通過標準線性代數方法求解，得到$N$個能量估計值$E_1 \leq E_2 \leq \cdots \leq E_N$。其中$E_1$是基態能量的最好估計，$E_k$是第$k$態能量的估計。

線性變分法的一個重要特例是赫茲-拉東基爾方法（Hartree-Fock方法），其中基函數是自洽決定的。

## 38.6 氫分子離子$H_2^+$的線性變分法

氫分子離子$H_2^+$是兩個質子和一個電子的系統。這是分子系統中最簡單的系統之一，可以用線性變分法處理。

我們使用線性組合原子軌跡（Linear Combination of Atomic Orbitals, LCAO）方法。設兩個質子位於$A$和$B$，電子在$A$和$B$的原子軌道上：
$$|\psi\rangle = c_A |1s_A\rangle + c_B |1s_B\rangle$$

構建哈密頓矩陣：
$$\begin{pmatrix} H_{AA} & H_{AB} \\ H_{BA} & H_{BB} \end{pmatrix}$$

其中$H_{AA} = H_{BB}$（由於對稱性），$H_{AB} = H_{BA}$是交換積分。

求解廣義本徵值問題，得到兩個能量本徵值：
- 鍵合態：兩個原子軌道同相位疊加，能量低於原子能量
- 反鍵合態：兩個原子軌道反相位疊加，能量高於原子能量

這個結果與分子軌道理論完全一致。鍵合態對應於分子能級低於原子能級，因此可以形成穩定分子。

## 38.7 氫分子的變分計算

氫分子$H_2$包含兩個電子和兩個質子。這是一個四體問題，比$H_2^+$更複雜，但仍可用變分法處理。

海特勒-倫敦（Heitler-London）方法是處理$H_2$的早期方案。設兩個電子分佈在兩個質子上：
$$\psi(\mathbf{r}_1, \mathbf{r}_2) = \psi_A(\mathbf{r}_1)\psi_B(\mathbf{r}_2) + \psi_B(\mathbf{r}_1)\psi_A(\mathbf{r}_2)$$

這個波函數對電子交換是反對稱的（總波函數需要反對稱），自旋部分取反平行單態。

計算能量隨核間距$R$的變化，發現：
- 當$R \to \infty$時，能量趨於兩個分離氫原子的能量
- 當$R \approx 1.4a_0$時，能量達到最小值
- 這給出鍵合能約為3.14 eV，接近實驗值4.75 eV

更精確的計算需要考慮更多配置（C和大小的標量因子），但變分原理的框架不變。

## 38.8 蒙特卡羅變分法

變分蒙特卡羅（Variational Monte Carlo, VMC）方法將變分原理與蒙特卡羅積分相結合，可以用於處理更複雜的多電子系統。

VMC的基本步驟：
1. 選擇試探波函數$\psi_T(\mathbf{R})$，通常包含大量可優化參數
2. 使用 Metropolis 演算法對局部能量加權的波函數分佈$\{|\psi_T|^2\}$進行採樣
3. 計算能量期望值$E = \langle\psi_T|H|\psi_T\rangle / \langle\psi_T|\psi_T\rangle$
4. 使用梯度或其他優化方法調整參數以最小化$E$

VMC的優勢是可以處理更複雜的試探波函數，包括：
- Slater行列式（來自Hartree-Fock）
- Jastrow因子（描述電子-電子關聯）
- 多次 Slater行列式（CIT或MCSCF類型）

VMC已經成為量子蒙特卡羅家族方法的基礎，被廣泛應用於凝聚態物理和量子化學。

## 38.9 擴散蒙特卡羅方法

擴散蒙特卡羅（Diffusion Monte Carlo, DMC）方法在VMC的基礎上，通過求解薛丁格方程式的投影來獲得更精確的能量估計。

DMC的核心思想是：對於虛時間$\tau$，薛丁格方程式的形式解為：
$$\psi(\mathbf{R}, \tau) = e^{-(H-E_T)\tau/\hbar} \psi(\mathbf{R}, 0)$$

其中$E_T$是試探能量。當$\tau \to \infty$時，$\psi$趨近於基態波函數（假設$\psi(\mathbf{R}, 0)$與基態有非零重疊）。

DMC方法可以獲得基態能量的精確估計（忽略固定節點近似）。對於電子系統，這是少數可以處理大規模多電子系統並獲得高精確度結果的方法之一。

## 38.10 密度泛函理論中的變分原理

密度泛函理論（Density Functional Theory, DFT）是當代量子化學和凝聚態物理最重要的計算方法之一。DFT的基礎是兩個數學定理：

**第一Hohenberg-Kohn定理**：對於外部勢場中的相互作用電子系統，基態電子密度$n(\mathbf{r})$唯一確定勢場，從而唯一確定系統的所有性質（包括基態能量）。

**第二Hohenberg-Kohn定理**：定義能量泛函$E[n]$，對於任意密度$n(\mathbf{r})$有$E[n] \geq E_0$，其中$E_0$是真實基態能量。

DFC的變分原理是：真實基態密度使得能量泛函取得最小值。這為自洽求解Kohn-Sham方程式提供了基礎。

Kohn-Sham方程式將互動電子系統映射到非相互作用電子系統：
$$-\frac{\hbar^2}{2m}\nabla^2 + V_{eff}(\mathbf{r}) \phi_i(\mathbf{r}) = \epsilon_i \phi_i(\mathbf{r})$$

其中$V_{eff}$包括外部勢、Hartree勢和交換-相關勢。

## 38.11 本章小結

本章我們系統介紹了變分原理及其在量子力學中的應用。變分原理的核心是瑞利-里茲變分原理：對於任意試探波函數，能量期望值給出基態能量的上界。

我們從氫原子基態的簡單例子出發，展示了變分法的基本步驟。然後我們將方法應用於氦原子基態、氫分子離子和氫分子的計算，展示了變分法在多電子系統中的威力。

線性變分法將問題轉化為矩陣本徵值問題，是處理複雜系統的有效工具。變分蒙特卡羅和擴散蒙特卡羅方法將變分原理與蒙特卡羅採樣相結合，可以處理更大型的系統。

最後，我們簡要介紹了密度泛函理論中的變分原理，這是當代計算材料科學的基石。

變分法是連接解析近似和數值計算的橋樑，是研究人員處理實際物理問題不可或缺的工具。

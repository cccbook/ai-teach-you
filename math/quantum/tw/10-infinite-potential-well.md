# 第十章：無限勢阱——量子化的起源

## 10.1 無限勢阱問題的物理背景

**無限勢阱**（Infinite Potential Well）是量子力學中最基本也是最重要的模型之一。這個模型描述了一個被無限高勢能牆限制在一維區間內的粒子。

這個模型在物理學中具有重要地位，因為：
1. 它是最簡單的可解量子系統之一
2. 它展示了量子化的本質特徵
3. 它為理解更複雜的勢阱問題提供了基礎
4. 它與許多實際物理問題（如半導體量子阱、原子核殼層模型）密切相關

在繼續之前，讓我們澄清一下這個模型的物理意義。假設粒子被限制在一個盒子中，盒壁是完全不可穿透的。在經典物理學中，粒子在盒子內自由運動，以各種速度反彈。這是個簡單的問題，沒有量子化，能量可以是任意值。但在量子力學中，情況完全不同。

## 10.2 問題的數學表述

讓我們精確定義無限勢阱問題。

勢能函數為：

$$V(x) = \begin{cases} 0 & 0 < x < a \\ \infty & x \leq 0 \text{ 或 } x \geq a \end{cases}$$

物理上，這相當於粒子被限制在一個「盒子」中，盒壁的勢能為無窮大，粒子無法穿過盒壁。

另一種常見的定義是對稱勢阱：

$$V(x) = \begin{cases} 0 & |x| < a \\ \infty & |x| \geq a \end{cases}$$

兩種定義在本質上相同，只是坐標原點的位置不同。我們將使用第一種定義。

## 10.3 求解薛丁格方程式

讓我們在阱內（$0 < x < a$）求解薛丁格方程式。

### 10.3.1 阱內的方程式

在阱內，$V = 0$，薛丁格方程式為：

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} = E\psi$$

整理得：

$$\frac{d^2\psi}{dx^2} + k^2\psi = 0$$

其中我們定義：

$$k = \sqrt{\frac{2mE}{\hbar^2}}$$

注意這裡 $E > 0$ 是能量（因為在阱內動能為正）。

### 10.3.2 通解

這個二階線性微分方程的一般解為：

$$\psi(x) = A\sin(kx) + B\cos(kx)$$

其中 $A$ 和 $B$ 是待定常數，由邊界條件和歸一化條件決定。

或者，我們也可以使用正弦和餘弦的組合來表示：

$$\psi(x) = C\sin(kx + \phi)$$

其中 $C$ 是振幅，$\phi$ 是相位。

### 10.3.3 邊界條件

邊界條件是這個問題的關鍵。

由於阱外勢能為無窮大，粒子不可能出現在阱外，因此波函數必須在邊界處為零：

$$\psi(0) = 0, \quad \psi(a) = 0$$

這些邊界條件稱為**第一類邊界條件**或**狄利克雷邊界條件**。

## 10.4 邊界條件與量子化

讓我們應用邊界條件來確定允許的波函數和能量。

### 10.4.1 第一個邊界條件

由 $\psi(0) = 0$ 可得：

$$A\sin(0) + B\cos(0) = 0 + B \cdot 1 = B = 0$$

因此餘弦項必須為零：

$$\psi(x) = A\sin(kx)$$

這表明波函數是正弦函數。

### 10.4.2 第二個邊界條件

由 $\psi(a) = 0$ 可得：

$$A\sin(ka) = 0$$

我們需要這個方程對非零 $A$ 成立（$A = 0$ 給出平凡解，物理上無意義），因此：

$$\sin(ka) = 0$$

正弦函數為零的條件是：

$$ka = n\pi, \quad n = 1, 2, 3, \ldots$$

注意這裡的 $n$ 從1開始，而不是從0開始。如果 $n = 0$，則 $k = 0$，波函數 $\psi(x) = A\sin(0) = 0$，這是平凡解，不是物理上有效的波函數。

### 10.4.3 能量量子化

由 $ka = n\pi$，我們得到：

$$k_n = \frac{n\pi}{a}, \quad n = 1, 2, 3, \ldots$$

由 $k = \sqrt{2mE/\hbar^2}$，可得能量本徵值：

$$E_n = \frac{\hbar^2 k_n^2}{2m} = \frac{n^2\pi^2\hbar^2}{2ma^2}, \quad n = 1, 2, 3, \ldots$$

這就是**能量量子化**！粒子只能具有這些離散的能量值。

讓我們引入一個有用的常數：

$$E_0 = \frac{\pi^2\hbar^2}{2ma^2}$$

這是基態能量（$n=1$），可以寫為：

$$E_n = n^2 E_0$$

## 10.5 定態波函數

讓我們確定歸一化的定態波函數。

由邊界條件給出的波函數形式為：

$$\psi_n(x) = A_n \sin\left(\frac{n\pi x}{a}\right), \quad n = 1, 2, 3, \ldots$$

### 10.5.1 歸一化

歸一化條件要求：

$$\int_0^a |\psi_n(x)|^2 dx = 1$$

計算這個積分：

$$\int_0^a \sin^2\left(\frac{n\pi x}{a}\right) dx = \frac{a}{2}$$

因此：

$$|A_n|^2 \cdot \frac{a}{2} = 1 \Rightarrow A_n = \sqrt{\frac{2}{a}}$$

我們可以選擇 $A_n$ 為正實數（因為整體相位在物理上是無關的）。

### 10.5.2 最終的波函數

歸一化的定態波函數為：

$$\psi_n(x) = \sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right), \quad n = 1, 2, 3, \ldots$$

讓我們驗證歸一化：

$$\int_0^a \left|\sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right)\right|^2 dx = \frac{2}{a} \cdot \frac{a}{2} = 1$$

## 10.6 基態能量與零點能

讓我們仔細分析基態能量。

最低能量（基態）為：

$$E_1 = \frac{\pi^2\hbar^2}{2ma^2}$$

注意這**不是零**！這與經典物理學完全不同。

在經典物理學中，粒子在阱中可以處於靜止狀態，能量為零。但在量子力學中，即使在基態，粒子也有非零能量，這稱為**零點能**（zero-point energy）。

零點能的存在是不確定性原理的直接後果。讓我們理解為什麼：

如果粒子靜止在某個確定位置，則位置不確定性 $\Delta x = 0$。根據不確定性原理，動量不確定性必須無限大：$\Delta p \to \infty$。動能與 $\Delta p^2$ 成正比，因此能量也會趨向無窮大。所以粒子不可能完全靜止在某個位置，必須有非零的動能。

從數學上說，零點能來自於波函數必須連續且歸一化的要求。將波函數限制在有限區間內，其二階導數必須非零，因此動能也不為零。

## 10.7 激發態能量

讓我們分析激發態的能量。

激發態能量與量子數的平方成正比：

$$E_n = n^2 E_1, \quad n = 2, 3, 4, \ldots$$

具體來說：
- 基態：$E_1 = E_0$
- 第一激發態：$E_2 = 4E_0$
- 第二激發態：$E_3 = 9E_0$
- 第三激發態：$E_4 = 16E_0$

相鄰能級之間的間隔：

$$\Delta E_{n \to n+1} = E_{n+1} - E_n = [(n+1)^2 - n^2]E_0 = (2n+1)E_0 = \frac{(2n+1)\pi^2\hbar^2}{2ma^2}$$

這個間隔隨 $n$ 增大而增大。

### 10.7.1 宏觀與微觀尺度的比較

對於宏觀尺度的「盒子」（例如 $a = 1$ cm），讓我們計算能量間隔。

電子的質量 $m_e = 9.109 \times 10^{-31}$ kg，$\hbar = 1.055 \times 10^{-34}$ J·s。

$$E_0 = \frac{\pi^2 \times (1.055 \times 10^{-34})^2}{2 \times 9.109 \times 10^{-31} \times (10^{-2})^2} \approx 6 \times 10^{-34} \text{ J} \approx 4 \times 10^{-15} \text{ eV}$$

這個能量極其微小，實際上與 $kT$（室溫下約為 0.025 eV）相比可以忽略。這就是為什麼在宏觀世界中我們觀察不到量子效應。

但對於原子尺度的「盒子」（例如 $a = 1$ Å $= 10^{-10}$ m）：

$$E_0 = \frac{\pi^2 \times (1.055 \times 10^{-34})^2}{2 \times 9.109 \times 10^{-31} \times (10^{-10})^2} \approx 40 \text{ eV}$$

這個能量與原子尺度相當，量子效應非常顯著。

## 10.8 波函數的節點

讓我們分析波函數的節點特性。

波函數 $\psi_n(x) = \sqrt{2/a}\sin(n\pi x/a)$ 在區間 $(0, a)$ 內有 $n-1$ 個**節點**（node），即波函數為零的點。

讓我們驗證：
- 基態 $\psi_1(x) = \sqrt{2/a}\sin(\pi x/a)$：在 $(0, a)$ 區間內，$n\pi x/a$ 從 $0$ 到 $\pi$，正弦函數不為零，**無節點**
- 第一激發態 $\psi_2(x) = \sqrt{2/a}\sin(2\pi x/a)$：在 $x = a/2$ 處，波函數為零，**有1個節點**
- 第二激發態 $\psi_3(x) = \sqrt{2/a}\sin(3\pi x/a)$：在 $x = a/3$ 和 $x = 2a/3$ 處，波函數為零，**有2個節點**

這個規律具有普遍性：**量子數 $n$ 越大，波函數的節點數越多**。

節點數與能量有直接的對應關係：能量越高的態，節點越多。這是量子力學的一個普遍規律。

## 10.9 機率密度分佈

讓我們分析機率密度 $P_n(x) = |\psi_n(x)|^2$ 的分佈。

$$P_n(x) = \frac{2}{a} \sin^2\left(\frac{n\pi x}{a}\right)$$

機率密度的特點：
- 基態（$n=1$）：機率密度在中心（$x = a/2$）最大，在邊界（$x = 0, a$）為零
- 激發態：機率密度有多個峰和谷，峰數等於量子數 $n$

### 10.9.1 與經典極限的對應

在經典物理學中，粒子在勢阱中以恒定速度來回運動。粒子在每個位置出現的時間相同，因此機率密度是**均勻分佈**的。

在量子力學中，當 $n$ 很大時，機率密度的平均值趨於均勻分佈：

$$\langle P_n(x) \rangle_{\text{小尺度平均}} \approx \frac{1}{a}$$

這與經典結果一致！這是對應原理的又一體現。

## 10.10 與經典極限的對應

當量子數 $n$ 很大時，量子力學應該趨於經典力學。讓我們驗證這一點。

### 10.10.1 能量趨於連續

當 $n$ 很大時，相鄰能級間隔 $\Delta E$ 遠小於 $E_n$，能量接近連續：

$$\frac{\Delta E}{E} \approx \frac{2nE_0}{n^2E_0} = \frac{2}{n} \to 0 \quad \text{當} n \to \infty \text{時}$$

### 10.10.2 機率分佈趨於均勻

如上所述，機率密度的平均值趨於均勻分佈，這與經典粒子在勢阱中各處出現概率相同的結論一致。

### 10.10.3 振盪週期趨於經典週期

量子波包的振盪週期與經典週期一致。

## 10.11 二維與三維推廣

一維勢阱可以推廣到二維和三維情況。

### 10.11.1 二維方形勢阱

考慮矩形區域 $0 < x < a$，$0 < y < b$，勢能為：

$$V(x,y) = \begin{cases} 0 & \text{在矩形內} \\ \infty & \text{在矩形外} \end{cases}$$

波函數可以分離變數：

$$\psi_{n_x,n_y}(x,y) = \sqrt{\frac{2}{a}}\sin\left(\frac{n_x\pi x}{a}\right) \sqrt{\frac{2}{b}}\sin\left(\frac{n_y\pi y}{b}\right)$$

能量為：

$$E_{n_x,n_y} = \frac{\pi^2\hbar^2}{2m}\left(\frac{n_x^2}{a^2} + \frac{n_y^2}{b^2}\right)$$

### 10.11.2 三維立方勢阱

對於立方體 $0 < x,y,z < a$，能量為：

$$E_{n_x,n_y,n_z} = \frac{\pi^2\hbar^2}{2ma^2}\left(n_x^2 + n_y^2 + n_z^2\right)$$

### 10.11.3 簡併

在二維和三維中，不同的量子數組合可能對應相同的能量，這稱為**簡併**（degeneracy）。

例如，在二維方形阱中，如果 $a = b$，則狀態 $(n_x = 2, n_y = 1)$ 和 $(n_x = 1, n_y = 2)$ 具有相同的能量。

## 10.12 無限勢阱的實際應用

雖然無限勢阱是一個理想化模型，但它在物理學中有很多實際應用。

### 10.12.1 半導體量子阱

在半導體物理中，電子被限制在薄層結構中，可以用量子阱模型描述。實際的勢阱是有限深的，但當阱深遠大於電子能量時，無限勢阱是一個很好的近似。

### 10.12.2 原子核殼層模型

原子核的殼層結構可以用類似的勢阱模型描述。核內的核子（質子和中子）被強核力約束在核內，表現出殼層結構。

### 10.12.3 量子點

半導體量子點是納米尺度的結構，電子在其中三維受限，表現出類似原子一樣的離散能級。量子點被稱為「人造原子」。

## 10.13 本章小結

本章我們討論了一維無限勢阱這個重要的量子力學模型。我們看到能量被量子化為 $E_n = n^2\pi^2\hbar^2/(2ma^2)$，波函數為正弦函數 $\psi_n(x) = \sqrt{2/a}\sin(n\pi x/a)$。基態能量不為零（零點能），這是不確定性原理的體現。波函數的節點數等於 $n-1$。當 $n$ 很大時，量子結果趨於經典結果，這是對應原理的體現。這個簡單模型揭示了量子化的許多本質特徵。

# 第四章：原子結構與波爾模型——早期量子論的勝利

## 4.1 原子結構的早期探索

人類對物質結構的探索可以追溯到古希臘時期。公元前400年左右，德謨克利特（Democritus）提出宇宙由不可分割的「原子」組成。然而，在兩千多年的時間裡，原子一直被認為是一個哲學概念，而非科學事實。

十九世紀初，化學家約翰·道爾頓（John Dalton）提出了現代原子論，解釋了化學反應的定量規律。十九世紀末，物理學家發現了電子（1897年，J.J.湯姆森），證明了原子內部存在帶電粒子。這些發現使人們認識到原子並非不可分割，而是具有內部結構的複雜系統。

1904年，J.J.湯姆森提出了著名的「葡萄乾布丁模型」（或稱「西瓜模型」）。根據這個模型，原子是一個均勻帶正電的球，電子像葡萄乾一樣嵌在其中。這個模型能夠解釋原子是電中性的——電子的負電荷與整個球體的正電荷相互抵消。

然而，湯姆森的模型很快就遇到了挑戰。

## 4.2 盧瑟福的原子核式結構模型

1909年，漢斯·蓋革（Hans Geiger）與歐內斯特·馬斯登（Ernest Marsden）在歐內斯特·盧瑟福（Ernest Rutherford，1871-1937）的指導下進行了著名的α粒子散射實驗，這一實驗徹底改變了我們對原子結構的理解。

實驗的基本思想是：用α粒子（氦原子核，帶正電）轟擊非常薄的金箔，通過觀察α粒子的散射角度來推斷原子的內部結構。

盧瑟福曾經說過：「這就像用一英尺口徑的炮彈轟擊一張紙，炮彈卻反彈回來打到自己一樣。」大多數α粒子應該几乎不改變方向地穿過金箔，因為按照湯姆森的模型，正電荷均勻分佈在整個原子中，α粒子不會遇到強大的靜電斥力。

實驗結果令人震驚：
- 大多数α粒子确实几乎直线穿过了金箔
- 但有约1/8000的α粒子发生大角度散射（大于90度）
- 甚至有极少数α粒子被反向弹回

這個結果只能用盧瑟福的原子核式結構模型來解釋：原子内部大部分是空的，几乎所有质量与正电荷都集中在一个极小的原子核上。α粒子在接近原子核时会受到强大的静电斥力，导致大角度散射。

盧瑟福於1911年發表了他的原子結構模型：原子由位於中心的原子核和圍繞原子核運動的電子組成。原子核帶正電，電子帶負電，兩者之間的庫侖引力提供電子運動所需的向心力。

## 4.3 原子光譜的規律

在波爾提出他的原子模型之前，物理學家已經積累了大量關於原子光譜的實驗數據。這些數據為後來的原子結構理論提供了重要的線索。

最簡單的是氫原子光譜。氫原子是宇宙中最簡單的原子，只有一個電子圍繞一個質子運動。當氫原子被激發後，會發射出特定波長的光，形成一系列離散的光譜線。

1885年，瑞士中學教師約翰·巴耳末（Johann Balmer，1825-1898）發現氫原子可見光波段的光譜線可以用一個簡單的公式描述：

$$\lambda = b \frac{n^2}{n^2 - 2^2}, \quad n = 3, 4, 5, \ldots$$

其中 $b = 3645.6$ Å 是常數。這個公式稱為巴耳末公式，它給出的波長與實驗測量值精確符合。

更一般地，後來的研究發現氫原子光譜可以分為多個系列，每個系列對應於不同的下能級：

$$\frac{1}{\lambda} = R_H \left( \frac{1}{n_1^2} - \frac{1}{n_2^2} \right)$$

其中 $n_1 < n_2$ 是正整數，$R_H = 1.0973731568508 \times 10^7$ m$^{-1}$ 是里德伯常數。不同的 $(n_1, n_2)$ 組合給出不同的光譜系列：

- **萊曼系列**（Lyman series）：$n_1 = 1$，位於紫外區
- **巴耳末系列**（Balmer series）：$n_1 = 2$，部分在可見光區
- **帕申系列**（Paschen series）：$n_1 = 3$，位於紅外區
- **布拉格系列**（Brackett series）：$n_1 = 4$
- **普豐德系列**（Pfund series）：$n_1 = 5$

巴耳末公式的發現具有里程碑意義。它表明氫原子內部存在離散的能級結構，這是理解原子結構的重要線索。

## 4.4 波爾模型的提出

丹麥物理學家尼爾斯·波爾（Niels Henrik David Bohr，1885-1962）於1913年發表了著名的「波爾原子模型」，成功地解釋了氫原子光譜。波爾模型的提出標誌著早期量子論的誕生。

波爾模型的建立受到了普朗克能量子假說和愛因斯坦光子假說的深刻影響。波爾意識到，要解釋原子光譜的離散性，必須引入量子化的概念。

波爾模型基於以下三個基本假設：

**假設一：穩定態假設**

電子只能在某些特定的軌道上運動，這些軌道稱為「穩定態」或「定態」。在這些軌道上，電子繞原子核做圓周運動，但不會向外輻射能量。

這個假設與經典电动力学直接矛盾。根據經典电磁理論，運動的電荷應該會輻射 electromagnetic wave，損失能量，最終會螺旋墜入原子核。但波爾假設存在穩定的軌道，電子可以在這些軌道上運動而不損失能量。

**假設二：量子化條件**

並非所有軌道都是允許的，只有當電子的角動量 $L$ 等於 $\hbar$ 的整數倍時，相應的軌道才是穩定的：

$$L = m_e v_n r_n = n\hbar, \quad n = 1, 2, 3, \ldots$$

其中 $m_e$ 是電子質量，$v_n$ 是第 $n$ 個軌道上的電子速度，$r_n$ 是軌道半徑，$n$ 稱為「主量子數」。

這個假設規定了電子軌道必須滿足的離散條件。

**假設三：頻率假設**

當電子從一個較高能量 $E_{n_2}$ 的軌道躍遷到較低能量 $E_{n_1}$ 的軌道時，會發射一個光子，其頻率由兩個能級的能量差決定：

$$\nu = \frac{E_{n_2} - E_{n_1}}{h}$$

反之，電子吸收一個頻率為 $\nu$ 的光子，可以從低能級躍遷到高能級。

這個假設連接了原子能級與觀測到的光譜線頻率。

## 4.5 波爾模型的計算

讓我們來計算氫原子中電子的軌道半徑與能量，這些計算展示了波爾模型如何給出與實驗一致的結果。

考慮一個電子在半徑為 $r$ 的圓軌道上繞原子核運動。向心力由庫侖引力提供：

$$\frac{m_e v^2}{r} = \frac{e^2}{4\pi\varepsilon_0 r^2}$$

其中 $e$ 是電子電荷，$\varepsilon_0$ 是真空介電常數。

由此可得：

$$v^2 = \frac{e^2}{4\pi\varepsilon_0 m_e r}$$

根據量子化條件 $m_e v r = n\hbar$，可得：

$$v = \frac{n\hbar}{m_e r}$$

代入上式：

$$\frac{n^2\hbar^2}{m_e^2 r^2} = \frac{e^2}{4\pi\varepsilon_0 m_e r}$$

解得第 $n$ 個軌道的半徑：

$$r_n = \frac{4\pi\varepsilon_0 \hbar^2}{m_e e^2} n^2 = a_0 n^2$$

其中 $a_0 = \frac{4\pi\varepsilon_0 \hbar^2}{m_e e^2} = 0.5291772108 \times 10^{-10}$ m 是**波爾半徑**，是氫原子基態（$n=1$）的軌道半徑。

波爾半徑是原子物理學中的一個重要長度單位，它代表著氫原子基態電子的典型軌道半徑。

現在計算電子的能量。電子的總能量（動能 + 庫侖勢能）為：

$$E = \frac{1}{2} m_e v^2 - \frac{e^2}{4\pi\varepsilon_0 r} = \frac{e^2}{8\pi\varepsilon_0 r} - \frac{e^2}{4\pi\varepsilon_0 r} = -\frac{e^2}{8\pi\varepsilon_0 r}$$

代入 $r_n$ 的表達式：

$$E_n = -\frac{m_e e^4}{2(4\pi\varepsilon_0)^2 \hbar^2} \frac{1}{n^2} = -\frac{R_y}{n^2}$$

其中 $R_y = \frac{m_e e^4}{2(4\pi\varepsilon_0)^2 \hbar^2} = 13.605693009$ eV 是**里德伯能量**，對應於將電子從基態遊離（電離）所需的能量。

這個能量公式與實驗觀測到的氫原子光譜完美一致！

## 4.6 波爾模型與氫原子光譜

根據波爾模型，電子從高能態 $n_2$ 躍遷到低能態 $n_1$ 時發射的光子頻率為：

$$\nu = \frac{E_{n_2} - E_{n_1}}{h} = \frac{R_y}{h} \left( \frac{1}{n_1^2} - \frac{1}{n_2^2} \right) = cR_H \left( \frac{1}{n_1^2} - \frac{1}{n_2^2} \right)$$

其中 $R_H = R_y / hc$ 是里德伯常數，與實驗測得的氫原子光譜常數完全一致！

這是一個驚人的成就。波爾模型不僅解釋了氫原子光譜的巴耳末系列，還預言了其他系列（如萊曼系列、帕申系列），這些預言後來都被實驗證實。

例如，萊曼系列（$n_1 = 1$）的波長為：

$$\lambda = \frac{1}{R_H} \frac{n_2^2}{n_2^2 - 1}$$

這個系列全部位於紫外區，與實驗觀測完全一致。

## 4.7 波爾模型的局限性

波爾模型在解釋氫原子光譜方面取得了巨大成功，但它存在根本性的缺陷。

首先，**只能處理氫原子**。波爾模型無法處理多電子原子。即使是氦離子 He$^{+}$（一個電子繞帶兩個正電荷的核運動），也需要引入一些人為的修正。

其次，**沒有解釋光譜線的強度**。波爾模型可以預言光譜線的頻率，但無法計算各條光譜線的相對強度。實際上，光譜線的強度與電子躍遷的機率有關，這需要更深入的量子力學處理。

第三，**沒有解釋精細結構**。高分辨率光譜顯示，氫原子的光譜線實際上是由多條非常接近的譜線組成的「精細結構」。波爾模型無法解釋這種現象。這種精細結構是由電子的相對論效應和自旋-軌道耦合引起的。

第四，**沒有解釋塞曼效應**。當原子處於外加磁場中時，光譜線會分裂成多條。波爾模型無法預言這種「塞曼效應」。

第五，**理論基礎不完善**。波爾模型本質上是半經典的：它保留了電子軌道這種經典概念，但又人為地加入了量子化條件。這種混合的理論在數學上是不自洽的。電子被限制在離散的軌道上，但為什麼這些軌道是穩定的？這個問題在波爾模型中沒有回答。

## 4.8 對應原理

波爾還提出了著名的「對應原理」（Correspondence Principle），指出量子理論在極限條件下應該回歸到經典物理學。

在波爾模型中，當主量子數 $n$ 很大時，相鄰能級之間的能量差趨於零，這時電子的運動趨於連續，與經典物理學一致。根據對應原理，量子躍遷發射的光在 $n$ 很大時應該趨於經典的輻射頻率（即電子繞核旋轉的頻率）。

讓我們驗證這一點。電子在第 $n$ 個軌道上運動的頻率為：

$$\nu_n = \frac{v_n}{2\pi r_n} = \frac{m_e e^4}{2(4\pi\varepsilon_0)^2 \hbar^3} \cdot \frac{1}{n^3}$$

兩個相鄰軌道之間躍遷發射的光的頻率為：

$$\nu_{n+1 \to n} = \frac{E_{n+1} - E_n}{h} = \frac{R_y}{h} \left( \frac{1}{n^2} - \frac{1}{(n+1)^2} \right) \approx \frac{2R_y}{h n^3} \quad \text{當} n \gg 1 \text{時}$$

比較兩個頻率，它們的比例在 $n \gg 1$ 時趨於1，這就是對應原理的體現。

對應原理在後續量子力學的發展中起了重要的指導作用。它幫助物理學家確保新的量子理論在極限條件下與經典理論一致。

## 4.9 波爾模型的歷史意義

波爾模型在物理學史上具有重要的歷史意義。

首先，**它是首個成功的量子原子模型**。波爾模型首次將量子化概念應用於原子結構，解釋了氫原子光譜的規律。這是量子論在微觀世界中的首次重大勝利。

其次，**它開創了早期量子論**。波爾模型的成功激發了物理學家對量子化概念的深入研究，為後續量子力學的建立奠定了基礎。

第三，**它促進了實驗與理論的結合**。波爾模型與光譜實驗的緊密結合，展示了解釋微觀世界的新方法。

第四，**它推動了哲學思考**。波爾模型中的電子軌道概念引發了關於量子測量和實在性的深刻哲學討論。

## 4.10 本章小結

本章我們介紹了盧瑟福的原子核式結構模型與波爾的原子模型。波爾模型基於穩定態假設、量子化條件與頻率假設三個基本假設，成功解釋了氫原子光譜的規律。我們計算了波爾半徑與能量公式 $E_n = -R_y/n^2$，與實驗完全一致。然而，波爾模型存在諸多局限性，只能處理氫原子這種最簡單的系統，無法解釋光譜線的強度、精細結構與塞曼效應等現象。這些問題需要等到完整的量子力學建立後才能得到解決。

# 第六十二章：量子相位估計

## 62.1 相位估計的物理動機

量子相位估計（Quantum Phase Estimation, QPE）是量子計算中最重要和最廣泛使用的演算法原語之一。它是連接量子計算理論與實際應用的關鍵橋樑，幾乎所有重要的量子演算法都依賴於它或其變體。

相位估計的問題在物理學中非常普遍。當我們測量一個么正算符$U$的本徵態時，我們得到對應本徵值的相位。例如，在氫原子中測量能量對應於測量哈密頓量$U = e^{-iHt/\hbar}$的本徵值。

從數學上，如果$U|\psi\rangle = e^{2\pi i \theta}|\psi\rangle$，則$\theta$是本徵值的相位。我們的任務是估計這個相位$\theta$。

為什麼這很重要？因為：
- Shor演算法的核心是估計與因數分解相關的么正算符的相位
- 量子化學中，測量分子哈密頓量的本徵能量
- 量子搜尋和優化中的相位處理

## 62.2 相位估計問題的陳述

量子相位估計問題的正式陳述如下：

給定一個么正算符$U$，以及$U$的一個本徵態$|\psi\rangle$，估計對應本徵值的相位$\theta \in [0, 1)$使得：
$$U|\psi\rangle = e^{2\pi i \theta}|\psi\rangle$$

我們要求估計的精度為$O(1/2^m)$，其中$m$是我們願意投入的資源。

更精確地說，目標是以高概率得到$\theta$的二進位表示的前$m$位。

## 62.3 相位估計電路

相位估計的量子電路結構如下：

1. **準備輔助量子位元**：$t$個量子位元初始化為$|0\rangle$，目標量子位元初始化為$|\psi\rangle$

2. **第一階段 - 並發控制旋轉**：
   - 對每個輔助量子位元$j = 0, 1, \ldots, t-1$，應用Hadamard閘
   - 對第$j$個輔助量子位元（控制）和目標量子位元（目標），應用控制-$U^{2^j}$

3. **第二階段 - 逆量子傅立葉變換**：
   - 對輔助量子位元應用$QFT^\dagger$

4. **測量**：測量所有輔助量子位元

整個電路的思想是：將本徵態的相位「編碼」到輔助量子位元的狀態中，然後通過逆QFT「解碼」這個相位。

## 62.4 第一階段的數學

讓我們分析第一階段的行為。經過Hadamard後，輔助量子位元$j$處於均等疊加態。

控制-$U^{2^j}$操作的效果是：當控制量子位元為$|1\rangle$時，在目標量子位元上應用$U^{2^j}$。

考慮目標量子位元是$U$的本徵態$|\psi\rangle$：
$$U^{2^j}|\psi\rangle = (e^{2\pi i\theta})^{2^j}|\psi\rangle = e^{2\pi i 2^j \theta}|\psi\rangle$$

因此，在第一階段結束時，狀態為：
$$\frac{1}{\sqrt{2^t}} \sum_{j=0}^{2^t-1} e^{2\pi i \theta j} |j\rangle |\psi\rangle$$

其中$j$是$t$位元的二進位表示。

這正是$e^{2\pi i \theta}$的傅立葉基狀態！這就是為什麼第二步需要逆QFT。

## 62.5 第二階段的分析

第二階段應用逆量子傅立葉變換（$QFT^\dagger$）。$QFT^\dagger$的定義是：

$$QFT^\dagger|j\rangle = \frac{1}{\sqrt{2^t}} \sum_{k=0}^{2^t-1} e^{-2\pi i jk/2^t} |k\rangle$$

經過$QFT^\dagger$後，狀態變為：
$$|\psi_{out}\rangle = \sum_{k=0}^{2^t-1} \alpha_k |k\rangle |\psi\rangle$$

其中振幅$\alpha_k$正比於對$k/2^t$與$\theta$接近程度的某種度量。

當$2^t\theta$是整數時（即$\theta$可以精確表示為$m$位二進位），測量將精確給出$\theta$。

當$2^t\theta$不是整數時，測量結果以高概率給出$\theta$的最接近估計。

## 62.6 精度分析

相位估計的精度由輔助量子位元的數目$t$決定。

**精確情況**：如果$2^t\theta$是整數，選擇$t$個輔助量子位元可以精確測定$\theta$。

**近似情況**：一般情況下，選擇足夠的餘量量子位元來提高成功概率。

分析顯示：如果我們想要$m$位精確度，應該使用$t = m + \lceil \log_2(2 + 1/(2\epsilon)) \rceil$個輔助量子位元，測量成功的概率至少為$1 - \epsilon$。

這意味著要達到$O(1/2^m)$的精度，我們需要$O(m)$個輔助量子位元和$O(m^2)$個量子門。

## 62.7 Shor演算法中的相位估計

相位估計最重要的應用之一是Shor的質因數分解演算法。讓我們看看它如何嵌入Shor演算法。

Shor演算法用於找到$N$的質因數。核心步驟是找到函數$f(x) = a^x \mod N$的週期$r$，使得$a^r \equiv 1 \pmod{N}$。

定義么正算符：
$$U|y\rangle = |ay \mod N\rangle$$

這個算符作用於$d = \lceil \log_2 N \rceil$個量子位元。

$U$的本徵態可以寫成：
$$|\psi_s\rangle = \frac{1}{\sqrt{r}} \sum_{x=0}^{r-1} e^{-2\pi i s x/r} |a^x \mod N\rangle$$

對應的本徵值為$e^{2\pi i s/r}$。

相位估計應用於這個本徵態，輸出$s/r$。通過連分數展開，可以從$s/r$恢復週期$r$，進而分解$N$。

## 62.8 量子化學中的相位估計

相位估計的另一個重要應用是量子化學。

在量子化學中，我們想要找出一個分子哈密頓量$H$的基態能量。通過Trotterization，我們可以構造$U = e^{-iHt}$的近似。

對基態$|E_0\rangle$應用相位估計，得到能量$E_0$。

問題是：我們通常不知道基態$|E_0\rangle$！這是「基態準備」問題，是量子化學計算的主要挑戰之一。

變分量子特徵值求解器（VQE）提供了一種混合量子-經典方法來解決這個問題：使用量子電腦準備一個試探態，使用相位估計估計能量，使用經典電腦優化參數。

## 62.9 迭代相位估計

原始相位估計需要大量量子位元。一種節省資源的替代方案是迭代相位估計（IPE）。

迭代相位估計的思想是：
- 每次迭代只使用一個輔助量子位元
- 從最重要的位元到最不重要的位元，逐位確定$\theta$
- 每步使用前一步的結果來調整

IPE的優點：
- 只使用1個輔助量子位元
- 總門數減少
- 更適合近期 NISQ 設備

缺點：
- 需要更多迭代（$O(m)$次）
- 測量結果需要反饋用於下一步

## 62.10 貝葉斯相位估計

傳統相位估計給出離散的近似答案。貝葉斯相位估計提供了一種更連續、更灵活的框架。

在貝葉斯方法中：
- 我們有一個關於$\theta$的先驗分佈$p(\theta)$
- 測量結果$y$提供似然函數$p(y|\theta)$
- 後驗分佈由貝葉斯定理給出：$p(\theta|y) \propto p(y|\theta)p(\theta)$

貝葉斯方法的好處：
- 可以獲得$\theta$的完整後驗分佈
- 可以自適應選擇測量策略
- 可以量化估計的不確定性

貝葉斯相位估計在量子度量學和量子實驗設計中有重要應用。

## 62.11 健壯相位估計

噪聲是實際量子硬體的主要挑戰。健壯相位估計（RPE）是一種在存在噪聲的情況下仍能工作的相位估計方案。

RPE的關鍵思想：
- 利用特定的量子門（如Clifford群門）的高精度
- 使用多個不同的測量基來提高精度
- 避免使用難以實現的高精度門

RPE在NMR量子計算和離子阱系統中已經實驗驗證，可以在存在顯著噪聲的情況下仍能精確估計相位。

## 62.12 本章小結

本章我們深入探討了量子相位估計——這是量子計算中最重要和最廣泛使用的演算法原語之一。

相位估計的任務是估計么正算符本徵態對應本徵值的相位。我們詳細介紹了相位估計的量子電路結構：使用輔助量子位元、通過控制旋轉編碼相位、最後用逆QFT解碼。

精度分析表明，要達到$O(1/2^m)$的精度，需要$O(m)$個輔助量子位元和$O(m^2)$個量子門。迭代相位估計提供了一種資源效率更高的替代方案。

相位估計在Shor因數分解演算法和量子化學計算中有核心應用。我們還討論了貝葉斯相位估計和健壯相位估計，這些是處理實際應用中噪聲和不确定性的重要技術。

相位估計是通往更高級量子演算法的門戶，是理解量子計算實際價值的關鍵。

# 第三十一章：氫原子理論

## 31.1 氫原子問題

**氫原子**是量子力學中唯一可以精確求解的原子（離子）系統。它為我們提供了理解原子結構的基礎。

氫原子由一個質子（固定在原點）與一個電子組成，庫侖勢為：

$$V(r) = -\frac{e^2}{4\pi\varepsilon_0 r}$$

哈密頓算符：

$$\hat{H} = -\frac{\hbar^2}{2m} \nabla^2 - \frac{e^2}{4\pi\varepsilon_0 r}$$

## 31.2 分離變數

在球坐標 $(r, \theta, \phi)$ 中，哈密頓算符可以分離。

令波函數：

$$\psi(r, \theta, \phi) = R(r) Y_{lm}(\theta, \phi)$$

其中 $Y_{lm}$ 是球諧函數，滿足角動量本徵值方程。

代入薛丁格方程式，得到徑向方程：

$$-\frac{\hbar^2}{2m} \frac{d^2u}{dr^2} + \left[ \frac{l(l+1)\hbar^2}{2mr^2} - \frac{e^2}{4\pi\varepsilon_0 r} \right] u = Eu$$

其中 $u(r) = rR(r)$。

## 31.3 徑向方程的求解

徑向方程可以用級數方法求解。設 $E < 0$（束縛態）。

引入無量綱變數：
$$\rho = \frac{r}{a_0}, \quad a_0 = \frac{4\pi\varepsilon_0\hbar^2}{me^2} \text{（波爾半徑）}$$
$$\varepsilon = \frac{-E}{Ry}, \quad Ry = \frac{me^4}{2(4\pi\varepsilon_0)^2\hbar^2} \text{（里德伯能量）}$$

求解得到能量本徵值：

$$E_n = -\frac{Ry}{n^2}, \quad n = 1, 2, 3, \ldots$$

主量子數 $n$ 決定能量，與角量子數 $l$ 無關！

## 31.4 氫原子量子數

氫原子的三個量子數：
- **主量子數** $n = 1, 2, 3, \ldots$：決定能量 $E_n = -Ry/n^2$
- **角量子數** $l = 0, 1, 2, \ldots, n-1$：決定軌道角動量 $L = \sqrt{l(l+1)}\hbar$
- **磁量子數** $m = -l, -l+1, \ldots, l$：決定 $L_z = m\hbar$

對於給定 $n$，$l$ 可以取 $0$ 到 $n-1$，每個 $l$ 有 $2l+1$ 個 $m$ 值。

## 31.5 徑向波函數

徑向波函數：

$$R_{nl}(r) = \frac{2}{n^2 a_0^{3/2}} \left[\frac{(n-l-1)!}{(n+l)!}\right]^{1/2} \rho^l L_{n-l-1}^{2l+1}(\rho) e^{-\rho/2}$$

其中 $L_k^{m}$ 是**伴隨拉蓋爾多項式**。

前幾個徑向波函數：
- $R_{10} = 2 a_0^{-3/2} e^{-r/a_0}$
- $R_{20} = \frac{1}{\sqrt{2}} a_0^{-3/2} (1 - r/a_0) e^{-r/2a_0}$
- $R_{21} = \frac{1}{\sqrt{24}} a_0^{-3/2} (r/a_0) e^{-r/2a_0}$

## 31.6 完整波函數

氫原子波函數：

$$\psi_{nlm}(r, \theta, \phi) = R_{nl}(r) Y_{lm}(\theta, \phi)$$

歸一化條件：

$$\int |\psi_{nlm}|^2 d^3r = 1$$

波函數滿足正交性：

$$\int \psi_{nlm}^* \psi_{n'l'm'} d^3r = \delta_{nn'}\delta_{ll'}\delta_{mm'}$$

## 31.7 氫原子的能級簡併

氫原子的能級簡併：
- 能量只依賴於 $n$，不依賴於 $l$
- 對於給定 $n$，$l$ 可以取 $0$ 到 $n-1$（$n$ 個值）
- 對於給定 $l$，$m$ 可以取 $-l$ 到 $l$（$2l+1$ 個值）

總簡併度：

$$g_n = \sum_{l=0}^{n-1} (2l+1) = n^2$$

這與精細結構中的 $j$ 依賴結合後會分裂。

## 31.8 本章小結

本章我們求解了氫原子的薛丁格方程式。我們發現能量本徵值 $E_n = -Ry/n^2$，只依賴於主量子數 $n$，與 $l$ 無關（偶然簡併）。波函數是徑向函數與球諧函數的乘積，量子數為 $(n, l, m)$。

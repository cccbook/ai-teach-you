# 第 28 章：宇宙學基礎

## 概述

廣義相對論不僅適用於局部引力現象，還是現代宇宙學的理論基礎。本章介紹宇宙學的基本概念，包括宇宙學原理、Robertson-Walker 度規、弗里德曼方程以及宇宙的演化歷史。宇宙學是研究宇宙整體結構、起源和演化的學科，廣義相對論在其中扮演著核心角色。

宇宙學的研究對象是整個可觀測宇宙——一個跨越數百億光年、包含數千億星系的龐大系統。在如此巨大的尺度上，物質的分布呈現出令人驚訝的均勻性，這使得宇宙學模型得以簡化。愛因斯坦在 1917 年首次將廣義相對論應用於宇宙學研究，此後近一個世紀的發展使宇宙學從純粹的理論推測轉變為精密的觀測科學。

## 28.1 宇宙學原理

### 28.1.1 均勻性

宇宙學原理的第一個假設是**均勻性**（homogeneity）：在大尺度上，宇宙中物質的分布是均勻的。

什麼是「大尺度」？觀測表明，在超過約 100 Mpc（3.26 億光年）的尺度上，物質密度趨於均勻。在小尺度上，宇宙是高度非均勻的：
- 行星系統：km 尺度
- 星系：kpc（數千光年）尺度
- 星系群/團：Mpc（數百萬光年）尺度
- 超星系團：10-100 Mpc 尺度
- 大尺度結構（纖維狀結構、空洞）：100-300 Mpc 尺度

這種層次化的結構形成被稱為「宇宙網」。但在更大的尺度上，這些結構消失了，宇宙看起來是均勻的。

均勻性假設的數學表述：存在一個切片（常數超曲面），在其中任意兩點的物理條件相同。這意味著度規在空間上是平移不變的。

### 28.1.2 各向同性

宇宙學原理的第二個假設是**各向同性**（isotropy）：從任何觀測點向各個方向觀測，宇宙看起來是相同的。

觀測支持這個假設：
- 宇宙微波背景輻射的温度在各方向上均匀到 $10^{-5}$ 精度
- 星系分布在大尺度上顯示各向同性
- 哈勃流的膨脹在各方向上相同

各向同性假設的數學表述：存在一個切片，其中任意兩點之間的 Killing 向量形成 O(3) 群（旋轉群）。

### 28.1.3 宇宙學原理的結合

均勻性和各向同性兩個假設結合起來，意味著宇宙中存在一個「優選切片」——常數 cosmic time 曲面。在這個切片上，空間度規是最大對稱的，即三維球面、平坦歐氏空間或三維雙曲空間之一。

這就是**完美宇宙學原理**——宇宙不僅在空間上是均勻各向同性的，而且在時間上也是不變的（如果忽略局部的結構形成和演化）。

### 28.1.4 宇宙學原理的觀測支持

宇宙學原理並非先驗為真，而是基於觀測的經驗假設。主要的觀測證據包括：

1. **哈勃定律**：遙遠星系的退行速度與距離成正比，顯示宇宙在大尺度上的膨脹是各向同性的
2. **宇宙微波背景**：CMB 的各向同性温度為 $10^{-5}$ 精度
3. **X 射線背景**：高能輻射源的分佈也顯示各向同性
4. **大型星系巡天**：如 Sloan Digital Sky Survey 顯示，在超過 100 Mpc 尺度後，星系分布趨於均勻

## 28.2 Robertson-Walker 度規

### 28.2.1 度規的推導

 Robertson-Walker 度規是滿足宇宙學原理的最一般度規。我們從均勻性和各向同性出發推導它。

空間部分必須是最大對稱的三維流形。最大對稱空間有 constant sectional curvature $k$，其度規可以寫為：

$$d\chi^2 + S_k^2(\chi)\, d\Omega^2$$

其中 $S_k(\chi)$ 為：
- $k = +1$（正曲率）：$S_k(\chi) = \sin\chi$
- $k = 0$（零曲率）：$S_k(\chi) = \chi$
- $k = -1$（負曲率）：$S_k(\chi) = \sinh\chi$

完整的四維 Robertson-Walker 度規為：

$$ds^2 = -c^2 dt^2 + a(t)^2\left[d\chi^2 + S_k^2(\chi)\, d\Omega^2\right]$$

其中 $a(t)$ 是尺度因子，描述宇宙的膨脹或收縮。

### 28.2.2 線元的顯式形式

使用更常見的徑向座標 $r$（與 $\chi$ 通過 $r = S_k(\chi)$ 關聯），度規可以寫為：

$$ds^2 = -c^2 dt^2 + a(t)^2 \left[\frac{dr^2}{1 - kr^2} + r^2 d\Omega^2\right]$$

這是宇宙學中最常用的形式。

當 $k = 0$ 時：
$$ds^2 = -c^2 dt^2 + a(t)^2(dr^2 + r^2 d\Omega^2)$$

這是平坦空間中的 FRW 度規。

當 $k = +1$ 時：
$$ds^2 = -c^2 dt^2 + a(t)^2\left[\frac{dr^2}{1-r^2} + r^2 d\Omega^2\right]$$

這是封閉宇宙的度規。

當 $k = -1$ 時：
$$ds^2 = -c^2 dt^2 + a(t)^2\left[\frac{dr^2}{1+r^2} + r^2 d\Omega^2\right]$$

這是開放宇宙的度規。

### 28.2.3 度規的物理意義

Robertson-Walker 度規包含兩個關鍵函數：
1. **尺度因子 $a(t)$**：描述宇宙的膨脹/收縮歷史
2. **曲率參數 $k$**：描述空間的整體幾何

度規的時間部分是靜止的（$g_{tt} = -c^2$），這定義了共動觀測者的世界線——這些觀測者隨宇宙膨脹而運動，坐標 $(r, \theta, \phi)$ 保持不變。

共動距離 $r$ 與固有距離 $d$ 的關係為：
$$d = a(t)\, r$$

這意味著當尺度因子增加時，所有固有距離都按比例增加。

### 28.2.4 共動坐標系

在 Robertson-Walker 度規中，使用共動坐標系 $(t, r, \theta, \phi)$：
- $t$ 是 cosmic time，由靜止觀測者測量的時間
- $(r, \theta, \phi)$ 是共動坐標，隨宇宙膨脹保持不變

共動觀測者之間的固有距離為 $d = a(t)\, \chi$，其中 $\chi$ 是共動距離（與 $r$ 不同，當 $k \neq 0$ 時）。

## 28.3 弗里德曼方程

### 28.3.1 Einstein 方程的應用

將 Robertson-Walker 度規代入 Einstein 方程，考慮完美流體的能量-動量張量：

$$T_{\mu\nu} = (\rho + p)u_\mu u_\nu + p\, g_{\mu\nu}$$

其中 $\rho$ 是能量密度，$p$ 是壓強，$u^\mu$ 是流體四速度（對於共動觀測者，$u^\mu = (1/c, 0, 0, 0)$）。

計算 Christoffel 符號和 Ricci 張量，代入 Einstein 方程，得到兩個獨立的方程：

### 28.3.2 第一弗里德曼方程

$$H^2 \equiv \left(\frac{\dot{a}}{a}\right)^2 = \frac{8\pi G}{3}\rho - \frac{kc^2}{a^2} + \frac{\Lambda c^2}{3}$$

其中 $H$ 是哈勃參數，$\Lambda$ 是宇宙學常數。

這個方程描述尺度因子的演化與物質能量密度的關係。

### 28.3.3 第二弗里德曼方程

$$\frac{\ddot{a}}{a} = -\frac{4\pi G}{3}\left(\rho + \frac{3p}{c^2}\right) + \frac{\Lambda c^2}{3}$$

這個方程顯示，壓強會促進宇宙的減速膨脹——正的壓強導致額外的引力效應。

### 28.3.4 連續性方程

能量守恆給出：

$$\dot{\rho} + 3H\left(\rho + \frac{p}{c^2}\right) = 0$$

這可以從兩個弗里德曼方程推導出來，因此不是獨立的。

### 28.3.5 狀態方程

完美流體的狀態方程為：

$$p = w\,\rho\, c^2$$

其中 $w$ 是常數參數：

- **物質**（灰塵）：$w = 0$
- **輻射**：$w = 1/3$
- **宇宙學常數**：$w = -1$
- **暗能量（ Quintessence）**：$w \neq -1$ 但接近 -1

從連續性方程，如果 $w$ 是常數：

$$\rho \propto a^{-3(1+w)}$$

### 28.3.6 弗里德曼方程的物理意義

第一弗里德曼方程可以重新排列為：

$$\frac{kc^2}{a^2} = \frac{8\pi G}{3}\rho - H^2 + \frac{\Lambda c^2}{3} = \frac{8\pi G}{3}\rho_{\text{crit}} - H^2$$

定義臨界密度：

$$\rho_{\text{crit}} = \frac{3H^2}{8\pi G}$$

定義密度參數：

$$\Omega = \frac{\rho}{\rho_{\text{crit}}} = \Omega_m + \Omega_r + \Omega_\Lambda + \Omega_k$$

其中：
- $\Omega_m$：物質（包括暗物質）
- $\Omega_r$：輻射
- $\Omega_\Lambda$：宇宙學常數/暗能量
- $\Omega_k = -k/(aH)^2$：曲率貢獻

曲率與 $\Omega$ 的關係：
$$k = \text{sign}(1 - \Omega)\quad \Rightarrow \quad \Omega > 1 \Rightarrow k = +1$$

## 28.4 宇宙的演化

### 28.4.1 物質主宰宇宙

當宇宙以非相對論性物質為主時（$w = 0$），能量密度：

$$\rho_m \propto a^{-3}$$

代入第一弗里德曼方程，在 $k = 0$ 的平坦宇宙中：

$$H^2 = \frac{8\pi G}{3}\rho_m \propto a^{-3}$$

因此：

$$\dot{a}^2 \propto a^{-1} \Rightarrow a \propto t^{2/3}$$

這是物質主宰宇宙的尺度因子演化。

現在的宇宙正在經歷這個階段嗎？答案是否定的——目前的宇宙由暗能量主宰，這是加速膨脹的階段。

### 28.4.2 輻射主宰宇宙

在宇宙極早期，輻射是主要的能量形式。對於輻射（$w = 1/3$）：

$$\rho_r \propto a^{-4}$$

尺度因子演化：

$$a \propto t^{1/2}$$

這個階段持續到宇宙年齡約 47,000 年（紅移 $z \approx 3400$），此後物質開始主宰宇宙。

### 28.4.3 暗能量主宰宇宙

當宇宙學常數 $\Lambda$ 或暗能量主宰時：

$$\rho_\Lambda = \text{const} \quad \Rightarrow \quad H^2 = \frac{\Lambda c^2}{3} = \text{const}$$

尺度因子：

$$a \propto e^{Ht}$$

這是指數膨脹——宇宙加速膨脹！

我們的宇宙目前正處於暗能量主宰階段，約佔總能量的 68%。

### 28.4.4 宇宙的時代劃分

宇宙的演化可以分為以下主要階段：

| 階段 | 時間 | 紅移 | 主要成分 |
|------|------|------|----------|
| 大爆炸 | 0 | $\infty$ | - |
| 輻射主宰 | ~10 秒 - 47,000 年 | $z > 3400$ | 輻射 |
| 物質主宰 | 47,000 年 - 現在 | $0 < z < 3400$ | 物質 |
| 暗能量主宰 | 現在 - 無限未來 | $z < 0$ | 暗能量 |

### 28.4.5 宇宙年齡

從弗里德曼方程可以計算宇宙的年齡：

$$t = \int_0^{t_0} dt = \int_0^{a_0} \frac{da}{aH(a)}$$

對於目前的 $\Lambda$CDM 模型（物質 + 宇宙學常數）：
$$t_0 \approx 13.8 \text{ Gyr}$$

## 28.5 宇宙微波背景輻射

### 28.5.1 CMB 的發現

1965 年，Arno Penzias 和 Robert Wilson 發現了宇宙微波背景輻射（CMB）。這是宇宙大爆炸理論的最重要證據之一。

他們在貝爾實驗室使用微波天線時，發現了一個無法解釋的背景噪聲。這個噪聲在天空各個方向都是均匀的，且對應於約 2.7 K 的黑體輻射。

CMB 的發現使 Penzias 和 Wilson 獲得 1978 年諾貝爾物理學獎。

### 28.5.2 CMB 的物理意義

CMB 來自宇宙年齡約 38 萬年時的宇宙狀態。此時：
- 宇宙溫度：約 3000 K
- 氫原子形成（復合）
- 宇宙從不透明變為透明
- 光子開始自由傳播

這些光子從「最後散射面」發出，紅移到今天的約 2.725 K。

### 28.5.3 CMB 的精確測量

從 1990 年代的 COBE 衛星開始，CMB 被精確測量：

- **COBE**（1992）：發現 CMB 的各向異性（10 K 尺度）
- **WMAP**（2003, 2013）：精確測量各向異性功率譜
- **Planck**（2013, 2018）：最精確的 CMB 測量

CMB 的主要參數（Planck 2018）：
- $H_0 = 67.4 \pm 0.5$ km/s/Mpc
- $\Omega_m = 0.315 \pm 0.007$
- $\Omega_\Lambda = 0.685 \pm 0.007$
- $\Omega_k \approx 0$

### 28.5.4 CMB 各向異性

CMB 不是完全均匀的，存在微小的温度漲落：
- 平均温度：$T_0 = 2.725$ K
- 漲落幅度：$\Delta T/T \sim 10^{-5}$

這些漲落對應於宇宙中的原初密度擾動——這些擾動後來形成了星系和結構。

CMB 各向異性功率譜顯示了多個峰，這些峰的位置和高度精確確定了宇宙的幾何和成分。

## 28.6 暗物質與暗能量

### 28.6.1 暗物質

暗物質是看不到但通過引力效應探測到的物質。

**觀測證據**：
1. **星系旋轉曲線**：星系外圍恆星的速度不隨距離衰減，暗示存在不可見的物質暈
2. **星系團質量**：X 射線氣體測量的質量遠小於引力約束的質量
3. **引力透鏡**：遙遠星系的畸變顯示存在看不見的質量
4. **CMB**：功率譜的結構需要暗物質

**暗物質的數量**：
$$\Omega_m \approx 0.315$$

其中普通物質只佔約 0.05，暗物質佔約 0.265。

**暗物質的候選**：
- WIMP（大質量弱相互作用粒子）
- Axion（軸子）
- 原始黑洞
- Sterile neutrino

### 28.6.2 暗能量

暗能量是導致宇宙加速膨脹的神秘能量形式。

**觀測證據**：
1. **超新星 Ia**：1998 年發現，遙遠超新星比預期更暗，暗示宇宙加速膨脹
2. **CMB**：宇宙的幾何是平坦的，但物質不足以解釋
3. **大尺度結構**：與 CMB 約束一致

**暗能量的數量**：
$$\Omega_\Lambda \approx 0.685$$

**暗能量的性質**：
- 如果是宇宙學常數，$w = -1$
- 如果是動力學暗能量（Quintessence），$w \approx -1$ 但可能變化

### 28.6.3 $\Lambda$CDM 模型

目前的宇宙學標準模型是 $\Lambda$CDM：
- **$\Lambda$**：宇宙學常數（暗能量）
- **CDM**：冷暗物質（Cold Dark Matter）

模型的預言與幾乎所有觀測一致，是目前最好的宇宙學模型。

## 28.7 宇宙的幾何

### 28.7.1 空間曲率

宇宙的空間幾何由曲率參數 $k$ 決定：

**$k = +1$（封閉宇宙）**：
- 正曲率空間
- 總體積有限
- 三角形的內角和大於 180°
- 類似於三維球面

**$k = 0$（平坦宇宙）**：
- 零曲率
- 體積無限
- 三角形的內角和等於 180°
- 歐幾里得幾何

**$k = -1$（開放宇宙）**：
- 負曲率空間
- 體積無限
- 三角形的內角和小於 180°
- 類似於雙曲幾何

### 28.7.2 曲率的觀測約束

CMB 和其他觀測顯示：
$$|\Omega_k| < 0.005$$

宇宙在大尺度上是非常平坦的。這與暴脹理論的預言一致——暴脹期間的指數膨脹將任何初始曲率稀釋到可以忽略的程度。

## 28.8 宇宙的最終命運

### 28.8.1 暗能量主宰的未來

如果暗能量是真正的宇宙學常數（$w = -1$），宇宙的未來是：
- 繼續加速膨脹
- 星系團之間的距離指數增加
- 遙遠星系的光永遠無法到達我們
- 宇宙變得越來越冷、越來越空

這被稱為「大撕裂」（Big Rip）之外的可能性——實際上是「大冻结」（Big Freeze）或「熱寂」（Heat Death）。

### 28.8.2 恆星的命運

在更短的時間尺度上：
- 恆星繼續演化
- 星系燃料耗盡
- 恆星死亡，留下白矮星、中子星、黑洞
- 黑洞通過 Hawking 輻射蒸發

### 28.8.3 量子漲落與真空

在極長的時間尺度上：
- 量子漲落可能產生新的宇宙
- 真空可能是不穩定的
- 可能發生量子隧道到新的真空態

## 28.9 數值計算示例

以下 Python 代碼計算宇宙的尺度因子演化：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

def friedmann(y, t, Om, OL, Ok):
    """弗里德曼方程的右端"""
    a = y[0]
    da_dt = y[1]
    H = np.sqrt(Om / a**3 + OL + Ok / a**2)
    d2a_dt2 = - (Om / (2 * a**2)) * da_dt + (3 * OL / 2) * a
    return [da_dt, d2a_dt2]

# 參數（Planck 2018）
Om = 0.315
OL = 0.685
Ok = 1 - Om - OL

# 初始條件：a = 10^-3，物質主宰時期
a0 = 1e-3
da0 = 0.5 * np.sqrt(Om / a0**3)  # 初始速度

y0 = [a0, da0]

# 時間：紅移從 1000 到現在
t = np.linspace(0, 13.8, 1000)  # Gyr

# 數值積分
solution = odeint(friedmann, y0, t, args=(Om, OL, Ok))
a = solution[:, 0]

# 繪圖
plt.figure(figsize=(10, 6))
plt.subplot(1, 2, 1)
plt.plot(t, a)
plt.xlabel('時間 (Gyr)')
plt.ylabel('尺度因子 a')
plt.title('宇宙尺度因子演化')
plt.grid(True)

plt.subplot(1, 2, 2)
z = 1/a - 1  # 紅移
plt.plot(z[::-1], a[::-1])
plt.xlabel('紅移 z')
plt.ylabel('尺度因子 a')
plt.title('尺度因子 vs 紅移')
plt.grid(True)
plt.xscale('log')

plt.tight_layout()
plt.savefig('cosmology_scale_factor.png', dpi=150)
plt.show()

# 計算宇宙年齡
print(f"宇宙年齡: {t[-1]:.2f} Gyr")
```

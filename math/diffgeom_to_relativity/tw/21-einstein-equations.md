# 第 21 章：愛因斯坦場方程

## 概述

愛因斯坦場方程是廣義相對論的核心——它將時空的幾何（曲率）與物質能量分佈聯繫起來。本章將推導這個方程，解釋其物理意義，並討論其在各種物理情形下的應用。

愛因斯坦場方程是 20 世紀最重要的物理方程之一。它告訴我們：物質告訴時空如何彎曲，時空告訴物質如何運動。這是一個優雅的幾何化描述，將引力從「力」的概念重新定義為時空本身的幾何性質。

## 21.1 從等效原理到場方程

### 21.1.1 牛頓引力回顧

牛頓引力方程：

$$\nabla^2 \Phi = 4\pi G \rho$$

其中 $\Phi$ 是引力勢，$\rho$ 是質量密度。

這個方程描述質量如何產生引力場。

### 21.1.2 類比

在廣義相對論中，類比關係是：

| 牛頓 | 廣義相對論 |
|-------|------------|
| 引力勢 $\Phi$ | 時空度量 $g_{\mu\nu}$ |
| 質量密度 $\rho$ | 能量-動量張量 $T_{\mu\nu}$ |
| Laplace 算子 $\nabla^2$ | 里奇張量 $R_{\mu\nu}$ |

### 21.1.3 愛因斯坦的洞察

愛因斯坦認識到，引力方程應該具有以下形式：

$$G_{\mu\nu} = \kappa T_{\mu\nu}$$

其中 $G_{\mu\nu}$ 是由時空幾何構造的張量，$T_{\mu\nu}$ 是物質的能量-動量張量。

## 21.2 愛因斯坦場方程

### 21.2.1 完整形式

$$R_{\mu\nu} - \frac{1}{2} g_{\mu\nu} R + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4} T_{\mu\nu}$$

這是完整的愛因斯坦場方程（帶宇宙學常數）。

各項的物理意義：
- $R_{\mu\nu}$：里奇張量，描述時空的局部彎曲
- $R$：里奇標量，里奇張量的收縮
- $\Lambda$：宇宙學常數
- $T_{\mu\nu}$：能量-動量張量

### 21.2.2 耦合常數的確定

耦合常數 $8\pi G/c^4$ 來自於牛頓極限。

在弱場、低速極限，愛因斯坦方程應該還原為牛頓方程。通過這個條件可以確定耦合常數。

### 21.2.3 宇宙學常數

$\Lambda$ 是宇宙學常數。

愛因斯坦最初引入它來獲得靜態宇宙的解。當 Hubble 發現宇宙膨脹後，愛因斯坦稱之為他「最大的錯誤」。

現代觀點認為 $\Lambda$ 對應於「暗能量」或「真空能量」。

### 21.2.4 真空方程

當 $T_{\mu\nu} = 0$（真空中，無物質）：

$$R_{\mu\nu} - \frac{1}{2} g_{\mu\nu} R + \Lambda g_{\mu\nu} = 0$$

當 $\Lambda = 0$：

$$R_{\mu\nu} = 0$$

這是真空中的愛因斯坦方程。

## 21.3 愛因斯坦張量

### 21.3.1 定義

**愛因斯坦張量**定義為：

$$G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2} g_{\mu\nu} R$$

這是由里奇張量和里奇標量構造的對稱張量。

### 21.3.2 重要性質

1. **對稱性**：$G_{\mu\nu} = G_{\nu\mu}$
2. **協變散度為零**：$\nabla^\mu G_{\mu\nu} = 0$
3. **蹤**：$G^\mu_{\ \mu} = R - 2R = -R$

### 21.3.3 協變散度

愛因斯坦張量的協變散度自動為零：

$$\nabla^\mu G_{\mu\nu} = 0$$

這個幾何恆等式來自於 Bianchi 恒等式。

### 21.3.4 物理意義

協變散度為零 $\nabla^\mu G_{\mu\nu} = 0$ 對應於能量-動量守恆！

因為愛因斯坦方程給出 $\nabla^\mu T_{\mu\nu} = 0$，這推廣了牛頓的能量守恆到彎曲時空。

## 21.4 作用量原理

### 21.4.1 愛因斯坦-希爾伯特作用量

愛因斯坦方程可以從作用量原理推導。

作用量為：

$$S = \int d^4x \sqrt{-g} \left( \frac{c^4}{16\pi G} R + \mathcal{L}_{matter} \right)$$

其中：
- $\sqrt{-g}$ 是度規行列式的因子
- $\mathcal{L}_{matter}$ 是物質場的拉格朗日密度

### 21.4.2 變分原理

對度規 $g_{\mu\nu}$ 變分，設作用量的變分為零：

$$\delta S = 0$$

這給出愛因斯坦場方程。

### 21.4.3 推導

變分計算給出：

$$\delta S = \frac{c^4}{16\pi G}\int d^4x \sqrt{-g}\left(G_{\mu\nu} + \Lambda g_{\mu\nu} - \frac{8\pi G}{c^4}T_{\mu\nu}\right)\delta g^{\mu\nu} + \text{邊界項} = 0$$

由於 $\delta g^{\mu\nu}$ 是任意的，得到愛因斯坦方程。

## 21.5 線性化引力

### 21.5.1 弱場極限

在弱場極限，假設：

$$g_{\mu\nu} = \eta_{\mu\nu} + h_{\mu\nu}$$

其中 $|h_{\mu\nu}| \ll 1$。

計算里奇張量到 $h$ 的一階：

$$R_{\mu\nu} = \frac{1}{2}(\partial_\alpha\partial_\mu \bar{h}^\alpha_\nu + \partial_\alpha\partial_\nu \bar{h}^\alpha_\mu - \partial_\mu\partial_\nu \bar{h} - \Box \bar{h}_{\mu\nu})$$

其中 $\bar{h}_{\mu\nu} = h_{\mu\nu} - \frac{1}{2}\eta_{\mu\nu}h$ 是跡反轉擾動。

### 21.5.2 協變規範

選擇 Lorenz 規範：

$$\partial^\nu \bar{h}_{\mu\nu} = 0$$

在此規範下，場方程化簡為：

$$\Box \bar{h}_{\mu\nu} = -\frac{16\pi G}{c^4}T_{\mu\nu}$$

這是波動方程——引力以光速傳播！

### 21.5.3 牛頓極限

在低速、靜態極限：
- $T_{00} \approx \rho c^2$
- 其他分量可以忽略
- $\bar{h}_{00} \approx -2\Phi/c^2$

代入方程得到：

$$\Box \Phi = 4\pi G \rho$$

這正是牛頓引力方程！

## 21.6 能量-動量張量的例子

### 21.6.1 理想流體

$$T^{\mu\nu} = (\rho + p) U^\mu U^\nu + p g^{\mu\nu}$$

其中：
- $\rho$ 是固有能量密度
- $p$ 是壓強
- $U^\mu$ 是流體四速度

### 21.6.2 电磁场

$$T^{\mu\nu} = \frac{1}{\mu_0}\left(F^{\mu\alpha}F^\nu_{\ \alpha} - \frac{1}{4}g^{\mu\nu}F_{\alpha\beta}F^{\alpha\beta}\right)$$

描述電磁場的能量和動量分布。

### 21.6.3 標量場

對於 Klein-Gordon 場：

$$T^{\mu\nu} = \partial^\mu\phi \partial^\nu\phi - g^{\mu\nu}\left(\frac{1}{2}\partial_\alpha\phi\partial^\alpha\phi + V(\phi)\right)$$

### 21.6.4 真空

$$T_{\mu\nu} = 0 \Rightarrow R_{\mu\nu} = 0$$

真空中的愛因斯坦方程。

## 21.7 守恆定律

### 21.7.1 協變守恆

$$\nabla_\mu T^{\mu\nu} = 0$$

這是能量-動量守恆的協變形式。

在平坦時空中，這化簡為 $\partial_\mu T^{\mu\nu} = 0$——這是熟悉的能量-動量守恆。

### 21.7.2 拖曳效應

在廣義相對論中，時空幾何也承載能量。但「總能量」的定義是微妙的。

能量-動量守恆在彎曲時空中需要小心處理，因為：
- 時空曲率會影響守恆量的定義
- 沒有簡單的「總能量」局部定義

### 21.7.3 ADM 能量

在漸近平坦時空中，可以定義總能量（ADM 能量）：

$$E_{\text{ADM}} = \frac{1}{16\pi G}\int_{S_\infty} (\partial_j h_{ij} - \partial_i h_{jj}) dS_i$$

這是在無窮遠處測量的總能量。

## 21.8 邊界條件

### 21.8.1 漸近平坦性

在大距離處，時空應該趨於平坦：

$$g_{\mu\nu} \to \eta_{\mu\nu} + O(1/r)$$

這確保了牛頓極限的存在。

### 21.8.2孤立系統

對於孤立系統（如兩個黑洞），邊界條件是：
- 在無窮遠處，度規趨於 Minkowski 度規
- 物質只存在於有限區域

### 21.8.3 星系動力學

在宇宙學中，邊界條件不同——使用宇宙學解（如 FRW 度規）作為背景。

## 21.9 數值相對論

### 21.9.1 3+1 分解

將四維愛因斯坦方程分解為 3+1 維：

$$ds^2 = -\alpha^2 dt^2 + \gamma_{ij}(dx^i + \beta^i dt)(dx^j + \beta^j dt)$$

其中 $\alpha$ 是 lapse，$\beta^i$ 是 shift。

### 21.9.2 約束方程

- **哈密頓約束**：$R + K^2 - K_{ij}K^{ij} = 16\pi\rho$
- **動量約束**：$\nabla_j(K^{ij} - \gamma^{ij}K) = 8\pi S^i$

### 21.9.3 演化方程

約束方程決定初始數據，演化方程描述時間演化。

## 重點回顧

| 概念 | 說明 |
|------|------|
| 愛因斯坦場方程 | $G_{\mu\nu} + \Lambda g_{\mu\nu} = 8\pi G T_{\mu\nu}/c^4$ |
| 里奇張量 $R_{\mu\nu}$ | 曲率的一次收縮 |
| 標量曲率 $R$ | 曲率的完全收縮 |
| 愛因斯坦張量 $G_{\mu\nu}$ | $R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R$ |
| 宇宙學常數 $\Lambda$ | 真空能量 |
| 協變守恆 | $\nabla_\mu T^{\mu\nu} = 0$ |

## 練習題

1. **耦合常數推導**：從牛頓極限出發，證明愛因斯坦場方程的耦合常數必須是 $8\pi G/c^4$。

2. **協變散度**：驗證 $G^{\mu\nu}$ 的協變散度為零，$\nabla_\mu G^{\mu\nu} = 0$。

3. **真空方程**：在真空中（$T_{\mu\nu} = 0$），愛因斯坦場方程變成什麼？證明當 $\Lambda = 0$ 時，$R_{\mu\nu} = 0$。

4. **弱場極限**：在弱場近似下，證明愛因斯坦方程給出牛頓引力方程 $\Box \Phi = 4\pi G\rho$。

5. **理想流體**：驗證理想流體的能量-動量張量 $T^{\mu\nu} = (\rho + p)U^\mu U^\nu + p g^{\mu\nu}$ 滿足協變守恆。

6. **度規變分**：從愛因斯坦-希爾伯特作用量出發，簡述如何通過變分得到愛因斯坦場方程。

7. **能量條件**：討論能量條件（弱能量條件、主導能量條件）在愛因斯坦方程中的物理意義。

8. **宇宙學常數**：討論宇宙學常數的物理意義，以及它與暗能量的關係。

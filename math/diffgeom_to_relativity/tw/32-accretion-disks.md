# 第 32 章：黑洞吸積盤

## 概述

黑洞本身是「看不見」的——它們不發射任何可探測的電磁輻射。然而，當黑洞從周圍環境吸積物質時，會產生宇宙中最明亮的天體現象。這些吸積過程產生的輻射使我們能夠探測和研究黑洞。

吸積盤是環繞黑洞的旋轉物質結構，物質通過螺旋軌道逐漸落入黑洞。在這個過程中，物質的引力勢能轉化為熱能和動能，產生高溫輻射。超大質量黑洞的吸積盤可以比整個星系還亮——類星體就是這種現象的極端例子。

吸積過程是天體物理學中能量轉換效率最高的方式之一。相比核聚變（約 0.7% 的質量-能量轉換效率），吸積可以將高達約 40% 的物質能量轉化為輻射（對於 Kerr 黑洞）。這使得吸積盤成為宇宙中最有效的能量引擎。

## 32.1 吸積過程基礎

### 32.1.1 吸積的物理機制

吸積是物質落入引力源的過程。對於黑洞來說：

1. **初始條件**：物質必須具有足夠的角動量，否則會直接落入黑洞
2. **角動量屏障**：具有角動量的物質不會直接落入，而是形成盤狀結構
3. **角動量轉移**：盤內的粘滯效應將內層物質的角動量向外傳遞
4. **質量內流**：失去角動量的物質逐漸向內螺旋，最終落入黑洞

這個過程類似於太陽系中行星形成的情況——原始星雲中的塵埃和氣體通過類似機制聚集形成行星。

### 32.1.2 能量釋放

物質在吸積過程中釋放的能量來自引力勢能。對於單位質量的物質，從半徑 $r$ 落入黑洞時釋放的能量為：

$$\Delta E = \frac{GM}{r_{\text{isco}}}$$

其中 $r_{\text{isco}}$ 是最內穩定圓軌道（ Innermost Stable Circular Orbit，ISCO）。

對於 Schwarzschild 黑洞，$r_{\text{isco}} = 6GM/c^2$，因此：

$$\Delta E \approx \frac{1}{12}mc^2 \approx 8.3\%$$

對於極端 Kerr 黑洞（$a = M$），$r_{\text{isco}} = GM/c^2$，能量釋放可達：

$$\Delta E \approx \frac{1}{2}mc^2 \approx 42\%$$

這遠高於核聚變的能量效率！

### 32.1.3 吸積率

吸積率是單位時間內落入黑洞的質量：

$$\dot{M} = -\frac{dM}{dt}$$

典型的吸積率：
- 恆星質量黑洞：$\dot{M} \sim 10^{-8}$ 至 $10^{-4} M_\odot$/年
- 超大質量黑洞：$\dot{M} \sim 1$ 至 $100 M_\odot$/年

吸積率決定了吸積盤的亮度：

$$L = \eta \dot{M}c^2$$

其中 $\eta$ 是能量轉換效率。

## 32.2 吸積盤物理

### 32.2.1 薄盤模型

Shakura 和 Sunyaev（1973）提出了標準薄盤模型，這是描述黑洞吸積盤的經典框架。

基本假設：
1. 盤是幾何薄的：$H \ll r$，其中 $H$ 是盤的厚度
2. 盤是光學厚的：光子在逃逸前經歷多次散射
3. 盤是軸對稱的：$\partial/\partial\phi = 0$
4. 盤是穩態的：$\partial/\partial t = 0$

### 32.2.2 開普勒旋轉

在黑洞周圍，假設牛頓近似（對於 $r \gg GM/c^2$），物質在開普勒軌道上運動：

$$v_\phi = \sqrt{\frac{GM}{r}}$$

角速度：

$$\Omega = \sqrt{\frac{GM}{r^3}}$$

這意味著內層物質運動更快——這與土星環類似。

### 32.2.3 粘滯應力

吸積盤中的粘滯力是角動量轉移的微觀機制。在標準模型中，粘滯應力與壓強成正比：

$$t_{r\phi} = \alpha P$$

其中 $\alpha$ 是粘滯參數（典型值 $\alpha \sim 0.01-0.1$）。

這個「$\alpha$ 粘滯」模型是一個唯象模型，真正的微觀機制是磁旋轉不穩定性（MRI）。

### 32.2.4 垂直結構

吸積盤的垂直結構可以分為幾層：

1. **盤的中平面**：溫度最高，密度最大
2. **光球層**：輻射逃逸的表面
3. **大氣層**：溫度梯度導致風和噴流的產生

中平面溫度 $T_{\text{mid}}$ 與半徑的關係：

$$T_{\text{mid}} \propto \dot{M}^{2/5} M_{\text{BH}}^{3/5} r^{-3/5}$$

## 32.3 吸積盤的光譜

### 32.3.1 多溫黑體譜

吸積盤的每個半徑環發射近黑體輻射。溫度隨半徑變化，因此整個盤的光譜是多溫黑體的疊加。

盤的光譜功率：

$$L_\nu \propto \int_{r_{\text{in}}}^{r_{\text{out}}} B_\nu(T(r)) \cdot 2\pi r\, dr$$

其中 $B_\nu(T)$ 是普朗克函數。

### 32.3.2 溫度-質量關係

對於最內穩定圓軌道附近發射的主要輻射：

$$T_{\text{peak}} \propto M_{\text{BH}}^{-3/4} \dot{M}^{1/4}$$

這給出了一個重要的觀測結果：

| 黑洞類型 | 典型質量 | 峰值溫度 | 峰值波段 |
|----------|----------|----------|----------|
| 恆星質量 | $10 M_\odot$ | $\sim 10^7$ K | X射線 |
| 中等質量 | $10^3-10^5 M_\odot$ | $\sim 10^5-10^6$ K | 極紫外 |
| 超大質量 | $10^6-10^9 M_\odot$ | $\sim 10^4-10^5$ K | 光學/紫外 |

### 32.3.3 X射線光譜

對於恆星質量黑洞，吸積盤的峰值在 X射線波段。X射線天體，如天鵝座 X-1，展示以下特徵：

1. **熱成分**：來自吸積盤的熱 X射線
2. **幂律成分**：來自冕區的非熱輻射
3. **鐵線**：來自反射過程的熒光線

### 32.3.4 Fe Kα 線

鐵的 Kα 線（6.4 keV）是吸積盤診斷的重要工具。

當 X射線照射冷的吸積盤時，鐵原子發生光電離並產生熒光，發射 6.4 keV 的特徵線。

這條線的輪廓被以下效應修改：
1. **都卜勒效應**：旋轉導致線的展寬
2. **光致移動**：光子在逃逸前下落一段距離，增加紅移
3. **相對論效應**：近黑洞區域的強引力場

測量鐵線輪廓可以約束：
- 黑洞的自旋
- 盤的傾角
- 盤的內邊界半徑

## 32.4 吸積模式

### 32.4.1 標準薄盤

標準薄盤（SS-disc）適用於中等吸積率的情況：
- 吸積率：$0.01 \lesssim \dot{M}/\dot{M}_{\text{Edd}} \lesssim 1$
- 幾何薄，光學厚
- 能量在盤中局部釋放

當吸積率接近愛丁頓極限時，輻射壓開始主導，盤可能不穩定。

### 32.4.2 愛丁頓極限

愛丁頓光度是吸積所能達到的最大光度：

$$L_{\text{Edd}} = \frac{4\pi G M m_p c}{\sigma_T} \approx 1.3 \times 10^{38} \frac{M}{M_\odot} \text{ erg/s}$$

當吸積光度接近 $L_{\text{Edd}}$ 時，輻射壓與引力平衡，限制進一步的吸積。

### 32.4.3 ADAF 模型

對流主宰吸積流（ADAF，Advection-Dominated Accretion Flow）適用於低吸積率情況：
- 吸積率：$\dot{M} \lesssim 0.01 \dot{M}_{\text{Edd}}$
- 幾何厚，光學薄
- 能量被流入物質「攜帶」而非輻射

在 ADAF 中：
- 電子溫度極高（$T_e \sim 10^9$ K）
- 主要發射 X射線和伽馬射線
- 效率低——大部分能量沒有釋放

### 32.4.4 Slim 盤

在超愛丁頓吸積率時，盤變得「厚」，稱為 slim 盤：
- 吸積率：$\dot{M} \gtrsim \dot{M}_{\text{Edd}}$
- 幾何厚度增加
- 徑向速度增加
- 能量被物質內流「運輸」

## 32.5 廣義相對論效應

### 32.5.1 最內穩定圓軌道

ISCO 是廣義相對論的獨特現象。在牛頓引力中，圓軌道可以存在於任意近的距離。但在廣義相對論中，當 $r < r_{\text{isco}}$ 時，圓軌道是不穩定的。

對於 Schwarzschild 黑洞：

$$r_{\text{isco}} = 6GM/c^2 = 3r_s$$

對於 Kerr 黑洞：
$$r_{\text{isco}} = \frac{GM}{c^2}\left[3 + Z_2 \mp \sqrt{(3-Z_1)(3+Z_1+2Z_2)}\right]$$

其中
$$Z_1 = 1 + (1-a^2)^{1/3}\left[(1+a)^{1/3} + (1-a)^{1/3}\right]$$
$$Z_2 = \sqrt{3a^2 + Z_1^2}$$

ISCO 位置與黑洞自旋的關係：
- $a = 0$（靜止）：$r_{\text{isco}} = 6GM/c^2$
- $a = M$（極端）：$r_{\text{isco}} = GM/c^2$（赤道）
- $a = -M$（逆旋轉）：$r_{\text{isco}} \approx 9GM/c^2$

### 32.5.2 能量釋放分布

在廣義相對論中，能量釋放率與半徑的關係為：

$$\frac{dE}{dr\, dt} \propto \frac{1}{r^3}\frac{d\ln\Omega}{d\ln r}\left|\frac{dr}{d\psi}\right|$$

對於 Schwarzschild 黑洞，ISCO 內側的釋放最顯著。

### 32.5.3 光線偏折與光譜

靠近黑洞的吸積盤發出的光會被時空幾何彎曲：
- 來自盤背面的光可以繞過黑洞被看到（背光像）
- 盤的表觀形狀被扭曲
- 光線行進時間延遲影響觀測信號

這是黑洞成像的基礎，也是事件視界望遠鏡工作的原理。

## 32.6 噴流

### 32.6.1 相對論性噴流的特點

許多黑洞系統顯示出高速等離子體噴流：
- 速度：$v \sim 0.9c$ 至 $0.99c$
- 方向：垂直於吸積盤
- 尺度：從恆星質量到百萬光年

噴流是宇宙中最長的單一結構之一，可達數十億光年。

### 32.6.2 Blandford-Znajek 機制

Blandford 和 Znajek（1977）提出，旋轉黑洞可以通過磁場提取能量。

機制：
1. 黑洞自轉在周圍時空產生參考系拖曳
2. 磁場線被時空拖曳
3. 磁應力提取旋轉能
4. 能量沿磁力線傳輸到遠處

功率估算：

$$P_{\text{BZ}} \sim \frac{B^2 r_H^2 a^2}{c}$$

其中 $B$ 是特徵磁場強度，$r_H$ 是視界半徑。

### 32.6.3 Blandford-Payne 機制

另一個能量來源是吸積盤本身：

1. 盤中的磁場與旋轉等離子體耦合
2. 離心力將等離子體沿磁力線向外拋出
3. 這是「磁離心主導」的噴流形成機制

### 32.6.4 噴流的準周期性振盪

一些黑洞 X射線雙星顯示準周期性振盪（QPO），頻率在 0.1-500 Hz。

可能的機制：
- 吸積盤的震盪模式
- 噴流中的 Kelvin-Helmholtz 不穩定性
- 黑洞的聲波模式

## 32.7 數值模擬

### 32.7.1 磁流體動力學

要模擬吸積盤，需要求解廣義相對論磁流體動力學（GRMHD）方程：

1. **連續性方程**：質量守恆
2. **能量動量方程**：廣義相對論 Euler 方程
3. **馬克斯韋方程**：磁場演化
4. **能量方程**：熱力學過程

### 32.7.2 挑戰

數值模擬面臨的挑戰：
- 多尺度問題：從視界到吸積盤外緣
- 強引力場：靠近視界
- 辐射傳輸：光子如何與物質相互作用
- 高溫等離子體：相對論性電子

### 32.7.3 模擬結果

現代 GRMHD 模擬可以：
- 再現吸積盤的結構
- 模擬噴流的形成
- 預言鐵線輪廓
- 計算黑洞陰影

## 32.8 觀測證據

### 32.8.1 X射線雙星

恆星質量黑洞存在於 X射線雙星系統中：
- 夥伴星提供吸積物質
- 黑洞通過 X射線輻射被探測

著名例子：
- 天鵝座 X-1：第一個確認的恆星質量黑洞候選
- GS 2023+338：X射線新星
- GRS 1915+105：顯示豐富的時變性

### 32.8.2 活動星系核

超大質量黑洞存在於星系中心：
- 吸積產生活動星系核（AGN）
- 核區比整個星系還亮
- 噴流可延伸至星系尺度

AGN 的分類：
- Seyfert 星系：中等光度
- 類星體：高光度
- Blazar：噴流指向地球

### 32.8.3 事件視界望遠鏡

2019 年，事件視界望遠鏡發布了 M87 星系黑洞的首張影像。

影像顯示：
- 一個明亮的環狀結構
- 中心的黑暗區域（黑洞陰影）
- 與廣義相對論預言一致

這是吸積盤光學成像的直接證據！

## 32.9 數值計算示例

以下 Python 代碼計算吸積盤的溫度分布和光譜：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import quad

def planck(nu, T):
    """普朗克函數（SI 單位）"""
    h = 6.626e-34  # 普朗克常數
    k = 1.381e-23  # 玻爾茲曼常數
    c = 3e8  # 光速
    return (2 * h * nu**3 / c**2) / (np.exp(h * nu / (k * T)) - 1)

def disk_temperature(M, m_dot, r):
    """計算吸積盤的局部溫度
    M: 黑洞質量（太陽質量）
    m_dot: 吸積率（太陽質量/年）
    r: 徑向距離（GM/c^2）
    """
    # 物理常數
    G = 6.674e-11
    c = 3e8
    M_sun = 1.989e30
    year = 365.25 * 24 * 3600
    
    M_kg = M * M_sun
    m_dot_kg = m_dot * M_sun / year
    r_m = G * M_kg * r / c**2  # 轉換為米
    
    # ISCO 半徑
    r_isco = 6 * G * M_kg / c**2
    
    if r < r_isco:
        return 0  # ISCO 內不形成盤
    
    # 能量釋放
    eta = 0.06  # 典型效率
    dE_dt = eta * m_dot_kg * c**2
    
    # 局部溫度（簡化估算）
    sigma = 5.67e-8  # 斯特藩-玻爾茲曼常數
    r_inner = r_isco
    r_outer = 100 * r_isco
    
    # 能量密度
    flux = dE_dt / (4 * np.pi * (r_outer**2 - r_inner**2))
    T = (flux / sigma)**0.25
    
    # 尺度
    T = T * (r / r_isco)**(-3/4)
    
    return T

# 參數
M_bh = 10  # 太陽質量
m_dot = 0.001  # 太陽質量/年

# 半徑
r = np.linspace(6, 100, 100)  # 以 GM/c^2 為單位

# 溫度分布
T = np.array([disk_temperature(M_bh, m_dot, ri) for ri in r])

# 光譜
nu = np.logspace(14, 20, 100)  # 頻率範圍
spectrum = np.zeros_like(nu)

for i, ri in enumerate(r):
    Ti = T[i]
    if Ti > 0:
        # 每個半徑環的光譜貢獻
        dr = r[1] - r[0] if i < len(r) - 1 else 0.1
        spectrum += planck(nu, Ti) * 2 * np.pi * ri * dr

# 繪圖
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.plot(r, T, 'r-', linewidth=2)
plt.xlabel('r (GM/c²)')
plt.ylabel('溫度 (K)')
plt.title(f'M = {M_bh} M⊙ 黑洞吸積盤溫度分布')
plt.grid(True)
plt.yscale('log')

plt.subplot(1, 2, 2)
plt.plot(nu, spectrum, 'b-', linewidth=2)
plt.xlabel('頻率 (Hz)')
plt.ylabel('譜功率')
plt.title('吸積盤光譜')
plt.xscale('log')
plt.yscale('log')
plt.grid(True)

plt.tight_layout()
plt.savefig('accretion_disk.png', dpi=150)
plt.show()

# 峰值估計
print(f"峰值溫度: {np.max(T):.2e} K")
```

## 練習題

1. **能量效率**：計算 Schwarzschild 黑洞和極端 Kerr 黑洞的吸積效率，證明後者可達 42%。

2. **愛丁頓吸積率**：計算質量為 $10^8 M_\odot$ 的超大質量黑洞的愛丁頓吸積率。

3. **盤溫度**：使用標準薄盤模型，推導吸積盤溫度與半徑的關係 $T \propto r^{-3/4}$。

4. **ISCO 位置**：證明 Schwarzschild 黑洞的 ISCO 在 $r = 6GM/c^2$。

5. **鐵線輪廓**：解釋為什麼鐵 Kα 線會被展寬和扭曲，以及如何用於約束黑洞參數。

6. **Blandford-Znajek 功率**：估算超大質量黑洞的 Blandford-Znajek 功率，討論其與活動星系核噴流的關係。

8. **M87 黑洞**：解釋事件視界望遠鏡如何成像 M87 的吸積盤，以及為什麼看到的是亮的環而非盤本身。

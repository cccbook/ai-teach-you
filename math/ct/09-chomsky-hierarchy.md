# 第 9 章：喬姆斯基層級

## 概述

1956年，語言學家**諾姆·喬姆斯基**（Noam Chomsky）在他的論文《語言的描述的三種模型》中，提出了語言的數學理論。

喬姆斯基的洞見是：**語法結構可以用數學方法描述**，而這些描述恰好對應於不同的計算能力層級。

這一章，我們將探索**喬姆斯基層級**（Chomsky Hierarchy），這是形式語言理論的核心概念。

## 9.1 語言學革命的背景

### 9.1.1 喬姆斯基的革命

1950年代的語言學被**結構主義**統治。結構主義者認為：

- 語言是行為的產物
- 語法是對觀察到的語言結構的描述
- 意義由使用決定

喬姆斯基提出了截然不同的觀點：

- 語言能力是天生的
- 語法是心智的計算系統
- 某些結構是普遍的

### 9.1.2 與計算理論的聯繫

喬姆斯基不僅是語言學家，也是數學家。他意識到：

- 語法可以用**生成規則**描述
- 這些規則對應於**自動機**
- 不同複雜度的語法對應於不同的自動機

這個洞見連接了：
- 語言學 ↔ 數學
- 語法 ↔ 計算
- 理論 ↔ 應用

## 9.2 形式語言基礎

### 9.2.1 什麼是形式語言？

**形式語言**是符號串的集合。我們用Σ表示**字母表**（符號集合），用Σ*表示所有可能的字串。

```
例如：
  Σ = {0, 1}
  Σ* = {ε, 0, 1, 00, 01, 10, 11, 000, ...}
  
  L = {所有以1結尾的二進制數}
    = {1, 01, 11, 001, 011, 101, 111, ...}
```

### 9.2.2 語法與語言

**語法**（Grammar）是一套生成規則，可以系統性地產生語言中的所有字串。

```
┌────────────────────────────────────────────────────────────────┐
│                      語法成分                                   │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1. 終結符 (Terminals)                                         │
│     語言中的基本符號                                            │
│     例如：單詞、字符                                            │
│                                                                │
│  2. 非終結符 (Non-terminals)                                   │
│     語法類別                                                  │
│     例如：名詞片語、動詞片語                                    │
│                                                                │
│  3. 起始符 (Start Symbol)                                     │
│     推導的起點                                                │
│     通常用 S 表示                                              │
│                                                                │
│  4. 生產規則 (Production Rules)                                │
│     替換規則                                                  │
│     例如：A → Bc, S → aSb                                      │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 9.2.3 推導過程

從起始符開始，反覆應用生產規則，直到只剩終結符：

```
例如：描述 {0ⁿ1ⁿ | n ≥ 1} 的語法

  生產規則：
    S → 01
    S → 0S1

  推導 0⁴1⁴：
    S
    → 0S1
    → 00S11
    → 000S111
    → 00001111
```

## 9.3 喬姆斯基層級

### 9.3.1 四層結構

喬姆斯基層級將形式文法分為四類，從最強到最弱：

```
┌────────────────────────────────────────────────────────────────┐
│                    喬姆斯基層級                                 │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  第0型：無限制文法 (Unrestricted)                              │
│  ───────────────────────────────────────                       │
│  生產規則：α → β (α, β任意非空字符串)                         │
│  對應：枚舉機 (Turing Machine)                                 │
│                                                                │
│  第1型：上下文敏感文法 (Context-Sensitive)                     │
│  ───────────────────────────────────────                       │
│  生產規則：αAβ → αγβ (A為非終結符，γ非空)                     │
│  對應：線性有界自動機 (LBA)                                    │
│                                                                │
│  第2型：上下文無關文法 (Context-Free)                          │
│  ───────────────────────────────────────                       │
│  生產規則：A → γ (A為非終結符)                                │
│  對應：下推自動機 (PDA)                                        │
│                                                                │
│  第3型：正規文法 (Regular)                                    │
│  ───────────────────────────────────────                       │
│  生產規則：A → aB 或 A → a (線性)                             │
│  對應：有限自動機 (DFA/NFA)                                    │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 9.3.2 包含關係

高層級文法產生的語言是低層級文法語言的超集：

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│              正規語言 ⊂ 上下文無關 ⊂ 上下文敏感 ⊂ 枚舉語言      │
│                                                                │
│                   ⊂ 表示「是真子集」                           │
│                                                                │
│  • 每個正規語言也是上下文無關的                                 │
│  • 每個上下文無關語言也是上下文敏感的                           │
│  • 每個上下文敏感語言也是枚舉的                                 │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 9.4 正規語言

### 9.4.1 定義

**正規語言**是最簡單的語言類型，可以用：
- **正規表示式**描述
- **有限自動機**識別

**正規文法**的生產規則：
- $A \rightarrow aB$
- $A \rightarrow a$
- $A \rightarrow \epsilon$（可選）

### 9.4.2 正規語言的例子

| 語言 | 正規表示式 |
|------|----------|
| 所有以0結尾的二進制數 | `1*01*` |
| 所有包含子串"ab" | `.*ab.*` |
| 所有長度為偶數的字串 | `(aa|bb|ab|ba)*` |
| 電子郵件格式 | `[a-z]+@[a-z]+\.[a-z]+` |

### 9.4.3 實際應用

正規語言廣泛用於：
- **詞法分析**：正規表示式定義程式的tokens
- **字串匹配**：grep、sed等工具
- **網絡協議**：簡單的協議格式
- **配置檔案**：某些簡單的配置文件

## 9.5 上下文無關語言

### 9.5.1 定義

**上下文無關語言**比正規語言更強大，可以表達嵌套結構。

**上下文無關文法**的生產規則：
- $A \rightarrow \alpha$（A為非終結符，α為任意字符串）

### 9.5.2 經典例子：括號匹配

正規語言**不能**匹配嵌套的括號：

```
正則表達式失敗：不能用正規表示式匹配 n 對括號

但上下文無關文法可以：

  S → SS
  S → (S)
  S → ε
```

### 9.5.3 實際應用

上下文無關語言用於：
- **程式語言語法**：大多數程式語言是上下文無關的
- **HTML/XML**：標籤的嵌套
- **JSON**：嵌套的數據結構
- **數學表達式**：正確嵌套的括號

## 9.6 上下文敏感語言

### 9.6.1 定義

**上下文敏感語言**可以表達一些上下文相關的約束。

**上下文敏感文法**的生產規則：
- $\alpha A \beta \rightarrow \alpha \gamma \beta$
- （A在上下文α...β中被替換為γ）

### 9.6.2 例子：複制語言

$$L = \{w\#w \mid w \in \{a,b\}^+\}$$

「複制」語言需要比較兩部分是否相同：

```
不能被上下文無關文法生成
但可以被上下文敏感文法生成：

  S → a S a
  S → b S b
  S → #
```

### 9.6.3 實際應用

上下文敏感語言用於：
- **自然語言處理**：某些需要上下文判斷的情況
- **類型系統**：上下文相關的類型規則
- **某些程式語言特徵**：如 Perl 的魔術引號

## 9.7 無限制文法

### 9.7.1 定義

**無限制文法**幾乎沒有約束：
- 生產規則：$\alpha \rightarrow \beta$（α非空）

這類文法可以產生**枚舉語言**——恰好是可以被圖靈機枚舉的語言。

### 9.7.2 與圖靈機的對應

```
┌────────────────────────────────────────────────────────────────┐
│                    語法與計算的對應                             │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  無限制文法 ⟺ 圖靈機                                          │
│  ─────────────────────────────                                │
│  • 圖靈機停機時生成的字符串 ∈ 語言                            │
│  • 語言的枚舉過程就是圖靈機的計算過程                         │
│                                                                │
│  枚舉語言 ⟺ 可計算枚舉的語言                                   │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 9.8 泵引理

### 9.8.1 正規語言泵引理

泵引理描述了正規語言的結構特性：

> **正規語言泵引理**
>
> 如果 L 是正規語言，則存在常數 n（依賴於L），使得：
> 任何 $w \in L$ 且 $|w| \geq n$ 可以被分解為 $w = xyz$
> 其中 $|y| \geq 1$ 且對所有 $i \geq 0$，$xy^iz \in L$

### 9.8.2 用泵引理證明非正規性

[程式檔案：09-1-pumping-lemma.py](../_code/09/09-1-pumping-lemma.py)

```python
"""
泵引理驗證工具
"""

def pumping_lemma_regular(w: str, n: int = 10) -> bool:
    """
    對給定的字串，嘗試找到泵引理所要求的分解
    
    如果找不到，則語言可能是非正規的
    """
    # 簡化版本：嘗試找到重複的模式
    for split in range(1, min(n, len(w) // 2) + 1):
        x = w[:split]
        y = w[split:split+split]
        z = w[split*2:]
        
        # 檢查 |xy| <= n
        if len(x) + len(y) <= n:
            # 檢查 y 的重複
            reconstructed = x + y * 0 + z  # xz
            reconstructed2 = x + y * 2 + z  # xy²z
            
            # 這裡我們只報告發現
            print(f"候選分解：x={x!r}, y={y!r}, z={z!r}")
            print(f"  |xy| = {len(x)+len(y)} <= {n}")
            return True
    
    return False


def prove_not_regular():
    """
    證明語言 L = {0ⁿ1ⁿ | n >= 0} 不是正規語言
    
    證明思路（泵引理）：
    假設 L 是正規的，設泵長度為 n
    取 w = 0ⁿ1ⁿ（|w| = 2n >= n）
    w 可以分解為 xyz，其中 |xy| <= n, |y| >= 1
    因此 y = 0ᵏ (k >= 1)
    泵送 y：xy²z = 0ⁿ⁺ᵏ1ⁿ 不在 L 中（0的個數多餘1的個數）
    矛盾！
    """
    print("=== 證明 L = {0ⁿ1ⁿ | n >= 0} 不是正規語言 ===\n")
    
    print("假設 L 是正規語言，設泵長度為 n")
    print(f"取 w = 0ⁿ1ⁿ\n")
    
    print(f"w 可以分解為：")
    print(f"  x = 0ᵃ")
    print(f"  y = 0ᵇ (b >= 1)")
    print(f"  z = 0ⁿ⁻ᵃ⁻ᵇ 1ⁿ")
    print(f"\n其中 |xy| = a+b <= n\n")
    
    print(f"泵送 y：xy²z = 0ⁿ⁺ᵇ1ⁿ")
    print(f"但 n+b ≠ n，所以 xy²z ∉ L")
    print(f"矛盾！因此 L 不是正規語言")


def prove_not_context_free():
    """
    證明語言 L = {0ⁿ1ⁿ0ⁿ1ⁿ | n >= 1} 不是上下文無關語言
    
    這需要 Ogden's Lemma 或交叉定理
    """
    print("=== 證明 L = {0ⁿ1ⁿ0ⁿ1ⁿ} 不是上下文無關語言 ===\n")
    
    print("假設 L 是上下文無關的")
    print("使用 Ogden's Lemma：存在泵長度 p")
    print(f"取 w = 0ᵖ1ᵖ0ᵖ1ᵖ\n")
    
    print("標記所有 0 為「標記」位置")
    print("根據 Ogden's Lemma，存在分解 w = uvxyz")
    print("泵送 v 和 y")
    print("結果將打破兩組 0ⁿ1ⁿ 的對稱性")
    print("矛盾！")


prove_not_regular()
print()
prove_not_context_free()
```

執行結果：

```

w 可以分解為：
  x = 0ᵃ

  z = 0ⁿ⁻ᵃ⁻ᵇ 1ⁿ

但 n+b ≠ n，所以 xy²z ∉ L
```

# 第 8 章：丘奇-圖靈論題

## 概述

1936年，兩位年輕的數學家幾乎同時提出了兩個看似不同但實質等價的計算模型：

- **阿隆佐·丘奇**：λ演算
- **艾倫·圖靈**：圖靈機

這兩個模型的等價性，以及由此產生的**丘奇-圖靈論題**，是計算理論中最深刻的思想之一。

## 8.1 兩種計算模型的誕生

### 8.1.1 丘奇的λ演算

丘奇在1936年發表論文，證明「一般递归函数」與「λ可定義函數」是等價的。

他的動機是解決希爾伯特的判定問題。丘奇的方法：

1. 定義了「λ可定義函數」的概念
2. 證明了某些函數不是λ可定義的
3. 由此推斷判定問題是不可解決的

### 8.1.2 圖靈的圖靈機

圖靈在同一年發表了著名的論文，引入了圖靈機的概念。

圖靈的方法：
1. 從人類計算者的直覺出發
2. 抽象出計算的本質操作
3. 證明了圖靈機的通用性

### 8.1.3 時間線

## 8.2 丘奇-圖靈論題

### 8.2.1 論題的內容

> **丘奇-圖靈論題（Church-Turing Thesis）**
>
> 直觀上可計算的函數（即人類計算者可以用算法計算的函數）
> 恰好是圖靈機可計算的函數。

這個論題不是定理，而是一個關於什麼是「可計算」的哲學主張。它的形式化表述：

$$\text{FT} \equiv \text{TH} \equiv \text{λ-def} \equiv \text{Turing}$$

其中：
- FT = 一般递归函数（Frege/Gödel/Herbrand）
- TH = 丘奇可計算
- λ-def = λ可定義
- Turing = 圖靈可計算

### 8.2.2 為什麼是論題而非定理？

丘奇-圖靈論題涉及**「直觀可計算」**這個直覺概念。數學可以精確定義「圖靈可計算」，但無法精確定義「直觀可計算」。

因此，我們不能數學地證明這個論題。它基於：
- 大量的例子
- 所有嘗試都失敗了
- 沒有反例被發現

```
┌────────────────────────────────────────────────────────────────┐
│                    論題 vs 定理                                │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  定理：我們可以證明                                           │
│  例如：「圖靈可計算 ⟺ λ可定義」                               │
│                                                                │
│  論題：我們無法完全形式化                                     │
│  例如：「直觀可計算 ⟺ 圖靈可計算」                            │
│                                                                │
│  區別在於「直觀可計算」不是數學概念                            │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 8.3 等價性的證明思路

### 8.3.1 λ演算 → 圖靈機

**思路**：構造一個模擬λ歸約的圖靈機。

1. **編碼**：將λ表達式編碼到圖靈機的紙帶上
2. **模擬**：圖靈機模擬β歸約的每一步
3. **终止**：當λ表達式是正規形式時停機

這個模擬是可行的，因為λ演算的歸約步驟可以逐步轉換為圖靈機的操作。

### 8.3.2 圖靈機 → λ演算

**思路**：用λ表達式編碼圖靈機的狀態和動作。

1. **狀態編碼**：用λ項表示圖靈機的狀態
2. **紙帶編碼**：用列表/對表示紙帶內容
3. **轉換編碼**：用λ函數表示狀態轉換函數

Church編碼使得任何數據結構都可以用λ表達式表示，因此圖靈機的計算可以完全翻譯為λ表達式。

## 8.4 其他計算模型

### 8.4.1 歷史上的等價模型

自1936年以來，人們提出了許多計算模型，都被證明與圖靈機等價：

| 模型 | 提出者 | 年份 | 等價性證明 |
|------|--------|------|-----------|
| λ演算 | 丘奇 | 1936 | ✓ |
| 一般递归函数 | Kleene | 1936 | ✓ |
| 寄存器機 | Shepherdson-Sturgis | 1963 | ✓ |
| 標籤系統 | Minsky | 1961 | ✓ |
| 生產系統 | Post | 1943 | ✓ |
| 有限狀態機+堆疊 | — | — | ✓ |

### 8.4.2 為什麼這些都等價？

這是丘奇-圖靈論題的最強支持：

> 如果這麼多不同的計算模型都互相等價，且都與圖靈機等價，
> 那麼很可能存在一個根本的原因——
> 這些模型都在捕捉同一個概念：**有效可計算性**。

### 8.4.3 擴展的丘奇-圖靈論題

標準的丘奇-圖靈論題只考慮時間和空間。現在有更強的版本：

**資源受限論題**：
- **線性有界自動機**捕獲**線性空間**可計算函數
- **多項式時間圖靈機**捕獲**多項式時間**可計算函數

這些論題更加精確，可以用數學方法證明。

## 8.5 圖靈機的極限

### 8.5.1 什麼是圖靈機不能做的？

儘管圖靈機非常強大，但存在固有的極限：

```
┌────────────────────────────────────────────────────────────────┐
│                    圖靈機的極限                                 │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1. 停機問題                                                   │
│     不能判定任意程式是否會停機                                  │
│                                                                │
│  2. 羅斯的巴別圖書館問題                                        │
│     不能判定任意λ表達式是否可歸約                              │
│                                                                │
│  3. 拓撲學家剪毛問題                                           │
│     某些自然問題是不可計算的                                   │
│                                                                │
│  4. 維爾斯特拉斯近似                                          │
│     存在處處不可導的連續函數                                   │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 8.5.2 停機問題的推論

停機問題的直接後果：

1. **沒有完美的編譯器優化器**
   - 不能總是判斷某段代碼是否多餘

2. **沒有完美的病毒檢測器**
   - 不能總是判斷一個程式是否為病毒

3. **沒有完美的程式驗證器**
   - 不能總是證明程式的正確性

## 8.6 強邱奇-圖靈論題

### 8.6.1 計算的物理極限

有一個更強的論題，聲稱：

> **強邱奇-圖靈論題**
>
> 任何物理上可實現的計算設備都可以被圖靈機模擬。

這個論題假設了物理規律可以被圖靈機完全模擬。

### 8.6.2 量子計算的挑戰

量子計算對強邱奇-圖靈論題提出了挑戰：

- 量子電腦可以在多項式時間內解決某些問題
- 這些問題在經典圖靈機上需要指數時間

**但是**：
- 量子電腦仍然可以被圖靈機模擬
- 只是模擬效率不同
- 強邱奇-圖靈論題仍然成立（只是計算複雜度不同）

### 8.6.3 超圖靈計算？

一些理論家提出了「超圖靈」的計算模型：

- **超計算**（Hypercomputation）：可以計算不可判定函數
- **Oracle機器**：配備「預言」的圖靈機
- **無限時間圖靈機**：允許超限時間

這些模型在物理上是否可實現仍有爭議。

## 8.7 實際意義

### 8.7.1 對程式語言設計的影響

丘奇-圖靈論題告訴我們：

1. **所有通用程式語言都等價**
   - Python、C、JavaScript、Haskell...
   - 任何一個可計算的，另一個也可以
   - 只是表達力和效率不同

2. **λ演算的理論價值**
   - 函數式編程的理論基礎
   - 類型理論的起源
   - 並行/並發編程的視角

### 8.7.2 對複雜度理論的影響

丘奇-圖靈論題是複雜度理論的基礎：

- **P問題**：圖靈機多項式時間可解
- **NP問題**：圖靈機非確定性多項式時間可解
- **P = NP？**：計算理論的核心問題

### 8.7.3 對人工智慧的影響

圖靈測試與丘奇-圖靈論題的交匯：

- 圖靈提出「機器能思考嗎？」這個問題
- 丘奇-圖靈論題暗示「可計算的」與「可機械化的」是同義詞
- 這為符號AI提供了理論基礎

## 8.8 小結

本章我們：

1. **回顧了歷史**：丘奇和圖靈幾乎同時提出了計算模型
2. **理解了丘奇-圖靈論題**：直觀可計算 ⟺ 圖靈可計算
3. **看到了等價性的證明**：λ演算和圖靈機可以互相模擬
4. **探索了其他模型**：寄存器機、標籤系統等都等價
5. **理解了論題的極限**：物理實現和超計算的可能性

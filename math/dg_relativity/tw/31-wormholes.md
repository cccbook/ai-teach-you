# 第 31 章：蟲洞與時間機器

## 概述

蟲洞是廣義相對論中一個令人著迷的話題，它是時空中連接兩個遙遠區域的「捷徑」。這個概念最早由愛因斯坦和 Nathan Rosen 在 1935 年提出，因此被稱為「Einstein-Rosen 橋」。儘管最初的想法是連接兩個不同的宇宙（所謂的「Einstein-Rosen 橋」），但後來的研究表明，蟲洞也可以連接同一個宇宙中相距甚遠的兩點。

蟲洞不僅是科幻小說中的概念，它們在廣義相對論的數學結構中是允許的。然而，維持一個可穿越的蟲洞需要一種奇異的物質——具有負能量密度的「奇異物質」。這種要求使得蟲洞的存在充滿了挑戰。

蟲洞與時間機器的概念密切相關。一些理論模型表明，通過修改蟲洞的內部結構或讓蟲洞旋轉，可以將蟲洞轉變為時間機器。這引發了關於因果性和時間旅行的深刻問題。Hawking 提出的「編時猜想」認為大自然不允許閉合類時曲線的存在，但這個猜想至今未被證明。

## 31.1 蟲洞的歷史

### 31.1.1 Einstein-Rosen 橋

1935 年，愛因斯坦和 Nathan Rosen 發表了一篇論文，提出了連接兩個黑洞視界的通道。這就是著名的 Einstein-Rosen 橋。

原始的 Einstein-Rosen 橋是最大化擴展 Schwarzschild 解的一部分。考慮 Kruskal 圖中的 Schwarzschild 時空，最大擴展的解包含兩個外部宇宙和兩個黑洞區域。連接兩個外部宇宙的「橋」就是 Einstein-Rosen 橋。

然而，這個橋是不可穿越的：
- 橋的寬度在接近視界時迅速收縮
- 要穿過橋需要無限長的坐標時間
- 任何觀測者都會在到達另一側之前落入奇點

因此，原始的 Einstein-Rosen 橋沒有實際的用途——它只是一個數學上的連接。

### 31.1.2 Wheeler 的思想

1960 年代，John Archibald Wheeler 提出了「 geometrodynamics」的概念，認為時空拓撲不是固定的，而是可以動態變化的。

Wheeler 引入了「量子幾何動力學」的思想，試圖將廣義相對論與量子力學結合。他猜測在普朗克尺度上，時空可能有複雜的拓撲結構，蟲洞可能是連接不同拓撲的通道。

### 31.1.3 Morris-Thorne 蟲洞

1988 年，Michael Morris 和 Kip Thorne 開創性地研究了可穿越蟲洞的可能性。

他們問了一個簡單但深刻的問題：什麼樣的時空幾何允許觀測者從一側穿過蟲洞到達另一側？

他們找到了答案：要維持可穿越蟲洞，必須使用具有負能量密度的奇異物質。這開創了蟲洞研究的新時代。

## 31.2 時空拓撲與蟲洞

### 31.2.1 拓撲的基本概念

蟲洞是時空流形的全局拓撲性質。它不是局部幾何，而是整體結構。

考慮一個簡單的比喻：時空是一張二維薄膜（想像成紙張）。蟲洞就像在紙上打了兩個洞，然後用一根吸管連接它們。螞蟻可以從一個洞進入，從另一個洞出來，而不必爬過整張紙。

在三維空間中，蟲洞的結構更難想像，但我們可以用類比理解。

### 31.2.2 蟲洞的數學描述

蟲洞可以用以下方式數學描述：

考慮時空流形 $\mathcal{M}$ 具有兩個「嘴巴」（mouths）$M_1$ 和 $M_2$。蟲洞的喉部（throat）是連接這兩個嘴巴的通道。

在Morris-Thorne 蟲洞模型中，靜止球對稱蟲洞的度規為：

$$ds^2 = -e^{2\Phi(r)}dt^2 + \frac{dr^2}{1 - \frac{b(r)}{r}} + r^2 d\Omega^2$$

其中：
- $\Phi(r)$ 是 redshift 函數
- $b(r)$ 是形狀函數，描述喉部的幾何

蟲洞的關鍵特點是 $b(r) < r$ 在整個通道中，確保不存在視界。

### 31.2.3 蟲洞的結構

Morris-Thorne 蟲洞的典型結構：

1. **嘴巴**：蟲洞的兩個入口，通常是半徑為 $b_0$ 的球面
2. **喉部**：蟲洞的最窄處，半徑為 $b_{\min}$
3. **通道**：連接兩個嘴巴的區域

從嘴巴進入，經過喉部，可以到達另一側的嘴巴。整個旅程可能只需要很短的原時。

## 31.3 可穿越蟲洞的條件

### 31.3.1 沒有事件視界

可穿越蟲洞的第一個條件是度規在喉部附近不能有事件 視界。這意味著 $g_{tt} \neq 0$ 處處成立。

如果存在事件視界，觀測者就無法從蟲洞中逃出——蟲洞變成了黑洞。

### 31.3.2 潮汐力約束

穿越蟲洞的觀測者必須能夠存活。這要求潮汐加速度不能太大。

Morris-Thorne 估計，安全的潮汐加速度約為一個重力加速度 $g$：

$$\Delta a \lesssim g \approx 10 \text{ m/s}^2$$

這對蟲洞的幾何參數設置了約束。

### 31.3.3 穿越時間

穿越時間必須是合理的。如果穿越需要數年甚至數十年，蟲洞就失去了實用價值。

Morris-Thorne 的分析表明，對於合理的穿越時間，蟲洞的尺寸必須是宏觀的——遠大於普朗克尺度。

## 31.4 負能量條件與奇異物質

### 31.4.1 能量條件

廣義相對論中有幾個能量條件：

**弱能量條件（WEC）**：
$$T_{\mu\nu}u^\mu u^\nu \geq 0$$

對所有類時矢量 $u^\mu$ 成立。這意味著能量密度不能為負。

**主導能量條件（DEC）**：
- WEC 成立
- 能量密度大於或等於任何方向的壓強

**強能量條件（SEC）**：
$$(T_{\mu\nu} - \frac{1}{2}T g_{\mu\nu})u^\mu u^\nu \geq 0$$

### 31.4.2 蟲洞違反能量條件

計算 Morris-Thorne 蟲洞的能量-動量張量，發現在喉部附近：

$$T_{tt} < 0 \quad \text{在喉部附近}$$

這意味著能量密度為負！蟲洞需要「奇異物質」來維持。

### 31.4.3 違反能量條件的物理系統

奇異物質在物理學中並非完全未知：

**卡西米爾效應**：
- 兩個平行導體板之間的真空能量密度為負
- 這已被實驗確認

**Hawking 輻射**：
- 在量子層面，視界附近的能量密度可以是負的
- 這是霍金蒸發的原因

**量子場論**：
- 在某些量子場論態中，能量條件可以被違反
- 但這些通常是量子漲落效應

### 31.4.4 約束與問題

即使負能量在某些情況下是可能的，維持宏觀蟲洞需要的負能量遠超過量子效應能提供的。

Morris-Thorne 估計：
- 對於半徑為 1 米的蟲洞
- 需要約 $10^{32}$ J 的負能量
- 這相當於整個太陽質量轉化為能量的約 1/1000

這個數字是如此巨大，使得宏觀蟲洞的實際存在變得非常可疑。

## 31.5 蟲洞的穩定性

### 31.5.1 動力學不穩定性

蟲洞可能是動力學不穩定的。任何微小的擾動——例如普通物質落入蟲洞——可能導致蟲洞迅速關閉或塌縮成黑洞。

量子效應可能穩定蟲洞，但這需要我們尚未擁有的量子引力理論。

### 31.5.2 量子漲落

在量子層面，即使經典上穩定的蟲洞也可能由於量子漲落而不穩定。蟲洞的嘴巴可能因為漲落而振動，最終連接和關閉。

## 31.6 時間機器與因果性

### 31.6.1 閉合類時曲線

閉合類時曲線（CTC）是時空中閉合的世界線，既是類時的（可允許粒子或觀測者沿其運動），又是閉合的（起點和終點相同）。

沿 CTC 運動的觀測者會回到自己的過去。這開啟了時間旅行的可能性。

### 31.6.2 Gott 的時間機器

1991 年，Richard Gott 提出了一個基於宇宙弦的時間機器模型。

兩條無窮長的平行宇宙弦以高速相互掠過。在某些參考系中，穿過兩根弦之間區域的路徑是閉合類時曲線。

然而，這個模型有問題：要形成 CTC，需要無窮長的弦，或者弦必須排列在特定的幾何中。

### 31.6.3 Tipler 柱

1974 年，Frank Tipler 提出，一個無窮長的、密度極大的旋轉cylinder（常被稱為 Tipler 柱）可以產生 CTC。

後來的分析表明，無窮長不是必要的——足夠長但有限的柱也可以。但所需的密度和旋轉速度仍然是物理上不可能的。

## 31.7 Morris-Thorne-Keshav 蟲洞

### 31.7.1 具體的度規形式

Morris-Thorne 考慮的最簡單蟲洞度規為：

$$ds^2 = -e^{2\Phi(r)}dt^2 + \frac{dr^2}{1 - \frac{b_0^2}{r^2}} + r^2 d\Omega^2$$

其中 $b_0$ 是喉部半徑，$\Phi(r)$ 是某個函數。

這個度規的特點是：
- 在 $r = b_0$ 處有喉部
- 當 $r \to \infty$ 時，趨於 Minkowski 時空
- 兩個「宇宙」通過喉部連接

### 31.7.2 嵌入圖

蟲洞的幾何可以通過嵌入圖來可視化。

考慮常數 $t$ 切片，其度規為：

$$ds^2 = \frac{dr^2}{1 - \frac{b_0^2}{r^2}} + r^2 d\Omega^2$$

將這個二維曲面嵌入三維歐幾里得空間中。嵌入條件為：

$$z(r) = \sqrt{r^2 - b_0^2}$$

這是一個旋轉曲面——由雙曲線 $z^2 - r^2 = b_0^2$ 繞 $z$ 軸旋轉而成。

這個形狀類似於蟲洞的「喉」——这就是「蟲洞」這個名字的由來。

### 31.7.3 總質量

使用廣義相對論的質量定義（ ADM 質量），Morris-Thorne 蟲洞的質量為：

$$M = b_0/2$$

這表明蟲洞的質量與喉部半徑成正比。

## 31.8 時間機器與蟲洞

### 31.8.1 將蟲洞轉化為時間機器

Morris 和 Thorne 指出，通過適當的操作，可以將蟲洞轉化為時間機器：

1. 取兩個蟲洞嘴巴
2. 將其中一個嘴巴加速到接近光速並帶回
3. 由於時間膨脹，兩個嘴巴的時間不同步
4. 結果是連接兩個嘴巴的路徑變成了閉合類時曲線

這展示了蟲洞與時間機器之間的深刻聯繫。

### 31.8.2 因果性保護猜想

Hawking 提出了「因果性保護猜想」（Chronology Protection Conjecture）：

> 「除了科幻小說，沒有任何可靠的物理學基礎支持時間旅行的可能性。」

這個猜想表明，隨著蟲洞變為時間機器，必將有某種物理效應阻止 CTC 的形成。

可能的機制：
- 量子漲落可能導致蟲洞在試圖形成 CTC 時關閉
- 真空極化可能產生巨大的正能量阻止時間旅行

### 31.8.3 自舉效應

如果時間機器可能存在，將面臨「自舉效應」問題。

假設你回到過去並阻止自己出生。會發生什麼？
- 如果你阻止了自己，你就不会出生，也就不會回去阻止
- 但如果你沒有阻止，你會出生並回去阻止

這導致邏輯悖論——沒有任何一致的解決方案。

量子力學的觀點：
- 量子態應該保持一致
- 可能存在多重歷史或退相干機制來避免悖論

## 31.9 蟲洞與量子糾纏

### 31.9.1 ER = EPR 猜想

Maldacena 和 Susskind 提出的 ER = EPR 猜想將蟲洞（Einstein-Rosen 橋）與量子糾纏（EPR 對）等同起來。

核心思想：
- 兩個糾纏粒子之間的關聯對應於微觀蟲洞
- 黑洞之間的 EPR 關聯對應於蟲洞連接
- 這可能解釋黑洞信息悖論

### 31.9.2 對蟲洞的影響

ER = EPR 猜想對蟲洞物理學有深遠的影響：

- 蟲洞可能是量子力學中固有的結構
- 可穿越蟲洞可能對應於特殊的量子態
- 時間機器可能與退相干和量子糾纏的結構有關

## 31.10 蟲洞的可視化

### 31.10.1 三維嵌入圖

蟲洞的常見可視化是三維嵌入圖。考慮蟲洞的 $t = \text{const}$、$\theta = \pi/2$ 切片。

度規簡化為：

$$ds^2 = \frac{dr^2}{1 - \frac{b_0^2}{r^2}} + r^2 d\phi^2$$

將其嵌入三維歐幾里得空間：

$$z^2 = r^2 - b_0^2$$

這給出兩個「漏斗」，通過喉部連接。

### 31.10.2 時空圖

在Penrose圖中，蟲洞連接兩個獨立的外部區域。

蟲洞的視圖通常顯示為連接兩個「宇宙」的「橋」。每個嘴巴在各自的宇宙中看起來像是一個黑洞（但沒有事件視界）。

## 31.11 數值計算示例

以下 Python 代碼計算並可視化 Morris-Thorne 蟲洞的嵌入圖：

```python
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

def wormhole_embedding(r, b0, num_slices=20):
    """計算蟲洞嵌入圖的坐標
    r: 徑向座標
    b0: 喉部半徑
    """
    # 計算 z 坐標
    z = np.sqrt(r**2 - b0**2)
    z = np.nan_to_num(z, nan=0)
    return z

# 參數
b0 = 1.0  # 喉部半徑
r = np.linspace(b0, 5, 100)

# 計算嵌入
z = wormhole_embedding(r, b0)

# 2D 截面圖
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.plot(r, z, 'b-', linewidth=2)
plt.plot(r, -z, 'b-', linewidth=2)
plt.xlabel('r')
plt.ylabel('z')
plt.title('蟲洞嵌入圖（2D 截面）')
plt.axhline(y=0, color='k', linestyle='--', alpha=0.3)
plt.grid(True)

# 3D 旋轉圖
ax = plt.subplot(1, 2, 2, projection='3d')

phi = np.linspace(0, 2*np.pi, 50)
R, PHI = np.meshgrid(r, phi)
Z = wormhole_embedding(R, b0)

X = R * np.cos(PHI)
Y = R * np.sin(PHI)

ax.plot_surface(X, Y, Z, alpha=0.7, cmap='Blues')
ax.plot_surface(X, Y, -Z, alpha=0.7, cmap='Reds')

ax.set_xlabel('X')
ax.set_ylabel('Y')
ax.set_zlabel('Z')
ax.set_title('蟲洞嵌入圖（3D 旋轉）')
ax.view_init(elev=30, azim=45)

plt.tight_layout()
plt.savefig('wormhole_embedding.png', dpi=150)
plt.show()

# 潮汐力計算
print("蟲洞潮汐力估計")
print(f"喉部半徑 b0 = {b0}")
print("假設穿越速度 v = 0.9c")
v = 0.9  # c = 1
gamma = 1 / np.sqrt(1 - v**2)
print(f"相對論因子 γ = {gamma:.2f}")
```

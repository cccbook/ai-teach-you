# 第 24 章：黑洞物理

## 概述

黑洞是廣義相對論最驚人和最深遠的預言之一——時空中存在一個連光都無法逃逸的區域。雖然約翰·米歇爾在 1783 年就提出了「暗星」的概念，但直到愛因斯坦發表廣義相對論後，黑洞的現代理論才開始發展。

黑洞物理是廣義相對論中最活躍的研究領域之一，連接了理論物理、天體物理和觀測天文學。2015 年 LIGO 探測到黑洞合併的引力波，2019 年事件視界望遠鏡拍攝到黑洞影像，這些里程碑標誌著黑洞研究進入了一個新時代。

本章將系統介紹黑洞的基本性質、分類、無毛定理和熱力學。

## 24.1 黑洞的定義

### 24.1.1 事件視界

黑洞的邊界稱為**事件視界**（event horizon）——這是因果結構的分界線。

事件視界的定義：時空中所有未來 directed null 的邊界。換句話說，一旦穿越事件視界，就沒有任何世界線能夠回到外部宇宙。

對於 Schwarzschild 黑洞，事件視界位於 $r = 2GM/c^2$。

### 24.1.2 視界不是物理表面

重要的是，事件視界不是物理表面——它沒有物質組成，也不是膜結構。

事件視界是一個**因果邊界**：從視界內部發出的任何信號都無法到達外部。視界內的粒子可以穿過視界向外運動，但這條世界線在有限時間內會遇到奇點。

### 24.1.3 視界的幾何

事件視界是三維超曲面（二維球面沿時間維度的演化）。它的面積定義為黑洞的「大小」：

對於 Schwarzschild 黑洞：
$$A = 4\pi r_s^2 = 16\pi M^2$$

（在自然單位中，$G = c = 1$）

## 24.2 Schwarzschild 黑洞

### 24.2.1 度規

Schwarzschild 度規（在自然單位中）：

$$ds^2 = -\left(1 - \frac{2M}{r}\right) dt^2 + \left(1 - \frac{2M}{r}\right)^{-1} dr^2 + r^2 d\Omega^2$$

其中 $M$ 是黑洞質量。

### 24.2.2 視界

Schwarzschild 黑洞的事件視界位於：
$$r_+ = 2M$$

視界內的區域（$r < 2M$）是黑洞的內部。從視界內部，任何物質或光都無法逃離——它們必然走向奇點 $r = 0$。

### 24.2.3 奇點

$r = 0$ 是時空奇點。在這裡，曲率不變量發散：
$$R_{\mu\nu\rho\sigma}R^{\mu\nu\rho\sigma} \propto \frac{1}{r^6}$$

廣義相對論在奇點處失效，需要量子引力來描述。

## 24.3 黑洞的分類

### 24.3.1 Schwarzschild 黑洞

只有質量 $M$——靜態黑洞。

這是最簡單的黑洞，不旋轉、不帶電。

### 24.3.2 Reissner-Nordström 黑洞

質量 $M$ + 電荷 $Q$——靜態帶電黑洞。

度規函數：
$$f(r) = 1 - \frac{2M}{r} + \frac{Q^2}{r^2}$$

視界位置：
$$r_\pm = M \pm \sqrt{M^2 - Q^2}$$

當 $|Q| = M$ 時，兩個視界合併，形成極端黑洞。

當 $|Q| > M$ 時，沒有視界——裸奇點。

### 24.3.3 Kerr 黑洞

質量 $M$ + 自旋 $J$——旋轉黑洞。

Kerr 度規使用 Boyer-Lindquist 座標，含有描述旋轉的參數 $a = J/M$。

視界位置：
$$r_\pm = M \pm \sqrt{M^2 - a^2}$$

Kerr 黑洞有兩個特點：
- Ergosphere：$r_E = M + \sqrt{M^2 - a^2\cos^2\theta}$
- 環形奇點（ring singularity）

### 24.3.4 Kerr-Newman 黑洞

質量 $M$ + 電荷 $Q$ + 自旋 $J$——最一般的黑洞解。

$$r_\pm = M \pm \sqrt{M^2 - a^2 - Q^2}$$

當 $r_\pm$ 為實數且正時，存在視界；否則形成裸奇點。

## 24.4 無毛定理

### 24.4.1 表述

**無毛定理**（No-Hair Theorem）斷言：

> 黑洞只需三個參數就能完全描述：
> - 質量 $M$
> - 電荷 $Q$
> - 角動量 $J$

換句話說，所有其他的自由度都在黑洞形成過程中通過引力輻射消散了。

### 24.4.2 歷史背景

無毛定理由 Wheeler 在 1960 年代提出，強調黑洞的簡單性。正式的證明需要假設：
- 真空中愛因斯坦-麥克斯韋方程成立
- 黑洞是靜態或穩態的
- 能量條件成立

### 24.4.3 物理意義

無毛定理意味著：
- 黑洞沒有「頭髮」——沒有額外的微觀結構
- 黑洞的外部幾何完全由 $M, Q, J$ 決定
- 落入黑洞的物質的詳細信息丢失了

這與熱力學中的「宏觀態由少數參數描述」類比，引導我們將黑洞視為熱力學系統。

### 24.4.4 對信息悖論的影響

無毛定理暗示落入黑洞的信息丢失了——這是黑洞信息悖論的起源。

然而，量子效應（霍金輻射）可能提供信息守恆的途徑。

## 24.5 黑洞的熱力學

### 24.5.1 面積-熵關係

黑洞的熵與事件視界面積成正比：

$$S = \frac{k_B c^3 A}{4G\hbar} = \frac{A}{4}$$

在自然單位中（$G = \hbar = c = k_B = 1$）：
$$S = \frac{A}{4}$$

這是 Bekenstein-Hawking 熵。

對於 Schwarzschild 黑洞：
$$S = 4\pi M^2$$

### 24.5.2 霍金溫度

黑洞具有溫度——這是量子效應的結果。

表面重力 $\kappa$ 是視界附近的「引力強度」。對於 Schwarzschild 黑洞：
$$\kappa = \frac{1}{4M}$$

霍金溫度：
$$T_H = \frac{\hbar\kappa}{2\pi k_B c} = \frac{\hbar c^3}{8\pi G k_B M}$$

或者（使用自然單位）：
$$T = \frac{1}{8\pi M}$$

黑洞不是完全黑的！它通過量子效應發出熱輻射。

### 24.5.3 面積定理

黑洞的熱力學第二定律：
$$\Delta A \geq 0$$

黑洞的面積永遠不會減少。這與熵增加的類比。

但霍金輻射導致黑洞蒸發，面積應該減少？這暗示需要將霍金輻射的熵考慮進來。

### 24.5.4 黑洞力學定律

黑洞具有與熱力學類似的定律：

| 熱力學定律 | 黑洞力學定律 |
|------------|--------------|
| 第零定律：$T$ 均勻 | $\kappa$ 在視界上為常數 |
| 第一定律：$dE = T dS$ | $dM = \frac{\kappa}{8\pi}dA + \Omega dJ + \Phi dQ$ |
| 第二定律：$S \geq 0$ | $A \geq 0$ |
| 第三定律：$T \neq 0$ | $\kappa \neq 0$ 無法達到 |

## 24.6 霍金輻射

### 24.6.1 量子效應

在彎曲時空中，量子場論預言黑洞會發出粒子！

霍金的推導（1974）：
- 考慮視界附近的量子真空漲落
- 一對虛粒子可能分離，一個落入黑洞，一個逃逸
- 對於外部觀測者，這看起來像熱輻射

### 24.6.2 溫度公式

霍金溫度：
$$T_H = \frac{\hbar c^3}{8\pi G k_B M}$$

對於太陽質量黑洞：
$$T_H \approx 6 \times 10^{-8} \text{ K}$$

這比宇宙微波背景（2.7 K）低得多，所以太陽質量黑洞目前吸收的能量多於發射的能量。

### 24.6.3 功率

霍金輻射的功率：
$$P = \frac{\hbar c^6}{15360\pi G^2 M^2}$$

對於太陽質量黑洞，功率約 $10^{-28}$ W——完全可以忽略。

### 24.6.4 蒸發

黑洞通過霍金輻射逐漸失去質量。蒸發時間：

$$t_{\text{ev}} = \frac{5120\pi G^2 M^3}{\hbar c^4}$$

對於太陽質量黑洞：
$$t_{\text{ev}} \approx 10^{67} \text{ 年}$$

對於原初黑洞（初始質量 $10^{15}$ g）：
$$t_{\text{ev}} \approx 10^{10} \text{ 年}$$

正在蒸發的原初黑洞可能產生可觀測的信號。

## 24.7 黑洞資訊悖論

### 24.7.1 問題

霍金輻射是完全熱的——它不攜帶關於落入黑洞物質的信息。

當黑洞蒸發完畢，霍金輻射攜帶的能量會擴散到宇宙中。但關於初始物質的信息呢？

如果信息丢失了，量子力學的么正性（信息守恆）就被破壞了。

### 24.7.2 可能的解決方案

1. **信息守恆**：霍金輻射不是完全熱的，攜帶隱藏信息
2. **信息丢失**：接受量子力學需要修改
3. **殘餘**：黑洞蒸發後留下微小殘餘，攜帶信息
4. **ER = EPR**：信息通過蟲洞結構保存

### 24.7.3 軟頭髮

Hawking 和 Perry（2016）提出「軟頭髮」假說：
- 視界附近存在低能 Goldstone 模式
- 這些模式存儲落入黑洞的信息
- 霍金輻射最終會讀取這些信息

## 24.8 觀測證據

### 24.8.1 X射線雙星

**天鵝座 X-1**（1978）：
- 最早的恆星質量黑洞候選
- 質量約 $15 M_\odot$
- X射線源來自吸積盤

其他著名的黑洞 X射線雙星：
- GRS 1915+105：具有超常的X射線活動
- GX 339-4：週期性 X射線源

### 24.8.2 銀河系中心

**人馬座 A***：
- 位於銀河系中心
- 質量約 $4 \times 10^6 M_\odot$
- 黑洞的間接證據來自繞它運動的恆星

S2 恆星以約 2000 km/s 的速度近距離經過黑洞，提供了強大的引力約束。

### 24.8.3 引力波

**LIGO 探測**（2015）：
- GW150914：兩個約 $30 M_\odot$ 黑洞合併
- 確認黑洞的存在
- 提供了黑洞質量和自旋的測量

截至 2024 年，LIGO-Virgo-KAGRA 已經探測到超過 100 個引力波事件。

### 24.8.4 事件視界望遠鏡

**M87 黑洞影像**（2019）：
- 位於室女座星系團的 M87 星系中心
- 質量約 $6.5 \times 10^9 M_\odot$
- 顯示明亮的環狀結構，中心是黑暗的陰影

**銀河系中心黑洞**（2022）：
- 人馬座 A* 的首張影像
- 確認了黑洞的存在

這些影像是廣義相對論預言的直接驗證。

## 24.9 數值計算

```python
import numpy as np

def schwarzschild_horizon(M):
    """計算 Schwarzschild 半徑"""
    G = 6.674e-11
    c = 2.998e8
    return 2 * G * M / c**2

def hawking_temperature(M):
    """計算霍金溫度"""
    hbar = 1.055e-34
    G = 6.674e-11
    c = 2.998e8
    k_B = 1.381e-23
    return hbar * c**3 / (8 * np.pi * G * k_B * M)

def evaporation_time(M):
    """計算蒸發時間（秒）"""
    hbar = 1.055e-34
    G = 6.674e-11
    c = 2.998e8
    return 5120 * np.pi * G**2 * M**3 / (hbar * c**4)

# 太陽質量黑洞
M_sun = 1.989e30
M = M_sun

print(f"太陽質量黑洞:")
print(f"  Schwarzschild 半徑: {schwarzschild_horizon(M):.2e} m")
print(f"  霍金溫度: {hawking_temperature(M):.2e} K")
print(f"  蒸發時間: {evaporation_time(M):.2e} 年")

# 原初黑洞
M_ph = 1e15 * 1e3  # 10^15 g = 10^12 kg
print(f"\n原初黑洞 (10^15 g):")
print(f"  Schwarzschild 半徑: {schwarzschild_horizon(M_ph):.2e} m")
print(f"  霍金溫度: {hawking_temperature(M_ph):.2e} K")
print(f"  蒸發時間: {evaporation_time(M_ph):.2e} 年")
```

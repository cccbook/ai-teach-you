# 第 18 章：相對論動力學

## 概述

本章將狹義相對論擴展到動力學領域。在牛頓力學中，質量、能量和動量是獨立的基本概念。但在相對論中，這些概念是相互關聯的——質量是能量的一種形式，而能量和動量構成統一的四維實體。

愛因斯坦的著名公式 $E = mc^2$ 揭示了質量和能量之間的深刻聯繫。這個公式不僅是理論上的結論，而且已被無數實驗驗證——從核反應到粒子物理，從醫學成像到能源生產。

我們將看到四動量如何守恆，以及這些概念如何為廣義相對論中的能量-動量張量奠定基礎。

## 18.1 質能等價

### 18.1.1 質能公式

$$E = mc^2$$

這是相對論最著名的公式，表明質量是能量的一種形式。這不是一個近似的關係，而是精確的等價——質量和能量是同一個物理量的不同表現。

公式的推導：考慮一個靜止質量為 $m_0$ 的物體。當它吸收能量 $E$ 時，它的靜止質量增加到 $m = m_0 + E/c^2$。

### 18.1.2 靜止質量的定義

在相對論中，**靜止質量** $m_0$ 是 Lorentz 不變量——所有觀測者都測量到相同的值：

$$m_0^2 c^4 = E^2 - p^2 c^2$$

這是相對論的核心恆等式。它定義了能量和動量之間的關係。

當粒子靜止（$p = 0$），得到 $E = m_0 c^2$——這就是著名的質能等價公式。

### 18.1.3 質能等價的物理意義

質量和能量可以相互轉換，這在核反應中得到了清晰的驗證：

**核裂變**：當重的原子核（如鈾-235）分裂成較輕的核時，生成物的總靜止質量小於原始核的質量。損失的質量轉化為動能（熱能）和伽馬射線——原子彈和核電廠的能量來源。

**核融合**：當輕核（如氫）融合成較重的核（如氦）時，生成物的總靜止質量也小於反應物的質量。太陽的能量就來自於這種核融合反應。

**粒子-反粒子湮滅**：當電子和正電子相遇時，它們會湮滅，轉化為光子。電子和正電子的靜止質量完全轉化為能量。

## 18.2 四動量

### 18.2.1 四動量的定義

四動量是四維動量向量，通過四速度和靜止質量定義：

$$P^\mu = m_0 U^\mu = m_0 \gamma(c, \vec{v}) = (\gamma m_0 c, \gamma m_0 \vec{v})$$

將能量和動量寫在一起：

$$P^\mu = (E/c, p_x, p_y, p_z)$$

其中：
- $E = \gamma m_0 c^2$ 是總能量
- $\vec{p} = \gamma m_0 \vec{v}$ 是三維動量

### 18.2.2 四動量的分量

四動量的四個分量：
$$P^\mu = (P^0, P^i) = (E/c, p_x, p_y, p_z)$$

在粒子的瞬時共動系中，四動量為：
$$P^\mu = (m_0 c, 0, 0, 0)$$

這顯示靜止質量 $m_0$ 是四動量的「長度」（模方）。

### 18.2.3 質能關係

從四動量的歸一化：

$$P^\mu P_\mu = -m_0^2 c^2$$

得到能量-動量關係：

$$E^2 = p^2 c^2 + m_0^2 c^4$$

或者寫成：

$$E = \sqrt{p^2 c^2 + m_0^2 c^4}$$

這個關係是狹義相對論的基本方程，取代了牛頓力學中的 $E = p^2/2m$。

## 18.3 能量與動量

### 18.3.1 能量

總能量包括靜止能量和動能：

$$E = \gamma m_0 c^2 = mc^2$$

其中 $m = \gamma m_0$ 有時被稱為「相對論質量」。

動能是總能量與靜止能量之差：

$$K = E - m_0 c^2 = (\gamma - 1)m_0 c^2$$

### 18.3.2 動量

動量的定義：

$$\vec{p} = \gamma m_0 \vec{v} = m\vec{v}$$

注意：這與牛頓力學的形式相同，但質量被相對論質量替換。

動量的大小：

$$p = \gamma m_0 v = \frac{m_0 v}{\sqrt{1 - v^2/c^2}}$$

當 $v \ll c$ 時，$p \approx m_0 v$，回到牛頓結果。

### 18.3.3 動能的表達

動能展開為幂级数：

$$K = (\gamma - 1)m_0 c^2 = \left(\frac{1}{\sqrt{1 - v^2/c^2}} - 1\right)m_0 c^2$$

當 $v \ll c$：

$$K \approx \left(\frac{1}{2}\frac{v^2}{c^2} - \frac{3}{8}\frac{v^4}{c^4} + \cdots\right)m_0 c^2 \approx \frac{1}{2}m_0 v^2$$

這還原為牛頓動能公式。

### 18.3.4 極限行為

當 $v \to c$ 時：
- $\gamma \to \infty$
- $E \to \infty$
- $p \to \infty$
- 需要無限能量才能將有質量粒子加速到光速

這與光子形成對比——光子的靜止質量為零，因此總是以光速運動，不需要加速過程。

## 18.4 四力和功率

### 18.4.1 四力

四力定義為四動量的變化率：

$$F^\mu = \frac{dP^\mu}{d\tau}$$

這是牛頓第二定律的四維推廣。

四力也可以寫成：

$$F^\mu = m_0 A^\mu$$

其中 $A^\mu$ 是四加速度。

### 18.4.2 與三維力的關係

四力的空間分量與牛頓力的關係：

$$\vec{F} = \frac{d\vec{p}}{dt} = \frac{d}{dt}(\gamma m_0 \vec{v})$$

當力與速度方向不同時，需要分解為平行和垂直分量：

$$F^\mu = \gamma\left(\frac{\vec{F}\cdot\vec{v}}{c}, \vec{F}_\parallel + \gamma \vec{F}_\perp\right)$$

或者，用加速度表示：

$$\vec{F} = \gamma^3 m_0 \vec{a}_\parallel + \gamma m_0 \vec{a}_\perp$$

這與牛頓第二定律的形式不同——當速度很大時，縱向和橫向的「有效質量」不同。

### 18.4.3 功率

四力的時間分量給出功率：

$$F^0 = \frac{dP^0}{d\tau} = \frac{1}{c}\frac{dE}{d\tau} = \frac{\gamma}{c}\frac{dE}{dt}$$

因此：

$$\frac{dE}{dt} = \vec{F}\cdot\vec{v}$$

這表明功率是力在速度方向的投影——與牛頓力學完全一致。

## 18.5 能量-動量守恆

### 18.5.1 四動量守恆

在孤立系統中，總四動量守恆：

$$P^\mu_{\text{total}} = \sum_i P^\mu_i = \text{常數}$$

這個守恆定律是：
- 四個分量都守恆
- 包括能量守恆和三個動量分量守恆
- 在所有慣性系中都成立

### 18.5.2 應用：粒子碰撞

在粒子碰撞過程中，總四動量守恆可以用來分析碰撞結果。

**彈性碰撞**：動能也守恆
**非彈性碰撞**：動能可能不守恆，但總能量（包括靜止能量）守恆

例如，在粒子物理實驗中，兩個粒子碰撞後可能產生新的粒子——新粒子的靜止質量來自於碰撞粒子的動能。

### 18.5.3 應用：粒子衰變

不穩定粒子會自發衰變為多個較輕的粒子。衰變前後，總四動量守恆。

例如，π介子衰變為μ子和中微子：
$$\pi^- \to \mu^- + \bar{\nu}_\mu$$

利用四動量守恆，可以計算衰變產物的能量和動量。

### 18.5.4 應用：能量產生

在核反應中，質量亏损轉化為能量：

$$E_{\text{released}} = \Delta m \cdot c^2$$

其中 $\Delta m$ 是反應前後的靜止質量差。

例如，在氘-氚融合反應中：
$$d + t \to \alpha + n$$

質量亏损约为 0.01863 u，释放能量约 17.6 MeV。

## 18.6 零質量粒子

### 18.6.1 光子

對光子，靜止質量 $m_0 = 0$，但有能量和動量：

$$E = pc$$

從能量-動量關係：

$$E^2 = p^2 c^2 + m_0^2 c^4 \Rightarrow E^2 = p^2 c^2 \Rightarrow E = pc$$

光子的速度總是 $c$，不需要加速過程。

光子的能量與頻率成正比：

$$E = h\nu = \hbar\omega$$

其中 $h$ 是普朗克常數。

### 18.6.2 廣義公式

對任意粒子：

$$E^2 = p^2 c^2 + m_0^2 c^4$$

- 當 $m_0 > 0$：粒子有靜止質量，可以靜止
- 當 $m_0 = 0$：粒子必須以光速運動

### 18.6.3 光子的四動量

光子的四動量：

$$P^\mu = (E/c, E/c \cdot \hat{n})$$

其中 $\hat{n}$ 是傳播方向的單位向量。

光子的四動量是零長度的（因為 $m_0 = 0$）：

$$P^\mu P_\mu = E^2/c^2 - E^2/c^2 = 0$$

## 18.7 能量-動量張量

### 18.7.1 理想流體

在廣義相對論中，連續介質的能量-動量張量為：

$$T^{\mu\nu} = (\rho + p) U^\mu U^\nu + p g^{\mu\nu}$$

其中：
- $\rho$ 是固有能量密度
- $p$ 是壓強
- $U^\mu$ 是流體四速度

這個張量是對稱的、協變散度為零：

$$\nabla_\mu T^{\mu\nu} = 0$$

### 18.7.2 質能等價的幾何化

能量-動量張量在廣義相對論中作為場方程的源項：

$$G_{\mu\nu} = \frac{8\pi G}{c^4} T_{\mu\nu}$$

這是愛因斯坦場方程的物理意義——時空幾何（$G_{\mu\nu}$）由物質和能量（$T_{\mu\nu}$）決定。

### 18.7.3 電磁場的能量-動量張量

電磁場的能量-動量張量為：

$$T^{\mu\nu} = \frac{1}{\mu_0}\left(F^{\mu\alpha}F^\nu_{\ \alpha} - \frac{1}{4}\eta^{\mu\nu}F_{\alpha\beta}F^{\alpha\beta}\right)$$

這個張量描述：
- $T^{00}$：電磁能量密度
- $T^{0i}$：Poynting 向量（能量流密度）
- $T^{ij}$：Maxwell 應力張量

## 18.8 粒子系統的質心運動

### 18.8.1 總四動量

對於由多個粒子組成的系統，總四動量為：

$$P^\mu_{\text{total}} = \sum_i P^\mu_i = \left(\frac{E_{\text{total}}}{c}, \vec{p}_{\text{total}}\right)$$

這個總四動量決定了系統的「整體運動」。

### 18.8.2 質心參考系

存在一個特殊的慣性系（質心系或 center-of-momentum frame），在其中總動量為零：

$$\vec{p}_{\text{total}} = 0$$

在這個參考系中，總能量正好是系統的不變質量：

$$M_{\text{total}} c^2 = E_{\text{total}}$$

### 18.8.3 不變質量

系統的不變質量（也稱為「有效質量」）為：

$$M_{\text{total}}^2 c^4 = E_{\text{total}}^2 - p_{\text{total}}^2 c^2$$

注意：這個不變質量不一定等於各粒子靜止質量的和！例如，兩個相向運動的光子具有總動量為零，但不變質量為 $2E/c^2$。

## 18.9 數值計算示例

```python
import numpy as np

def lorentz_factor(beta):
    """計算 Lorentz 因子"""
    return 1.0 / np.sqrt(1 - beta**2)

def relativistic_energy(m0, v, c=1):
    """計算總能量"""
    gamma = lorentz_factor(v/c)
    return gamma * m0 * c**2

def relativistic_momentum(m0, v, c=1):
    """計算動量"""
    gamma = lorentz_factor(v/c)
    return gamma * m0 * v

def kinetic_energy(m0, v, c=1):
    """計算動能"""
    gamma = lorentz_factor(v/c)
    return (gamma - 1) * m0 * c**2

# 電子參數
m_e = 9.109e-31  # kg
c = 2.998e8  # m/s

# 不同速度下的能量
velocities = [0.1, 0.5, 0.9, 0.99, 0.999]

print("電子在不同速度下的能量：")
print(f"靜止能量: {m_e * c**2:.4e} J = {m_e * c**2 / 1.602e-13:.4f} MeV")
print()

for v in velocities:
    E = relativistic_energy(m_e, v, c)
    K = kinetic_energy(m_e, v, c)
    gamma = lorentz_factor(v)
    print(f"v = {v:.3f}c:")
    print(f"  γ = {gamma:.4f}")
    print(f"  總能量 E = {E:.4e} J = {E/(1.602e-13):.4f} MeV")
    print(f"  動能 K = {K:.4e} J = {K/(1.602e-13):.4f} MeV")
    print()

# 質能轉換
mass_kg = 1.0  # 1 kg
energy_joules = mass_kg * c**2
energy_tnt = energy_joules / 4.184e9  # TNT當量

print(f"1 kg 質量完全轉化為能量：")
print(f"  = {energy_joules:.4e} J")
print(f"  = {energy_tnt:.4f} 噸 TNT")
```

# 第九章：喬姆斯基階層

## 9.1 諾姆·喬姆斯基：語言學革命

諾姆·喬姆斯基（Noam Chomsky，1928-）是二十世紀最具影響力的語言學家和思想家之一，他的工作徹底改變了我們對人類語言的理解。喬姆斯基不僅是生成語法（Generative Grammar）的創始人，還是著名的政治評論家和公共知識分子。他的理論不僅影響了語言學，還深刻地影響了計算機科學、認知心理學和哲學。

喬姆斯基1928年12月7日出生於美國費城，他的父親是一位希伯來語學者，母親是一位教育工作者。喬姆斯基從小就展現出極強的語言天賦和學術能力。1945年，17歲的喬姆斯基進入賓夕法尼亞大學學習，最初學習哲學和語言學，後來轉向語言學。1951年，喬姆斯基在賓夕法尼亞大學獲得碩士學位，1955年獲得博士學位，其博士論文題為「変換分析」（Transformational Analysis），這篇論文奠定了他後來發展的生成轉換語法的基础。

1957年，喬姆斯基發表了劃時代的著作《句法結構》（Syntactic Structures），這本書徹底顛覆了當時的語言學理論。喬姆斯基提出，語言能力是人類心智的內在特徵，而不是後天學習的結果。他認為，人類大腦中存在一個「語言器官」，它使人類能夠學習和使用語言。這種觀點被稱為「普遍語法」（Universal Grammar）假說。

喬姆斯基在計算機科學領域的貢獻同樣重要。1956年，他發表了「語言的數學模型」（Three Models for the Description of Language）這篇論文，提出了形式語言的數學理論。在這篇論文中，他定義了現在我們所說的「喬姆斯基階層」（Chomsky Hierarchy），這是一個根據語法生成能力對形式語言進行的分類。這個階層與計算理論密切相關：不同類型的形式語言對應於不同類型的計算模型。

## 9.2 形式語言的基本概念

在我們深入討論喬姆斯基階層之前，讓我們先建立一些基本概念。形式語言（Formal Language）是計算機科學和數理邏輯中的核心概念，它指的是一個由字符串组成的集合，這些字符串由某個字母表（alphabet）中的符號構成。

**字母表（Alphabet）** 是一個非空的有限符號集合。例如，Σ = {0, 1} 是一個二進制字母表，Σ = {a, b, c, d, e, f, ..., z} 是一個小寫英語字母表。字母表通常用 Σ 表示。

**字符串（String）** 是由字母表中的符號組成的有限序列。例如，01011 是二進制字母表上的一個字符串。空字符串記為 ε（epsilon），它是長度為 0 的字符串。

**語言（Language）** 是某個字母表上字符串的集合。例如，語言 L = {0ⁿ1ⁿ | n ≥ 1} = {01, 0011, 000111, ...}。語言可以是有限的（如 L = {hello, world}），也可以是無限的（如上述例子）。

**語法（Grammar）** 是生成語言的裝置。一個形式語法由以下部分組成：
- 終結符集合 V_t：最終出現在字符串中的符號
- 非終結符集合 V_n：用於幫助定義結構的輔助符號
- 產生式集合 P：描述如何將一個字符串轉換為另一個字符串的規則
- 起始符 S：推導的起點

形式語法的工作方式是：從起始符 S 出發，反覆應用產生式規則，最終得到一個只包含終結符的字符串。所有可以這樣導出的字符串構成的集合就是該語法生成的語言。

## 9.3 喬姆斯基階層的層次結構

諾姆·喬姆斯基在1956年的論文中提出了形式語言的階層分類，這就是著名的「喬姆斯基階層」（Chomsky Hierarchy）。這個階層由四個層次組成，每一層次對應於一類具有不同生成能力的語法。

**第零型：無限制文法（Unrestricted Grammar）**

無限制文法是最強大的語法類型。在無限制文法中，產生式規則沒有任何限制，可以是任何形式的 α → β，其中 α 和 β 是任意非空字符串，且 α 至少包含一個非終結符。

無限制文法可以生成所有遞歸可枚舉語言（recursively enumerable language）。也就是說，一個語言是遞歸可枚舉的，當且僅當存在一個無限制文法可以生成它。這與圖靈機的計算能力完全一致。

**第一型：上下文敏感文法（Context-Sensitive Grammar）**

上下文敏感文法的產生式規則具有形式 αAβ → αγβ，其中 A 是一個非終結符，α, β, γ 是任意字符串（γ 不能為空）。這意味著，規則的應用「取決於」上下文——只有當 A 處於由 α 和 β 確定的上下文中時，才能將其替換為 γ。

等價地，我們可以說上下文敏感文法是這樣的文法：所有產生式的左邊字符串的長度不大於右邊字符串的長度。這個條件確保了推導過程中字符串的長度只會增加（或保持不變），不會減少。

上下文敏感語言對應於線性有界自動機（Linear Bounded Automaton），這是一種受限的圖靈機，其讀寫頭只能在輸入的範圍內移動，不能超出輸入的左右邊界。

**第二型：上下文無關文法（Context-Free Grammar）**

上下文無關文法是我們最常見的語法類型。它的產生式規則具有簡單的形式 A → α，其中 A 是一個非終結符，α 是任意字符串（可以包含終結符和非終結符）。這種規則不考慮「上下文」——無論 A 出現在什麼位置，都可以將其替換為 α。

上下文無關語言對應於下推自動機（Pushdown Automaton），這是一種配備了「棧」的有限狀態機。棧提供了額外的存儲能力，使機器能夠處理某些需要「記憶」的語言，如匹配括號的語言 {ⁿbⁿ | n ≥ 1}。

上下文無關文法在程序設計語言的語法描述中有著廣泛的應用。幾乎所有主流編程語言的語法都是用上下文無關文法（或其擴展）來描述的。

**第三型：正規文法（Regular Grammar）**

正規文法是最簡單的語法類型，它的產生式規則具有受限的形式：
- A → aB 或 A → a（A, B 是非終結符，a 是終結符）
- 或者 A → ε

正規文法也可以分為「右線性文法」和「左線性文法」，兩者是等價的。

正規語言對應於有限狀態自動機（Finite State Automaton），這是一種沒有額外存儲設備的簡單計算模型。正規語言在計算機科學中有著廣泛的應用，包括正則表達式、詞法分析器設計等。

## 9.4 層次結構的包含關係

喬姆斯基階層的一個重要性質是：各層次之間存在嚴格的包含關係。讓我們用集合論的語言來描述：

- 正規語言 ⊂ 上下文無關語言 ⊂ 上下文敏感語言 ⊂ 遞歸可枚舉語言

這個包含關係是「真」的——每一層都存在語言不被下一層包含。讓我們給出一些例子：

- 語言 L = {0ⁿ1ⁿ | n ≥ 1} 是上下文無關但不是正規的。這可以用泵引理（pumping lemma）來證明。
- 語言 L = {ww | w ∈ {0,1}*} 是上下文敏感但不是上下文無關的。這意味著「判斷一個字符串是否是兩個相同字符串的連接」需要上下文敏感的能力，而不是上下文無關的能力。
- 語言 L = {w | w 編碼的圖靈機最終會停機} 是遞歸可枚舉但不是遞歸的（即不可判定的）。

這種層次結構與計算模型之間有著美麗的對應關係：
- 正規語言 ↔ 有限狀態自動機
- 上下文無關語言 ↔ 下推自動機
- 上下文敏感語言 ↔ 線性有界自動機
- 遞歸可枚舉語言 ↔ 圖靈機

## 9.5 正規語言與有限自動機

正規語言是喬姆斯基階層中最簡單的層次，但它們在計算機科學中有著極其重要的應用。讓我們更詳細地探討這類語言。

**有限自動機（Finite Automaton）** 是識別正規語言的計算模型。有兩種等價的定義方式：確定性有限自動機（DFA）和非確定性有限自動機（NFA）。

確定性有限自動機可以定義為一個五元組 M = (Q, Σ, δ, q₀, F)，其中：
- Q 是有限狀態集合
- Σ 是輸入字母表
- δ : Q × Σ → Q 是轉換函數
- q₀ ∈ Q 是初始狀態
- F ⊆ Q 是接受狀態集合

確定性有限自動機的工作方式很簡單：從初始狀態出發，讀取輸入字符串的每一個符號，根據轉換函數更新當前狀態。如果讀完整個字符串後，機器處於接受狀態，則接受該字符串；否則拒絕。

非確定性有限自動機放寬了確定性的要求：在同一個狀態下，對於同一個輸入符號，可能有多個可能的轉換。形式上，NFA 的轉換函數 δ : Q × Σ → P(Q)，其中 P(Q) 是 Q 的冪集。

有趣的是，確定性和非確定性有限自動機是等價的——對於任何 NFA，都存在一個 DFA 識別同樣的語言。這可以通過「子集構造」技術來證明。

**正則表達式（Regular Expression）** 是描述正規語言的另一種強大工具。正則表達式使用以下運算符：
- 連接（concatenation）：ab 表示字符串 "ab"
- 或（union）：a|b 表示 a 或 b
- 克林閉包（Kleene star）：a* 表示任意多個 a（包括 0 個）

例如，正則表達式 (0|1)*1(0|1)* 表示所有包含至少一個 1 的二進制字符串。

## 9.6 上下文無關語言與下推自動機

上下文無關語言是喬姆斯基階層中極其重要的一類，它們與程序設計語言的語法有著直接的聯繫。

**下推自動機（Pushdown Automaton, PDA）** 是識別上下文無關語言的計算模型。與有限自動機相比，下推自動機多了一個「棧」（stack），這是一種後進先出（LIFO）的存儲設備。棧提供了額外的存儲能力，使機器能夠「記住」某些信息。

下推自動機的形式定義包括：
- 有限狀態集合 Q
- 輸入字母表 Σ
- 棧字母表 Γ
- 轉換函數 δ : Q × (Σ ∪ {ε}) × (Γ ∪ {ε}) → P(Q × (Γ ∪ {ε}))
- 初始狀態 q₀ ∈ Q
- 初始棧符號 Z₀ ∈ Γ
- 接受狀態集合 F ⊆ Q

下推自動機可以是非確定性的。非確定性下推自動機可以識別所有上下文無關語言，但確定性下推自動機只能識別確定性上下文無關語言（這是上下文無關語言的一個真子集）。

**泵引理（Pumping Lemma）** 是證明某些語言不是上下文無關的重要工具。泵引理說：對於任何上下文無關語言 L，存在一個「泵長度」p，使得任何長度 ≥ p 的字符串 w ∈ L 都可以被分解為 w = uvxyz，其中 |vxy| ≤ p，且 vy 不都為空，並且對於所有 i ≥ 0，字符串 uvⁱxyⁱz 都在 L 中。

這個引理可以用來證明語言 {0ⁿ1ⁿ | n ≥ 1} 不是正規的（但它是上下文無關的），以及語言 {ww | w ∈ {0,1}*} 不是上下文無關的。

## 9.7 上下文敏感語言與線性有界自動機

上下文敏感語言是喬姆斯基階層中介於上下文無關語言和遞歸可枚舉語言之間的一類。它們比上下文無關語言更強大，但比遞歸可枚舉語言受到更多限制。

**線性有界自動機（Linear Bounded Automaton, LBA）** 是識別上下文敏感語言的計算模型。從本質上說，線性有界自動機是一個受限的圖靈機：它的讀寫頭不能離開輸入的範圍，這意味著它使用的空間與輸入長度成線性關係。

形式上，確定性線性有界自動機可以定義為一個七元組 M = (Q, Σ, Γ, δ, q₀, B, F)，其中：
- Q 是有限狀態集合
- Σ 是輸入字母表
- Γ 是帶字母表，包含 Σ 和空格符 B
- δ : Q × Γ → Q × Γ × {L, R} 是轉換函數（受限，不能寫入空格）
- q₀ 是初始狀態
- B 是空格符
- F 是接受狀態集合

上下文敏感語言的一個重要例子是語言 L = {aⁿbⁿcⁿ | n ≥ 1}。這個語言要求三個符號的數量相等，它不能由上下文無關文法生成，但可以由上下文敏感文法生成。

有一個著名的未解決問題——「LBA 問題」：確定性 LBA 和非確定性 LBA 是否識別同樣的語言？這個問題直到2007年才被證實：確定性和非確定性 LBA 確實識別同樣的語言。

## 9.8 形式語言與計算理論的統一

喬姆斯基階層的美妙之處在於它與計算理論的緊密聯繫。這種聯繫揭示了語言的複雜性與計算的複雜性之間的深刻對應關係。

讓我們總結一下對應關係：
- **正規語言 ↔ 有限狀態自動機 ↔ 有界空間（O(1)）**
- **上下文無關語言 ↔ 下推自動機 ↔ 有界空間（O(n)）**
- **上下文敏感語言 ↔ 線性有界自動機 ↔ 線性空間**
- **遞歸可枚舉語言 ↔ 圖靈機 ↔ 無限制空間**

這種對應關係不僅有理論意義，還有實踐價值。它告訴我們，不同類型的語言需要不同複雜度的計算模型來處理。在設計編程語言時，我們可以使用正規語言來處理詞法分析（因为词法规则相对简单），使用上下文無關語言來處理語法分析（因为语法规则需要更强大的表示能力）。

喬姆斯基階層的提出標誌著理論計算機科學的一個重要里程碑。它連接了形式語言理論與自動機理論，為後來的編譯器設計、語言處理、計算複雜度理論等領域奠定了基礎。

## 9.9 小結

本章我們詳細介紹了喬姆斯基階層，這是形式語言理論中的核心概念。我們從喬姆斯基的生平開始，講述了他在語言學和計算機科學領域的貢獻。我們定義了形式語言和語法的基本概念，然後詳細介紹了階層的四個層次：正規文法、上下文無關文法、上下文敏感文法和無限制文法。我們還討論了這些語法類型與相應計算模型之間的對應關係。

喬姆斯基階層揭示了語言複雜性與計算能力之間的深刻聯繫。它告訴我們，不同複雜度的語言需要不同能力的計算模型來識別。這個發現對於計算機科學的許多領域——從編譯器設計到人工智能——都有深遠的影響。

在下一章中，我們將轉向計算複雜度理論，特別是 P vs NP 問題和 NP-完全性理論。這些問題是當代計算科學面臨的最重要的理論挑戰之一。

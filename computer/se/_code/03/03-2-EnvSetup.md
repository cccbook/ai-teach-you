## Cursor 快速上手

### 基本快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| Cmd+K | 輸入 Prompt |
| Cmd+L | 開啟 Chat |
| Cmd+I | 開啟 Composer |
| Cmd+Enter | 接受建議 |
| Esc | 拒絕建議 |

### 設定步驟

```bash
# 1. 下載 Cursor
open https://cursor.sh/

# 2. 安裝並開啟

# 3. 登入 GitHub 帳號

# 4. 開啟專案資料夾
File → Open Folder

# 5. 開始使用
#   - Cmd+K: 輸入你的需求
#   - 例如："建立一個 REST API"
```

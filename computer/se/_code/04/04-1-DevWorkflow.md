# 一人團隊開發流程

> 來源：第 4 章「單一 Agent 開發」

---

## 每日開發循環

```
09:00  - 任務規劃（15 分鐘）
        向 AI 描述今日目標

09:15  - AI 開發（3 小時）
        AI 生成程式碼
        人工審核修正

12:00  - 午餐休息

14:00  - 測試時間（1 小時）
        AI 生成測試
        執行驗證

15:00  - 部署時間（30 分鐘）
        部署到測試環境

15:30  - 文件時間（30 分鐘）
        更新 README
        記錄問題

16:00  - 審核時間（1 小時）
        檢視 AI 產出
        品質把關
```

---

## 開發檢查清單

- [ ] 任務目標清晰
- [ ] AI Prompt 具體明確
- [ ] 程式碼邏輯正確
- [ ] 命名清晰易懂
- [ ] 錯誤處理完善
- [ ] 測試覆蓋主要功能
- [ ] 部署成功
- [ ] 功能驗證通過

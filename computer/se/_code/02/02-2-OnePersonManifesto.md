# 一人團隊宣言

> 來源：第 2 章「為何一人團隊」

---

## 核心理念

我是獨立開發者，我相信：

1. **AI 是最強大的開發夥伴**
   - 它可以處理 80% 的技術工作
   - 我專注於 20% 的策略決策

2. **速度勝於完美**
   - 先上線，再迭代
   - 完美是持續改進的結果

3. **外包彌補 AI 不足**
   - AI + 外包 = 完整能力覆蓋
   - 專業的事交給專業的人

4. **人類的獨特價值**
   - 創意與審美
   - 商業判斷
   - 人際溝通

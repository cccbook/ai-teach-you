# ccc: 你們改程式的時候，如果只要改其中一小部分，但整個檔案很大，你們會如何下 shell 指令去讀取，修改，和寫入呢？

针对大文件只修改一小部分，常用方法：
读取特定部分
- head -n 100 file - 前100行
- tail -n 50 file - 后50行  
- sed -n '100,200p' file - 第100-200行
修改一小部分
- sed -i '100s/old/new/' file - 修改第100行
- sed -i '100,200s/old/new/g' file - 修改指定范围
- awk 'NR==100 {gsub(/old/,"new")} 1' file - 条件修改
更精确的修改（行内替换）
# 替换第50行的特定内容
sed -i '50s/要改的内容/新内容/' largefile.txt
只写入修改部分
- 使用 sed -i 直接原文件修改
- 或 sed ... file > newfile 输出到新文件
关键技巧：先用 grep -n 或 sed -n 找到目标位置，再用行号定位修改，避免读取整个文件。

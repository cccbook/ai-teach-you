# 專有名詞索引

## 程式語言

* [C 語言](c.md) : 1972 年由 Dennis Ritchie 發明的系統程式語言。
* [LLVM IR](llvm-ir.md) : LLVM 中間表示，一種與目標機器無關的中間碼。

## 編譯器與直譯器

* [編譯器 (Compiler)](compiler.md) : 將原始碼翻譯為機器碼的工具。
* [連結器 (Linker)](linker.md) : 將多個目的檔合併為可執行檔的工具。
* [虛擬機 (Virtual Machine)](vm.md) : 軟體實現的 CPU，執行位元碼。

## 編譯器元件

* [詞法分析 (Lexical Analysis)](lexer.md) : 將字元序列轉換為 Token 序列。
* [語法分析 (Syntax Analysis)](parser.md) : 檢查 Token 序列是否符合語法規則。
* [語意分析 (Semantic Analysis)](semantic.md) : 檢查程式的語義正確性。
* [最佳化 (Optimization)](optimization.md) : 改進程式碼效能的過程。

## 資料結構

* [抽象語法樹 (AST)](ast.md) : 程式碼的樹狀表示。
* [符號表 (Symbol Table)](symbol-table.md) : 儲存識別字資訊的資料結構。

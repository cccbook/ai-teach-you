> 來源：第 9 章「案例：企業 AI 代理」

## 安全架構

### 網路安全

| 層面 | 措施 |
|------|------|
| 防火牆 | WAF + 雲端防火牆 |
| 加密 | TLS 1.3 |
| DDoS | CDN 保護 |

### 應用安全

| 項目 | 措施 |
|------|------|
| 認證 | OAuth 2.0 + JWT |
| 授權 | RBAC |
| 輸入驗證 | 白名單過濾 |
| API 安全 | Rate Limiting |

### 資料安全

| 項目 | 措施 |
|------|------|
| 儲存加密 | AES-256 |
| 傳輸加密 | TLS |
| 脫敏處理 | 敏感資料遮罩 |
| 備份 | 異地備份 |

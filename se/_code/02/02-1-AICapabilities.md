## AI 擅長的任務矩陣

| 任務類型 | 擅長程度 | 範例 |
|----------|----------|------|
| 程式碼生成 | ⭐⭐⭐⭐⭐ | 根據描述生成 REST API |
| 測試生成 | ⭐⭐⭐⭐ | 生成單元測試案例 |
| 重構程式 | ⭐⭐⭐⭐ | 提取函數、改善命名 |
| 除錯 Debug | ⭐⭐⭐⭐ | 分析堆疊追蹤錯誤 |
| 文件生成 | ⭐⭐⭐⭐ | 生成 API 文件 |
| 程式碼翻譯 | ⭐⭐⭐⭐ | JS → TypeScript |
| 效能優化 | ⭐⭐⭐ | 建議優化點 |
| 架構建議 | ⭐⭐⭐ | 給出技術選型建議 |

---

## AI 不擅長的任務矩陣

| 任務類型 | 限制 | 替代方案 |
|----------|------|----------|
| 產品策略 | 缺乏商業直覺 | 人類決策 |
| UI/UX 設計 | 審美有限 | 專業設計師 |
| 客戶溝通 | 無情感理解 | 人類處理 |
| 複雜架構 | 可能過度設計 | 技術顧問 |
| 資安審計 | 可能有盲點 | 滲透測試 |
| 法務合規 | 可能出錯 | 專業律師 |
| 原型設計 | 創意有限 | 設計師原創 |
| 談判協商 | 無情感智慧 | 人類談判 |

---

## 能力評估 Prompt

```
請評估以下任務是否適合由 AI 完成：

任務：[描述你的任務]

請從以下維度評分（1-5）：
1. 規則明確程度
2. 創意要求程度
3. 情感理解需求
4. 安全風險程度
5. 長期維護需求

最後給出建議：AI 適合度（高/中/低）+ 原因。
```

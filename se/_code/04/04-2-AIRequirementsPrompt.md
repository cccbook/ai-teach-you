# AI 輔助需求分析 Prompt

> 來源：第 4 章「單一 Agent 開發」

---

## 需求分析 Prompt 模板

### 基本模板

```
我要建立一個 [產品類型]，
目標用戶是 [描述目標用戶]，
核心功能是 [列出功能]，
請幫我：
1. 細化需求
2. 產出使用者故事
3. 列出 MVP 功能優先順序
```

### 實際範例

```
我要建立一個線上代購平台，
目標用戶是海外代購者和國內買家，
核心功能是：
- 商品搜尋與展示
- 訂單管理
- 支付整合
- 物流追蹤

請幫我：
1. 細化需求
2. 產出使用者故事
3. 列出 MVP 功能優先順序
```

### 輸出格式

```
## 細化需求
...

## 使用者故事
1. 作為 [角色]，我想 [功能]，以便 [價值]
...

## MVP 功能優先順序
### P0 - 必要功能
- [功能 1]
- [功能 2]

### P1 - 重要功能
- [功能 3]
- [功能 4]

### P2 - 錦上添花
- [功能 5]
```

## AI Prompt 範例

### 市場分析 Prompt

```
請幫我分析三家禮品電商網站的特色：
1. GiftNow.io
2. PresentBox
3. Fancy

請用表格呈現功能比較、重點差異化、建議。
```

### 程式碼生成 Prompt

```
請用 FastAPI 建立一個商品 API，包含：
1. GET /products - 商品列表
2. GET /products/{id} - 商品詳情
3. POST /orders - 創建訂單

使用 SQLAlchemy + SQLite。
```

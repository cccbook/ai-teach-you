## 技術選型矩陣

### 前端技術比較

| 技術 | 學習成本 | AI 擅長度 | 生態規模 | 適合規模 |
|------|----------|-----------|----------|----------|
| Next.js | 中 | 高 | 大 | 任何 |
| Nuxt.js | 中 | 中 | 中 | 任何 |
| Vue 3 | 低 | 中 | 中 | 小型 |
| React | 高 | 高 | 大 | 任何 |
| 純 HTML | 低 | 低 | - | 微型 |

**選擇：Next.js 14**

理由：Claude Code 對 Next.js 生成品質最高，生態完整

### 後端技術比較

| 技術 | 學習成本 | AI 擅長度 | 效能 | 適合場景 |
|------|----------|-----------|------|----------|
| FastAPI | 低 | 高 | 高 | API 優先 |
| Express | 中 | 高 | 中 | 全端 JS |
| Django | 中 | 中 | 中 | 全功能 |
| Flask | 低 | 中 | 中 | 微型 |

**選擇：FastAPI**

理由：AI 生成 Python 程式碼品質極高，類型支援好

### 資料庫技術比較

| 資料庫 | 類型 | 設定複雜度 | 免費額度 | 適合階段 |
|--------|------|------------|----------|----------|
| SQLite | 檔案 | 極簡 | 免費 | MVP |
| PostgreSQL | 關聯 | 中 | Render 免費 | 生產 |
| MongoDB | 文件 | 中 | Atlas 免費 | 文件資料 |
| Supabase | 全端 | 簡單 | 免費額度大 | 快速開發 |

**選擇：SQLite (MVP) → PostgreSQL (後期)**

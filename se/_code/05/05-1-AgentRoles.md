# Agent 分工模式

> 來源：第 5 章「多 Agent 協作」

---

## Agent 角色矩陣

| Agent | 主要職責 | 擅長領域 | 工具 |
|-------|----------|----------|------|
| Planner | 任務分解、規劃 | 邏輯思考 | Claude |
| Executor | 程式碼生成 | 程式開發 | Cursor |
| Reviewer | 程式碼審核 | 品質把關 | Claude |
| Tester | 測試生成 | 測試設計 | Claude |
| Documenter | 文件生成 | 文件寫作 | Claude |
| Deployer | 部署自動化 | DevOps | Claude |

---

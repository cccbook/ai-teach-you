# 訪談分析 Prompt

> 來源：第 11 章「使用者研究」

---

## 分析 Prompt 模板

```
請分析以下訪談記錄：

[訪談內容]

請產出：

## 1. 主要發現
- 發現 1
- 發現 2

## 2. 痛點分類
| 痛點 | 嚴重性 | 頻率 |
|------|--------|------|
| 痛點 1 | 高 | 經常 |

## 3. 機會點建議
- 建議 1
- 建議 2

## 4. 驗證的假設
- 假設 1：確認/否認
- 假設 2：確認/否認
```

---

## 批量分析 Prompt

```
我有多個訪談記錄，請批次分析：

[訪談 1]
...
[訪談 2]
...

請識別：
1. 共同主題
2. 差異點
3. 建議優先順序
```

### 測試類型

| 類型 | 說明 |
|------|------|
| 黑盒測試 | 不提供資訊，模擬駭客 |
| 白盒測試 | 提供原始碼，全面審查 |
| 灰盒測試 | 提供部分資訊，平衡測試 |

| 漏洞 | 嚴重性 | 說明 |

| SQL Injection | 高 | 資料庫注入 |

| CSRF | 中 | 跨站請求偽造 |
| 密碼儲存 | 高 | 明文儲存密碼 |
| 權限控制 | 高 | 未驗證權限 |
| API 安全 | 高 | 無限速/驗證 |

# 外包合約模板

> 來源：第 6 章「外包策略」

---

## 設計外包合約要點

```
## 設計服務合約

### 1. 服務內容
- 設計頁面數量：X 頁
- 交付格式：Figma/Sketch
- 響應式設計：桌面/平板/手機

### 2. 費用與付款
- 總費用：$XXX
- 預付：50%
- 尾款：50%（驗收後）

### 3. 智慧財產權
- 設計稿智慧財產權歸甲方所有
- 乙方保留展示權

### 4. 驗收標準
- [ ] 符合設計 brief
- [ ] 響應式正確顯示
- [ ] 顏色字體一致

### 5. 保密條款
雙方同意對專案資訊保密

### 6. 違約處理
- 乙方未如期交付：退還款項
- 甲方未付款：暫停設計
```

## 成長階梯

### 初級工程師（0-2 年）

```
目标：建立基础，掌握工具

核心能力：
├─ 编程基础（语法、数据结构、算法）
├─ 一门主攻语言
├─ Git 版本控制
├─ 基本数据库知识
├─ 调试和问题解决
└─ 使用 AI 辅助开发

成长方法：
├─ 完成实际项目
├─ 学习设计模式
├─ 编写测试
├─ Code Review 学习
└─ 学会提问
```

### 中級工程師（2-5 年）

```
目标：深化专业，扩展广度

核心能力：
├─ 系统设计能力
├─ 性能优化经验
├─ 安全意识
├─ 架构思维
├─ 团队协作
└─ AI 协同工作

成长方法：
├─ 主导中型项目
├─ 技术分享
├─ Mentoring 新人
├─ 参与架构讨论
└─ 建立技术影响力
```

### 高級工程師（5-10 年）

```
目标：技术领导，业务洞察

核心能力：
├─ 复杂系统设计
├─ 技术选型决策
├─ 跨团队协作
├─ 人才培养
├─ 技术战略
└─ AI 深度应用

成长方法：
├─ 技术选型实践
├─ 跨团队项目
├─ 技术写作/演讲
├─ 开源贡献
└─ 建立个人品牌
```

### 架構師/技術總監（10+ 年）

```
目标：技术战略，技术文化

核心能力：
├─ 技术战略规划
├─ 架构治理
├─ 组织建设
├─ 业务理解
├─ 创新推动
└─ AI 前沿关注

成长方法：
├─ 技术战略制定
├─ 团队建设
├─ 行业交流
├─ 持续学习
└─ 布道推广
```

### AI 輔助學習

```
1. 学习新概念
   「请用简单的语言解释 X」
   「这个概念和 Y 有什么不同？」

2. 制定学习计划
   「请帮我制定 X 的学习路径」
   「每周学习 2 小时，3 个月能学到什么程度？」

3. 练习和反馈
   「请出 5 道关于 X 的练习题」
   「请评价我的答案」

4. 知识巩固
   「请帮我总结 X 的核心要点」
   「我应该重点关注哪些？」

5. 项目驱动
   「我想通过做项目学习 X，给我一个练手项目」
```

```

✓ 多写代码，多踩坑
✓ 学会使用工具和 AI
✓ 建立学习习惯
✓ 找到技术热情

✗ 不要只依赖 AI
✗ 不要忽视基础
✗ 不要急于跳槽
✗ 不要闭门造车
```

✓ 学会架构设计
✓ 建立影响力

✗ 不要只关注技术
✗ 不要停止学习

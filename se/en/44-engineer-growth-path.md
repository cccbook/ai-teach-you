## Growth Ladder

### Junior Engineer (0-2 Years)

```
Goal: Build foundations, master tools

Core Skills:
├─ Programming basics (syntax, data structures, algorithms)
├─ One primary language
├─ Git version control
├─ Basic database knowledge
├─ Debugging and problem solving
└─ Using AI to assist development

Growth Methods:
├─ Complete real projects
├─ Learn design patterns
├─ Write tests
├─ Learn from Code Review
└─ Learn how to ask questions
```

### Mid-Level Engineer (2-5 Years)

```
Goal: Deepen expertise, expand breadth

Core Skills:
├─ System design ability
├─ Performance optimization experience
├─ Security awareness
├─ Architecture thinking
├─ Team collaboration
└─ AI collaboration

Growth Methods:
├─ Lead medium-sized projects
├─ Technical sharing
├─ Mentor newcomers
├─ Participate in architecture discussions
└─ Build technical influence
```

### Senior Engineer (5-10 Years)

```
Goal: Technical leadership, business insight

Core Skills:
├─ Complex system design
├─ Technical selection decisions
├─ Cross-team collaboration
├─ Talent development
├─ Technical strategy
└─ Deep AI application

Growth Methods:
├─ Technical selection practice
├─ Cross-team projects
├─ Technical writing/speaking
├─ Open source contribution
└─ Build personal brand
```

### Architect/Technical Director (10+ Years)

```
Goal: Technical strategy, technical culture

Core Skills:
├─ Technical strategic planning
├─ Architecture governance
├─ Organization building
├─ Business understanding
├─ Innovation driving
└─ AI frontier awareness

Growth Methods:
├─ Technical strategy formulation
├─ Team building
├─ Industry exchange
├─ Continuous learning
└─ Evangelism
```

### AI-Assisted Learning

```
1. Learning New Concepts
   "Please explain X in simple terms"
   "How is this concept different from Y?"

2. Creating Learning Plans
   "Please create a learning path for X"
   "What level can I reach in 3 months studying 2 hours a week?"

3. Practice and Feedback
   "Please give me 5 exercises about X"
   "Please evaluate my answers"

4. Knowledge Consolidation
   "Please summarize the key points of X"
   "What should I focus on?"

5. Project-Driven Learning
   "I want to learn X through projects, give me a practice project"
```

## Advice for Different Stages

### Junior Engineers

```
✓ Build solid foundations, don't rush
✓ Write more code, make more mistakes
✓ Learn to use tools and AI
✓ Build learning habits
✓ Find your technical passion

✗ Don't rely solely on AI
✗ Don't neglect fundamentals
✗ Don't job hop too quickly
✗ Don't work in isolation
```

### Mid-Level Engineers

```
✓ Expand technical breadth
✓ Take on more responsibility
✓ Learn architecture design
✓ Build influence
✓ Use AI deeply

✗ Don't focus only on technology
✗ Don't stop learning
✗ Don't be afraid of challenges
✗ Don't neglect soft skills
```

### Senior Engineers

```
✓ Cultivate talent
✓ Think strategically about technology
✓ Cross-team collaboration
✓ Build personal brand
✓ Lead AI practices

✗ Don't stop coding
✗ Don't ignore business
✗ Don't become stagnant
✗ Don't stop sharing
```

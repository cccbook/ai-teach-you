# 17. RISC-V 指令集——精簡指令的設計哲學

## 17.1 什麼是 RISC-V？

RISC-V（發音 "risk-five"）是一個開放的指令集架構（ISA），起源於 UC Berkeley 實驗室。

### 為什麼選擇 RISC-V？

- **開放標準**：任何人可以免費使用
- **精簡設計**：易於理解和實現
- **模組化**：可根據需求選擇擴展
- **支援 32/64/128 位元**
- **廣泛的生態系統支持**

## 17.2 RISC-V 設計原則

RISC（Reduced Instruction Set Computer）的核心原則：

1. **簡單性**：指令格式統一，解碼簡單
2. **規整性**：所有指令操作都在暫存器中進行
3. **讓簡單的事情簡單**：常見操作有專用指令
4. **速度來自於好的編譯器**：讓編譯器做優化

## 17.3 指令格式

RISC-V 有 6 種基本格式（所有 32 位元指令）：

```
R 類型：register-register 操作
+--------+----+--+----+---+----+------+
| 31-25 |24-20|19-15|14-12|11-7|  6-0 |
| funct7 | rs2 | rs1 |funct3| rd |opcode|
+--------+----+--+----+---+----+------+

I 類型：立即數操作、載入
+--------+------+----+---+----+------+
|  11-0  |19-15 |14-12|11-7|  6-0 |
| imm[11:0]| rs1 |funct3| rd |opcode|
+--------+------+----+---+----+------+

S 類型：儲存
+-------+----+--+----+------+---+----+------+
|31-25|24-20|19-15|11-7 |  | 6-0 |
|imm[11:5]|rs2 | rs1 |imm[4:0]|opcode|
+-------+----+--+----+------+---+----+------+

B 類型：分支（位移是 2 的倍數）
+---------+----+--+----+------+---+----+------+
|31    25|24-20|19-15|11-8|7|  | 6-0 |
|imm[12]|imm[10:5]|rs2|rs1|imm[4:1]|imm[11]|opcode|
+---------+----+--+----+------+---+----+------+

U 類型：上位立即數
+----------------+---+----+------+
|    31-12       |11-7|  6-0 |
|  imm[31:12]    | rd |opcode|
+----------------+---+----+------+

J 類型：跳躍（位移是 2 的倍數）
+---------+--------+--+----+---+----+------+
|31    25|24-21|20|19-12|11-8|7|  | 6-0 |
|imm[20]|imm[10:1]|imm[11]|imm[19:12]|imm[11]|opcode|
+---------+--------+--+----+---+----+------+

```

## 17.4 暫存器

| 暫存器 | 別名 | 用途 | 是否保存 |
|--------|------|------|---------|
| x0 | zero | 常數 0 | - |

| x2 | sp | 堆疊指標 | 是 |
| x3 | gp | 全域指標 | - |
| x4 | tp | 執行緒指標 | - |
| x5 | t0 | 暫存 | 否 |
| x6 | t1 | 暫存 | 否 |
| x7 | t2 | 暫存 | 否 |
| x8 | s0/fp | 保存/框架指標 | 是 |
| x9 | s1 | 保存 | 是 |
| x10 | a0 | 參數0/返回值0 | 否 |
| x11 | a1 | 參數1/返回值1 | 否 |
| x12 | a2 | 參數2 | 否 |
| x13 | a3 | 參數3 | 否 |
| x14 | a4 | 參數4 | 否 |
| x15 | a5 | 參數5 | 否 |
| x16 | a6 | 參數6 | 否 |
| x17 | a7 | 參數7 | 否 |
| x18-x27 | s2-s11 | 保存 | 是 |
| x28-x31 | t3-t6 | 暫存 | 否 |

### 17.4.2 特殊暫存器

- **PC (Program Counter)**：儲存當前指令位址

## 17.5 整數指令（RV32I/RV64I）

### 17.5.1 算術指令

```asm
# 加法
add   a0, a0, a1    ; a0 = a0 + a1
addi  a0, a0, 10    ; a0 = a0 + 10

# 減法
sub   a0, a0, a1    ; a0 = a0 - a1

# 負數（沒有 neg 指令，用 sub）
neg   a0, a0        ; a0 = 0 - a0
sub   a0, x0, a0    ; 等價於 neg
```

### 17.5.2 邏輯指令

```asm
and   a0, a0, a1    ; a0 = a0 & a1
andi  a0, a0, 0xFF  ; a0 = a0 & 0xFF

or    a0, a0, a1    ; a0 = a0 | a1
ori   a0, a0, 0xFF  ; a0 = a0 | 0xFF

xor   a0, a0, a1    ; a0 = a0 ^ a1
xori  a0, a0, 0xFF  ; a0 = a0 ^ 0xFF
```

### 17.5.3 移位指令

```asm
# 左移
sll   a0, a0, a1    ; a0 = a0 << a1 (邏輯)
slli  a0, a0, 3     ; a0 = a0 << 3

# 右移（邏輯）
srl   a0, a0, a1    ; a0 = a0 >> a1 (補零)
srli  a0, a0, 3     ; a0 = a0 >> 3

# 右移（算術，保留符號）
sra   a0, a0, a1    ; a0 = a0 >> a1 (算術)
srai  a0, a0, 3     ; a0 = a0 >> 3 (算術)
```

### 17.5.4 比較指令

```asm
# 設定小於
slt   a0, a0, a1    ; a0 = (a0 < a1) ? 1 : 0 (signed)
slti  a0, a0, 10    ; a0 = (a0 < 10) ? 1 : 0 (signed)

# 設定小於（無號）
sltu  a0, a0, a1    ; a0 = (a0 < a1) ? 1 : 0 (unsigned)
sltiu a0, a0, 10    ; a0 = (a0 < 10) ? 1 : 0 (unsigned)
```

## 17.6 負載和儲存

RISC-V 是**負載/儲存架構**，記憶體只能透過專用指令存取。

```asm
# 載入
lw   a0, 0(sp)      ; a0 = *((int*)(sp + 0))
lb   a0, 0(sp)      ; a0 = *((int8*)(sp + 0)) (有號)
lbu  a0, 0(sp)      ; a0 = *((uint8*)(sp + 0)) (無號)
lhu  a0, 0(sp)      ; a0 = *((uint16*)(sp + 0)) (無號)

# 儲存
sw   a0, 0(sp)      ; *((int*)(sp + 0)) = a0
sb   a0, 0(sp)      ; *((int8*)(sp + 0)) = a0
sh   a0, 0(sp)      ; *((int16*)(sp + 0)) = a0

# 沒有直接記憶體到記憶體的指令！
# 必須：load → operate → store
```

## 17.7 分支指令

```asm
# 無條件相等/不等
beq  a0, a1, label   ; if (a0 == a1) goto label
bne  a0, a1, label   ; if (a0 != a1) goto label

# 有號比較
blt  a0, a1, label   ; if (a0 < a1) goto label (signed)
bge  a0, a1, label   ; if (a0 >= a1) goto label (signed)

# 無號比較
bltu a0, a1, label   ; if (a0 < a1) goto label (unsigned)
bgeu a0, a1, label   ; if (a0 >= a1) goto label (unsigned)

# 組合條件（需要多條指令）
bgt  a0, a1, label   ; if (a0 > a1) goto label
                        ; slt t0, a1, a0
                        ; bne t0, x0, label
```

## 17.8 跳躍指令

```asm
# 跳躍並連結
jal  ra, label       ; ra = pc + 4; goto label

# 跳躍並連結（指標）
jalr ra, 0(a0)      ; ra = pc + 4; goto *a0

# 返回（偽指令）
ret                  ; jalr x0, 0(ra)
```

## 17.9 上位立即數

```asm
# 載入上位立即數
lui  a0, 0x12345    ; a0 = 0x12345000

# 上位立即數加到 PC
auipc a0, 0x12345    ; a0 = pc + 0x12345000

# 配合 addi 形成大立即數
lui  a0, 0x12345
addi a0, a0, 0x678   ; a0 = 0x12345678
```

## 17.10 偽指令

RISC-V 組語提供很多偽指令，方便程式設計：

```asm
# 無操作
nop                  ; addi x0, x0, 0

# 載入立即數到暫存器
li   a0, 42          ; 會展開為 lui + addi
li   a0, 0x12345678  ; lui + addi

# 移動
mv   a0, a1          ; addi a0, a1, 0

# 清除
clr  a0              ; xor a0, a0, a0
```

## 17.11 RISC-V 擴展

RISC-V 是模組化的，可以選擇不同的擴展：

| 擴展 | 名稱 | 內容 |
|------|------|------|
| M | 乘除法 | mul, div, rem |
| F | 單精度浮點 | flw, fsw, fadd.s |
| D | 雙精度浮點 | fld, fsd, fadd.d |
| A | 原子操作 | amoadd, amoswap |
| C | 壓縮指令 | 16 位元指令 |
| G | 一般用途 | 基本 + M, F, D, A |

完整標記：`RV64IMAFDC` = RV64GC

## 17.12 實際範例

### C 程式

```c
long long add(long long a, long long b) {
    return a + b;
}
```

### RISC-V 組語（O0）

```asm
add:
    # 函式進入
    addi sp, sp, -16    ; 分配棧框架
    sd ra, 8(sp)         ; 保存返回位址
    sd a0, 0(sp)         ; 保存 a (用於返回值)
    
    # a1 已經是 b
    
    # 加法
    add a0, a0, a1
    
    # 函式離開
    ld ra, 8(sp)         ; 恢復返回位址
    addi sp, sp, 16      ; 釋放棧框架
    ret
```

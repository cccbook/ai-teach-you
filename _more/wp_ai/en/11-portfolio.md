# 11. Creating a Portfolio Website

## The Significance of Portfolio Websites

A portfolio website is a creator's "visual resume."

It's not about putting all your work together in one go, but:

> **Strategically showcase your best work to make your target audience believe in your capabilities**

### Purposes of Portfolio Websites

| Purpose | Audience |
|---------|----------|
| Job application | HR, design supervisors |
| Client acquisition | Potential clients |
| Brand building | Industry peers |
| Self-documentation | Yourself |

## Essential Pages for Portfolio Websites

### Required Pages

```
1. Home (Homepage)
   └→ Immediately understand who you are and what you do

2. Portfolio (Work)
   └→ Showcase your best work

3. About
   └→ Who you are, why hire you

4. Contact
   └→ How to reach you
```

### Optional Pages

- **Services**: If you offer paid services
- **Process**: Explain collaboration methods
- **Blog/Tutorials**: Share knowledge, build professional image

## Choosing a Platform

### Portfolio Platform Comparison

| Platform | Pros | Cons | Suitable For |
|----------|------|------|---------------|
| Behance | Large traffic, industry recognized | Limited customization | Designers' standard |
| Dribbble | Active designer community | Requires application | UI/UX designers |
| Cargo | Strong design sense | Simple features | Artists, photographers |
| Squarespace | Many professional templates | Higher cost | Everyone |
| Wix | AI building | General design sense | Beginners |
| Self-hosted | Complete freedom | Needs maintenance | Advanced users |

### Recommended Solutions in This Chapter

- **First choice**: Squarespace (good design quality, easy to use)
- **Alternative**: Wix (complete features)
- **Complement**: Behance as work showcase supplement

## Squarespace Practice

### Step 1: Choose Template

1. Go to `squarespace.com`
2. Click "Get Started"
3. Browse "Portfolio" or "Design" category templates
4. Choose a style you like

**Recommended templates**:
- **Avenue**: For photographers, large image display
- **Tudor**: For designers, grid-based work wall
- **Paloma**: For artists, gallery style

### Step 2: Set Basic Information

After entering editor, set:

1. **Site Title**: Your name or studio name
2. **Tagline**: One-sentence introduction
3. **Logo**: Upload your logo or text

### Step 3: Upload Work

1. Click "Pages" → "Add"
2. Choose "Gallery" or "Portfolio"
3. Upload images or create projects

**Project creation tips**:
```
Each project includes:
- Cover image (most important visual)
- Project name
- Client/year (optional)
- Project description
- Detailed images
```

### Step 4: Edit Project Descriptions

Good project descriptions make work more persuasive:

```
Format:
[Project name] - [Client/brand]
[One sentence describing project goal]
[Your role]: What you were responsible for
[Tools/techniques]: [Tool names]
[Results/impact]: [Data if available]
```

**Example**:
```
Starbucks Brand Image Update - Starbucks Taiwan
Redesigned brand visual system to improve younger demographic's favorability.
My role: Brand strategy, visual design
Tools used: Illustrator, Photoshop, Figma
Result: Brand favorability increased by 25%
```

## AI Helps You Write Portfolio Content

### About Page Prompt

```
Role: You are a creative personal brand coach

Task: Help me write copy for the "About" page

Background:
- Name: [name]
- Profession: [Designer/Photographer/Artist]
- Expertise: [Specific area you're good at]
- Experience: [Brief important experience]
- Style: [Your design style or philosophy]

Needs:
1. Total length: 300-400 words
2. Structure:
   - Opening: Who I am, what I focus on
   - Middle: Background story, design philosophy
   - Ending: Why hire me, how to collaborate
3. Tone: Confident but not arrogant, has personality
4. Add a small story or feature that makes people remember you
```

### Project Description Prompt

```
Role: You are a copywriter focused on design

Task: Help me write copy for my work

Work:
- Name: [work name]
- Type: [Poster/Website/Brand image, etc.]
- Client: [Client name if any]
- Goal: [Goal of this project]
- My role: [What I was responsible for]
- Tools used: [Tools]

Needs:
1. Each work 50-100 words
2. Focus on explaining:
   - Project background/challenge
   - Your solution
   - Final results (if any)
3. Tone professional, persuasive
4. Avoid empty adjectives, use specific descriptions
```

### Collaboration Invitation Copy Prompt

```
Role: You are a creative worker good at communication

Task: Help me write copy for contact area or collaboration invitation

Background:
- Services offered: [Type]
- People suitable to contact me: [Ideal client profile]
- Collaboration method: [Project/on-site/consultant, etc.]
- Contact method: [Email/form link]

Needs:
1. 50-100 words
2. Tone enthusiastic but professional
3. Clearly explain:
   - What problems I'm suitable to solve
   - How to take the first step
4. Clear CTA
```

## Portfolio Visual Design Principles

### Image Display Tips

1. **Quality first**: Better to show fewer images than low-quality ones
2. **Consistent sizes**: Same type of work should use same display ratio
3. **White space**: Give work room to breathe
4. **Detail enlargement**: Important design details can be displayed separately

### Color Strategy

Portfolio website color suggestions:

1. **Minimalist black and white**: Let work itself be the visual focus
2. **Neutral gray and white**: Clean background, highlight work
3. **Brand colors**: If you have obvious personal brand colors

**Avoid**:
- Too flashy backgrounds
- Designs that compete with work for attention
- Too many decorative elements

## Key Points for Different Creators' Portfolios

### Graphic Designer

```
Essential elements:
✓ Logo design work
✓ Brand visual system
✓ Packaging design
✓ Posters/Print materials

Display focus:
- Design process (draft → final)
- Design explanation (why designed this way)
- Before/After
```

### Web/UI Designer

```
Essential elements:
✓ Website design mockups
✓ App interface design
✓ User flow diagrams
✓ Responsive design showcase

Display focus:
- Complete page screenshots
- Animation effect demos (GIF/video)
- Design system components
```

### Photographer

```
Essential elements:
✓ Series work (not scattered single images)
✓ Clear shooting themes
✓ Multiple style displays (optional)

Display focus:
- High-quality large images
- Same series works collected
- EXIF or shooting info (optional)
```

### Illustrator

```
Essential elements:
✓ Illustration portfolio
✓ Style showcase (multiple styles can add points)
✓ Commercial application cases (optional)

Display focus:
- Complete illustration works
- Character design showcase
- Creation process
```

### Video/Motion Designer

```
Essential elements:
✓ Work videos (can embed YouTube/Vimeo)
✓ Static frame screenshots
✓ Design process

Display focus:
- Video demo reel
- Playable GIFs or short videos
- Behind-the-scenes making-of
```

## AI Helps You Generate Portfolio Assets

### Concept Image Generation Prompt

```
Using Midjourney:

"[Describe the concept image you need],
digital illustration, [style description],
professional design portfolio quality,
high detail, 4k"

Example:
"Abstract geometric shapes forming a modern logo concept,
minimalist digital illustration, clean lines,
professional design portfolio quality, high detail, 4k"
```

### Mockup Generation Prompt

```
Using AI or online tools:

Upload your design on Smartmockups or Placeit,
choose suitable scene to generate mockup.

Common scenes:
- Phone display (App interface)
- Website display (computer screen)
- Packaging display (product shelf)
- Business card display (office desk)
```

## Portfolio Checklist

### Content Check

- [ ] At least 6-10 representative works
- [ ] Work descriptions clear, in-depth
- [ ] About page makes people remember who you are
- [ ] Contact information easy to find
- [ ] Clear CTA (consultation/collaboration)

### Design Check

- [ ] Work images high quality (at least 1500px wide)
- [ ] Layout tidy, appropriate white space
- [ ] Website style consistent with work tone
- [ ] Mobile version displays properly
- [ ] Loading speed reasonable

### Technical Check

- [ ] All links work
- [ ] Images have alt text
- [ ] Has SSL certificate
- [ ] Social links valid

### Strategy Check

- [ ] Works arranged in order by time or type
- [ ] Best works placed at front
- [ ] Focused display for target audience
- [ ] Update frequency reasonable (regular updates)

## Maintaining Portfolio Over Time

### Regular Updates

- Update after completing each project
- Check at least quarterly
- Remove outdated or non-representative works

### Collect Feedback

- Ask clients if they can use on their websites
- Collect client testimonials or feedback
- Record project data results

### Build External Links

- Link to Behance/LinkedIn
- Share work on social media
- Participate in design communities to increase visibility

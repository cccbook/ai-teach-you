# 14. Let AI Help You Write Copy and Translate

## Advantages and Limitations of AI Copywriting

### Advantages of AI Copywriting

| Advantage | Description |
|-----------|-------------|
| Fast speed | Can output draft in seconds |
| Large quantity | Can generate quickly in volume |
| Many variations | Can easily adjust style and tone |
| When don't know what to write | Can help brainstorm |
| When don't know how to write | Can give examples and directions |

### Limitations of AI Copywriting

| Limitation | Description |
|------------|-------------|
| Lacks unique perspective | Says what everyone knows |
| May not be accurate enough | Factual data needs verification |
| Grammar may be wrong | Chinese sometimes has strange usage |
| Too templated | Lacks human touch |
| Needs human oversight | You still need to confirm |

## Correct Ways to Use AI Copywriting

### Correct Approach

```
1. AI helps you brainstorm → You choose direction
2. AI helps you write draft → You modify and polish
3. AI helps you translate → You proofread and adjust
4. AI helps you duplicate → You add personal touch
```

### Incorrect Approach

```
✗ Directly copy-paste and use
✗ Completely rely on AI without any modifications
✗ Not check facts before publishing
✗ Expect AI to completely match your style
```

## AI Copywriting Generation Prompt Techniques

### Basic Prompt Structure

```
Role: You are a professional [field] copywriter

Task: [Specifically what to write]

Background info:
- [Related background]
- [Target audience]
- [Product/service introduction]

Writing requirements:
- Tone: [Formal/Relaxed/Professional/Friendly, etc.]
- Length: [Specific word count]
- Style: [Specific description]
- Format: [Special requirements]

Prohibited:
- [Elements you don't want to appear]
- [Style to avoid]

Extra requirements:
- [Elements that must be included]
```

## Common Copywriting Type Prompts

### Social Media Posts

```
Role: You are a social media marketing expert, proficient in
      Instagram, Facebook, Line content strategies.

Task: Help me write a week's social media posts

Brand/Individual: [name]
Field: [field]
Style: [style description]
This week's theme: [Core message to convey this week]

Needs:
1. One post per day, 7 posts total
2. Each post includes:
   - Main copy (30-100 characters, adjust for platform)
   - Hashtags (5-10)
   - Image suggestions (optional)
3. Diverse styles, not all the same
4. Interactive elements: At least 3 posts should have questions or CTA

Format: Present in table, clearly mark date and platform
```

### Email Marketing Copy

```
Role: You are an email marketing expert focused on conversion rate optimization.

Task: Help me write a promotional email

Background:
- Email purpose: [New product/Promotion/Event, etc.]
- Target readers: [Describe readers]
- Main message: [Core selling point]
- CTA goal: [What you want readers to do]

Needs:
1. Subject Line: Provide 5 versions
2. Preview text: Match subject line
3. Email body structure:
   - Opening hook (keep them reading)
   - Main content (1-2 paragraphs)
   - Social proof (optional)
   - CTA button
4. Keep length at 200-300 characters
5. A/B test versions (if any)

Format: Simulate actual email presentation
```

### Product Copy

```
Role: You are a top e-commerce copywriter.

Task: Help me write copy for product page

Product name: [name]
Product type: [type]
Target audience: [description]
Main functions: [function list]
Unique selling point: [USP]

Needs:
1. Product title (eye-catching, includes keywords)
2. Product subtitle (one sentence explaining biggest selling point)
3. Product description (100-150 characters)
   - Opening: Grab attention
   - Middle: Feature explanation
   - Ending: Why choose this
4. Bullet Points (5-7 feature points)
5. Common FAQ (3-4 questions)

Tone: Professional but not stiff, emphasize benefits not features
```

### Complete Landing Page Copy

```
Role: You are a high-conversion landing page copywriter.

Task: Help me write complete landing page copy

Product/Service: [name]
Target audience: [description]
Main value: [Core selling point]
Target conversion: [What visitors should do]

Needs:
1. Hero Section
   - Main headline: Pain point/benefit oriented
   - Subheadline: Detailed explanation
   - CTA button text

2. Value Proposition Section (3-4 features)
   - Each feature: Title + description

3. Social Proof (2-3 testimonials)

4. FAQ Section (4-5 questions)

5. Final CTA

Length: Complete but concise, each section has substantial content
Style: Persuasive but not cheesy
```

### Ad Copy

```
Role: You are a creative advertising copywriter.

Task: Help me write [Google/FB/IG] ad copy

Ad purpose: [Brand exposure/Product sales/Registration, etc.]
Target audience: [description]
Product/Service: [name]
Main message: [Key point to convey]
Offer (if any): [offer content]

Needs:
1. Provide 3 versions for each format:
   - Headline (30 characters or less)
   - Description (90 characters or less)
   - CTA (5 characters maximum)
2. Different styles: Rational/Emotional/Humorous
3. Include numbers, questions, comparisons and other techniques

Format: Clearly mark each version
```

## AI Translation Techniques

### Basic Translation Principles

```
English → Chinese:
- Don't translate word by word
- Smooth, natural Chinese expression
- Adjust sentence structure appropriately
- Cultural conversion (examples, metaphors)

Chinese → English:
- Clear subjects
- Avoid redundant words
- Follow English idioms
```

### AI Translation Prompt

```
Role: You are a professional [Chinese-English/English-Chinese] translator,
      proficient in cultural backgrounds of both languages.

Task: Translate the following content

Source language: [language]
Target language: [language]
Content type: [Website/Marketing/Technical/General]

Original text:
[Paste content to translate]

Translation requirements:
1. Accurate meaning, don't omit important information
2. Conform to target language expression habits
3. Adjust sentence structure for smooth reading
4. Retain professional terms (if appropriate)
5. If there are localization needs, make adjustments

Format:
- Translation result
- Notes (important terms, translation choices, etc.)
```

### Post-Translation Polishing

After translation, ask AI to polish:

```
Task: Help me polish the following Chinese-English translation,
      making it more natural and conforming to native speaker habits.

[Paste translation content]

Requirements:
1. Check if grammar is correct
2. Check if wording is natural
3. Check if tone is consistent
4. Mark modified places and explain reasons
```

## Post-Processing of AI Copy

### Human Review Checklist

After getting AI-generated copy, must check:

- [ ] Are facts and data correct
- [ ] Are names and terms accurate
- [ ] Grammar errors
- [ ] Punctuation
- [ ] Any typos
- [ ] Is logic smooth
- [ ] Does style match brand tone
- [ ] Any sensitive content

### Adding Human Elements

The problem with AI copy is it's "too AI." You can add human elements like this:

1. **Add your own story**
   ```
   AI: "This product uses the latest technology, greatly improving efficiency."
   Human version: "In my first week using this product, I saved 3 hours of time."
   ```

2. **Add specific details**
   ```
   AI: "Many people have this problem."
   Human version: "Have you also experienced..." (reader visualization)
   ```

3. **Change sentence structure and wording**
   ```
   AI: "Our product is the best choice."
   Human version: "You won't want to go back once you've used it."
   ```

4. **Add emotion**
   ```
   AI: "Thank you for your purchase."
   Human version: "So happy you chose us!"
   ```

## Avoiding AI Copywriting Pitfalls

### Pitfall 1: High Duplicate Rate

Google may lower rankings for duplicate content.

**Solutions**:
- Don't completely copy others' content
- Modify after AI generation
- Add unique perspectives and experiences

### Pitfall 2: Strange Grammar

AI-translated or generated Chinese sometimes feels unnatural.

**Solutions**:
- Read out loud
- Ask native speakers to review
- Use Google Docs grammar check

### Pitfall 3: Outdated Information

AI's training data has a cutoff date and may have outdated information.

**Solutions**:
- Verify important facts
- Update with latest data
- Add current events

### Pitfall 4: Copyright Issues

Copying others' copy and translating may have copyright issues.

**Solutions**:
- Only reference structure and style
- Don't copy verbatim
- Add original content

## Exercises to Improve AI Copywriting Skills

### Exercise 1: Style Conversion

```
Take AI-generated copy and try rewriting with different tones:
- Formal → Relaxed
- Business → Playful
- Emotional → Rational

Compare which version is more suitable for your goal.
```

### Exercise 2: Topic Variation

```
Use the same Prompt to generate 3 versions,
compare differences, choose best elements to combine.
```

### Exercise 3: Accumulate Feedback

```
After each use of AI copy, record:
- Which version worked best?
- What feedback did readers/clients give?
- What can be improved next time?

Gradually build your own "effective copy library."
```

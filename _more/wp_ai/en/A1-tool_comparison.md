## AI Tools Selection Decision Tree

```
Are you a complete beginner?
├── Yes → Wix ADI or Durable
│    └→ Canva + ChatGPT (assistance)
│
├── Have some technical background?
│   ├── Want to learn coding?
│   │   └── Cursor + Claude
│   │
│   └── Just want to build quickly?
│       └── Durable or Wix
│
└── Are you a designer or developer?
    ├── Want to keep design control?
    │   └── Framer or Webflow
    │
    └── Want to write code yourself?
        └── VS Code + GitHub Copilot + Claude
```
